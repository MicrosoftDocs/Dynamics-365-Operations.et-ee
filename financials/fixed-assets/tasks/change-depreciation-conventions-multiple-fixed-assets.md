[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="05228-104">See ülesanne värskendab konkreetse põhivara grupi kulumiarvestusreegli.</span><span class="sxs-lookup"><span data-stu-id="05228-104">This task updates the depreciation convention for a specified fixed asset group.</span></span> <span data-ttu-id="05228-105">See ülesande juhend kasutab demoettevõtet USMF.</span><span class="sxs-lookup"><span data-stu-id="05228-105">This task guide uses the USMF demo company.</span></span>

3. <span data-ttu-id="05228-108">Klõpsake loendis valitud real olevat linki.</span><span class="sxs-lookup"><span data-stu-id="05228-108">In the list, click the link in the selected row.</span></span>
4. <span data-ttu-id="05228-109">Sisestage kuupäev väljale Teenusesse saatmise algus.</span><span class="sxs-lookup"><span data-stu-id="05228-109">In the Placed in service start field, enter a date.</span></span>

    * <span data-ttu-id="05228-111">Värskendatakse ainult varasid, mis kuuluvad valitud kulumiraamatusse ja mis on teenusesse saadetud nende kuupäevade vahel.</span><span class="sxs-lookup"><span data-stu-id="05228-111">Only assets that are a part of the select depreciation book and that have been placed in service between these dates will be updated.</span></span>

    * <span data-ttu-id="05228-115">Kontrollige aruande printimist soovitud sihtkohta.</span><span class="sxs-lookup"><span data-stu-id="05228-115">Verify the report will print to the desired destination.</span></span>

9. <span data-ttu-id="05228-117">Klõpsake käsku Filtreeri.</span><span class="sxs-lookup"><span data-stu-id="05228-117">Click Filter.</span></span>
10. <span data-ttu-id="05228-118">Valige loendist Põhivaragrupp.</span><span class="sxs-lookup"><span data-stu-id="05228-118">In the list, select the Fixed asset group.</span></span>

12. <span data-ttu-id="05228-120">Valige soovitud Põhivaragrupp.</span><span class="sxs-lookup"><span data-stu-id="05228-120">Select the desired Fixed asset group.</span></span>
13. <span data-ttu-id="05228-121">Klõpsake loendis valitud real olevat linki.</span><span class="sxs-lookup"><span data-stu-id="05228-121">In the list, click the link in the selected row.</span></span>

15. <span data-ttu-id="05228-123">Klõpsake nuppu OK.</span><span class="sxs-lookup"><span data-stu-id="05228-123">Click OK.</span></span>
