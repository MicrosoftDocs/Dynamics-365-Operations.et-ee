# <a name="set-a-users-preferred-time-zone"></a>Kasutaja eelistatud ajavööndi seadistamine

[!include[task guide banner](../../includes/task-guide-banner.md)]

Järgmine protseduur selgitab, kuidas süsteemiadministraatori rollis kasutaja saab määrata kasutaja ajavööndit.

1. Minge jaotisse Süsteemihaldus > Kasutajad > Kasutajad.
2. Otsige loendist ja valige soovitud kirje.
3. Klõpsake loendis valitud real olevat linki.
4. Klõpsake suvandit Kasutaja suvandid.
5. Klõpsake vahekaarti Eelistused.
6. Valige suvand väljal Ajavöönd.
7. Klõpsake nuppu Salvesta.
