<span data-ttu-id="bdf86-114">Suviti palkate tavaliselt ettevõtte pakutavate praktikakohtade täitmiseks 15–20 osalise tööajaga üliõpilast.</span><span class="sxs-lookup"><span data-stu-id="bdf86-114">In the summer, you usually hire 15-20 part-time college students to fill available internships in your company.</span></span> <span data-ttu-id="bdf86-115">Sel aastal soovite palgata viis raamatupidajat, viis tellimuse töötlejat ja viis kassapidajat.</span><span class="sxs-lookup"><span data-stu-id="bdf86-115">This year, you want to hire five accountants, five order processors, and five cashiers.</span></span> <span data-ttu-id="bdf86-116">Selle asemel et luua iga ametikoha ja töötaja kirje eraldi, saate luua ühe hulgivärbamisprojekti nimega „SuvePraktikandid”.</span><span class="sxs-lookup"><span data-stu-id="bdf86-116">Instead of creating each position record and worker record separately, you create one mass hire project called “SummerInterns”.</span></span> <span data-ttu-id="bdf86-117">Projekti algus- ja lõppkuupäev korreleeruvad hulgivärbamisprojektis loodud ametikohtade algus- ja lõppkuupäevadega.</span><span class="sxs-lookup"><span data-stu-id="bdf86-117">The project start and end dates correlate with the start and end dates of the position durations for the positions you create for the mass hire project.</span></span>

<span data-ttu-id="bdf86-122">Pärast üliõpilaste valimist praktikakohtadele saate iga tudengi kohta sisestada jaotises **Ametikoha üksikasjad** teabe, millisele ametikohale te nad palkasite.</span><span class="sxs-lookup"><span data-stu-id="bdf86-122">After selecting students to hire for the internship positions, you'll enter each student’s information in the **Position details** for the position that you're hiring them for.</span></span> <span data-ttu-id="bdf86-123">Kui olete kõik ametikoha üksikasjad sisestanud, valige lehel Hulgivärbamisprojektid ametikoht ja seejärel klõpsake valikut **Palka**.</span><span class="sxs-lookup"><span data-stu-id="bdf86-123">When you have entered all of the position details, select the position in the Mass hire projects page, and then click **Hire**.</span></span> <span data-ttu-id="bdf86-124">Igale ametikohale luuakse ametikoha ning töötaja kirje, mis määratakse õigele ametikohale iga palgatava inimese jaoks.</span><span class="sxs-lookup"><span data-stu-id="bdf86-124">A position record will be created for each position and a worker record will be created and assigned to the correct position for each person who you hire.</span></span>
