# <a name="set-up-i-9-document-types"></a>I-9 dokumenditüüpide seadistamine

[!include[task guide banner](../../../includes/task-guide-banner.md)]

See protseduur näitab, kuidas saate vaadata ja seadistada I-9 kontrollimiseks kasutatavaid dokumenditüüpe. Enne I-9 dokumenditüüpide seadistamist peate seadistama ka väljastavad asutused ja tuvastustüübid. Selle protseduuri loomiseks kasutati demoettevõtte USMF andmeid, mis hõlmab näiteid protseduuri lõpetamiseks vajalikest väljastavatest asutustest ja tuvastustüüpidest.

1. Avage Personaliarvestus > Töötajad > I-9 > I-9 dokumenditüübid.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale I-9 dokumenditüüp.
    * Näide: kooli ID.  
4. Valige ID tüüp.  Näide: kooli ID.
    * Tuvastustüüpide näideteks on isikukood (SSN), viisa number, passi ID või juhiluba.  
5. Valige dokumenditüübi puhul kasutatav I-9 dokumendiloend.
    * I-9 protsessi osana peavad töötajad esitama dokumente, mis näitavad tööandjale nende identiteeti ja tööluba. USA Kodakondsus- ja immigratsiooniteenuste veebisait sisaldab teavet selle kohta, millised dokumendid on vastuvõetavad ja millisesse loendisse need kuuluvad.  http://www.uscis.gov  
6. Sisestage dokumendi tüübi ametlik vormi number väljale Vorm. Näide: kooli ID.
    * Ametlikud vormi numbrid leiate USA Kodakondsus- ja immigratsiooniteenuste veebisaidilt.  http://www.uscis.gov  
7. Märkige või tühjendage ruut Aktiivne.
8. Määrake väljal Aegumine kuupäevaks 27.10.2019.
    * Aegumiskuupäev on valikuline.  
9. Valige dokumenditüübi väljastanud asutus. Näide: Provints/territoorium
10. Klõpsake nuppu Salvesta.
