# <a name="view-an-expense-report"></a><span data-ttu-id="ef56b-103">Kuluaruande vaatamine</span><span class="sxs-lookup"><span data-stu-id="ef56b-103">View an expense report</span></span>

[!INCLUDE [banner](../includes/banner.md)]

<span data-ttu-id="ef56b-104">Kui teete päringu kannete nägemiseks pearaamatus või hangetes, võib päring tagastada liiga palju teavet.</span><span class="sxs-lookup"><span data-stu-id="ef56b-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="ef56b-105">Seetõttu võite soovida otsingut kitsendada.</span><span class="sxs-lookup"><span data-stu-id="ef56b-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="ef56b-106">Vajaliku teabe saamiseks saate kandesse süvitsi minna, et näha kuluaruannet, millel kanne kuvati.</span><span class="sxs-lookup"><span data-stu-id="ef56b-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="ef56b-107">Kuluaruannete nägemiseks saate kandest minna süvitsi algse aruande juurde, mis on talletatud kulude haldusse.</span><span class="sxs-lookup"><span data-stu-id="ef56b-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="ef56b-108">Lisaks kulude kogusummale saate vaadata ka üksikasjalikke kandeid, kandeid, mis on sobilikud käibemaksu korvamiseks, ja kuludega seotud kviitungeid.</span><span class="sxs-lookup"><span data-stu-id="ef56b-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="ef56b-109">Et minna süvitsi algse kuluaruandeni, millel kanne kuvati, valige kanne ja seejärel valige **Algdokument**.</span><span class="sxs-lookup"><span data-stu-id="ef56b-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="ef56b-110">Kui vaatate individuaalseid kandeid, saate vaadata ka kandega seotud lisaandmeid.</span><span class="sxs-lookup"><span data-stu-id="ef56b-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="ef56b-111">Näiteks saate üksikasjaliku hotellikulu korral vaadata kulude üksikasjalikku loendit, nagu Interneti-juurdepääs, telefonikõned ja toateenindus.</span><span class="sxs-lookup"><span data-stu-id="ef56b-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
