<span data-ttu-id="ee1f1-104">See ülesande juhend kasutab demoettevõtet USMF.</span><span class="sxs-lookup"><span data-stu-id="ee1f1-104">This task guide uses the USMF demo company.</span></span>  <span data-ttu-id="ee1f1-105">See loob uue põhivara, kasutades Põhivara loendilehte.</span><span class="sxs-lookup"><span data-stu-id="ee1f1-105">It will create a new fixed asset using the Fixed asset list page.</span></span>

    - <span data-ttu-id="ee1f1-116">Sisestage lisateave, mida teie ettevõte selle raamatu puhul vajab.</span><span class="sxs-lookup"><span data-stu-id="ee1f1-116">Enter the additional information that your business needs for this book.</span></span>
