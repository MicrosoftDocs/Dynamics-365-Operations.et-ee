<span data-ttu-id="5946d-104">See ülesande juhend kasutab demoettevõtet USMF.</span><span class="sxs-lookup"><span data-stu-id="5946d-104">This task guide uses the USMF demo company.</span></span>  <span data-ttu-id="5946d-105">See loob uue põhivara, kasutades Põhivara loendilehte.</span><span class="sxs-lookup"><span data-stu-id="5946d-105">It will create a new fixed asset using the Fixed asset list page.</span></span>

1. <span data-ttu-id="5946d-106">Minge jaotisse Põhivarad > Põhivarad > Põhivarad.</span><span class="sxs-lookup"><span data-stu-id="5946d-106">Go to Fixed assets > Fixed assets > Fixed assets.</span></span>
2. <span data-ttu-id="5946d-107">Klõpsake valikut Uus.</span><span class="sxs-lookup"><span data-stu-id="5946d-107">Click New.</span></span>
3. <span data-ttu-id="5946d-108">Sisestage väärtus väljale Põhivara grupp või valige sellelt väljalt.</span><span class="sxs-lookup"><span data-stu-id="5946d-108">In the Fixed asset group field, enter or select a value.</span></span>
    * <span data-ttu-id="5946d-109">Number tuuakse vaikimisi, kui olete põhivarade parameetrites ja põhivara grupis lubanud funktsiooni Nummerda põhivarad automaatselt.</span><span class="sxs-lookup"><span data-stu-id="5946d-109">The Number will default if you have enabled Autonumber fixed assets functionality in the Fixed assets parameters and the Fixed asset group.</span></span>  <span data-ttu-id="5946d-110">Kui te seda teinud pole, sisestage põhivara tuvastamiseks kordumatu number.</span><span class="sxs-lookup"><span data-stu-id="5946d-110">If not, you must enter a unique number to identify the fixed asset.</span></span>  
4. <span data-ttu-id="5946d-111">Sisestage väärtus väljale Nimi.</span><span class="sxs-lookup"><span data-stu-id="5946d-111">In the Name field, type a value.</span></span>
    * <span data-ttu-id="5946d-112">Sisestage lisateave, mida teie ettevõte selle vara puhul vajab.</span><span class="sxs-lookup"><span data-stu-id="5946d-112">Enter the additional information that your business needs for this asset.</span></span>  
5. <span data-ttu-id="5946d-113">Klõpsake valikut Raamatud.</span><span class="sxs-lookup"><span data-stu-id="5946d-113">Click Books.</span></span>

7. <span data-ttu-id="5946d-115">Sisestage number väljale Soetusmaksumus.</span><span class="sxs-lookup"><span data-stu-id="5946d-115">In the Acquisition price field, enter a number.</span></span>
    * <span data-ttu-id="5946d-116">Sisestage lisateave, mida teie ettevõte selle raamatu puhul vajab.</span><span class="sxs-lookup"><span data-stu-id="5946d-116">Enter the additional information that your business needs for this book.</span></span>  
    * <span data-ttu-id="5946d-117">Sisestage täiendav teave, mida teie ettevõte ülejäänud raamatute jaoks vajab.</span><span class="sxs-lookup"><span data-stu-id="5946d-117">Enter the additional information that your business needs for the remaining books.</span></span>
