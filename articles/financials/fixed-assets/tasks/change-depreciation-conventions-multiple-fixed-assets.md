# <a name="change-depreciation-conventions-for-multiple-fixed-assets"></a>Kulumimääramise tava muutmine mitme põhivara puhul

[!include [task guide banner](../../includes/task-guide-banner.md)]

See ülesanne värskendab konkreetse põhivara grupi kulumiarvestusreegli. See ülesande juhend kasutab demoettevõtet USMF.

1. Avage Põhivarad > Perioodilised ülesanded > Hulgivärskendamine
2. Klõpsake väljal Kulumiraamat otsingu avamiseks ripploendi nuppu.
3. Klõpsake loendis valitud real olevat linki.
4. Sisestage kuupäev väljale Teenusesse saatmise algus.
5. Sisestage kuupäev väljale Teenusesse saatmise lõpp.
    * Värskendatakse ainult varasid, mis kuuluvad valitud kulumiraamatusse ja mis on teenusesse saadetud nende kuupäevade vahel.  
6. Valige suvand väljalt Praegune kulumiarvestusreegel.
    * Värskendatakse ainult praeguse kulumiarvestusreegli varasid.  
7. Valige suvand väljalt Uus kulumiarvestusreegel.
    * Kontrollige aruande printimist soovitud sihtkohta.  
8. Jaotise kaasamiseks laiendage kirjeid.
9. Klõpsake käsku Filtreeri.
10. Valige loendist Põhivaragrupp.
11. Klõpsake väljal Kriteeriumid otsingu avamiseks ripploendi nuppu.
12. Valige soovitud Põhivaragrupp.
13. Klõpsake loendis valitud real olevat linki.
14. Klõpsake nuppu OK.
15. Klõpsake nuppu OK.
    *  Protsessi tulemused kuvatakse aruandel Hulgivärskendamine.
