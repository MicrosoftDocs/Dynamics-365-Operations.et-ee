# <a name="create-depreciation-proposal"></a>Kulumisoovituse koostamine

[!include [task guide banner](../../includes/task-guide-banner.md)]

See protseduur kirjeldab, kuidas töötavad kulumi partiisoovitused, ja selles selgitatakse, kuidas põhivarade kulumit soovitada. See ülesanne kasutab demoettevõtet USMF ja raamatupidaja rolli.


## <a name="create-depreciation-proposal"></a>Kulumisoovituse koostamine
1. Avage Põhivarad > Töölehe kirjed > Kulumisoovituse koostamine.
2. Klõpsake väljal Töölehe nimi otsingu avamiseks ripploendi nuppu.
3. Klõpsake loendis valitud real olevat linki.
4. Sisestage kuupäev väljale Lõpukuupäev.
    * Valige suvand Summeeri kulum, et esitada igakuiste kulumite kokkuvõte ühel töölehe real.  
    * Näiteks, kui lõpukuupäeva väärtus on 31. märts 2015, siis luuakse järgmine kirjeldus: "Kulum alates 31. jaanuar 2015." Soovitatud tööleheridade kuupäevavälja väärtuseks seatakse siis 31. märts 2015.  
    * Kulumisoovitust saab filtrida vara, varagrupi või muude kriteeriumide järgi, kasutades suvandit Filtri.  
    * Kui kasutate vormi Looge põhivara soetamise või kulumi pakkumisi, saate soovitada kulumit partiidena. See on soovitatav suuremate soovituste puhul, mis kasutavad rohkem süsteemi ressursse. Kui valite partii suvandi, saate töö ajal endiselt muid ülesandeid teha. Kui soovitate kulumit sel moel, arvutatakse põhivarade väärtusmudelite kulum.  
5. Klõpsake suvandit Loo tööleht.

## <a name="review-depreciation-entries"></a>Kulumiraamatu kannete ülevaade
1. Avage Põhivarad > Töölehe sisestused > Põhivarade tööleht.
2. Otsige loendist ja valige soovitud kirje.
3. Klõpsake valikut Read.
4. Klõpsake valikut Sisesta.
