# <a name="set-up-and-create-depreciation-profiles"></a>Kulumireeglite seadistamine ja loomine

[!include [task guide banner](../../includes/task-guide-banner.md)]

Kulumireeglid määratlevad vara kulumi tüübi ja sageduse.   Salvestamine kasutab demoettevõtet USMF ja raamatupidaja rolli.


## <a name="create-a-depreciation-profile"></a>Kulumireeglite loomine
1. Minge jaotisse Põhivarad > Seadistus > Kulumireeglid.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Kulumireegel.
4. Sisestage väärtus väljale Nimi.
5. Valige suvand väljalt Meetod.
    * Väheneva saldo valimisel peate sisestama protsendi väljale Protsent.  
6. Valige suvand väljalt Kulumiarvestusaasta.
7. Valige suvand väljalt Perioodi sagedus.
8. Sulgege vorm.
