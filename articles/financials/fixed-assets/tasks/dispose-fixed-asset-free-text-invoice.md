# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="00d6d-103">Põhivara käibelt kõrvaldamine vabas vormis arve abil</span><span class="sxs-lookup"><span data-stu-id="00d6d-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="00d6d-104">See protseduur kirjeldab põhivara likvideerimist vabas vormis arve abil.</span><span class="sxs-lookup"><span data-stu-id="00d6d-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="00d6d-105">Avage Müügireskontro > Arved > Kõik vabas vormis arved.</span><span class="sxs-lookup"><span data-stu-id="00d6d-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="00d6d-106">Klõpsake valikut Uus.</span><span class="sxs-lookup"><span data-stu-id="00d6d-106">Click New.</span></span>
3. <span data-ttu-id="00d6d-107">Valige või sisestage väärtus väljal Kliendi konto.</span><span class="sxs-lookup"><span data-stu-id="00d6d-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="00d6d-108">Kinnitage arve vaikekuupäev ja vajaduse korral redigeerige seda.</span><span class="sxs-lookup"><span data-stu-id="00d6d-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="00d6d-109">Kinnitage ülejäänud vaike-päiseväljad, nagu Valuuta, ja vajaduse korral redigeerige neid.</span><span class="sxs-lookup"><span data-stu-id="00d6d-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="00d6d-110">Sisestage arve reale kirjeldus.</span><span class="sxs-lookup"><span data-stu-id="00d6d-110">Enter a Description into the invoice line.</span></span>
7. <span data-ttu-id="00d6d-111">Sisestage või valige arve real põhikonto.</span><span class="sxs-lookup"><span data-stu-id="00d6d-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="00d6d-112">Kinnitage vaike-käibemaksugrupp ja kauba käibemaksugrupp.</span><span class="sxs-lookup"><span data-stu-id="00d6d-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="00d6d-113">Sisestage põhivara ühiku hind või müügisumma.</span><span class="sxs-lookup"><span data-stu-id="00d6d-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="00d6d-114">Klõpsake valikut Rea üksikasjad.</span><span class="sxs-lookup"><span data-stu-id="00d6d-114">Click Line details.</span></span>  
11. <span data-ttu-id="00d6d-115">Valige müüdava põhivara number.</span><span class="sxs-lookup"><span data-stu-id="00d6d-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="00d6d-116">Klõpsake valikut Sisesta.</span><span class="sxs-lookup"><span data-stu-id="00d6d-116">Click Post.</span></span>
