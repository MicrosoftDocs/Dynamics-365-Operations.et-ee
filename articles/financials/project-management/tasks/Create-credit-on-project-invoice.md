# <a name="create-a-credit-note-on-project-invoices"></a>Projektiarvete jaoks kreeditarve loomine

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

1. Avage Projektihaldus ja -arvestus > Projektid > Kõik projektid. 
2. Otsige loendist ja valige soovitud kirje. 
3. Valige projekti ID, mida kreeditarve jaoks kasutatakse. Klõpsake loendis valitud real olevat linki. 
4. Klõpsake tegumiribal valikut Haldamine. 
5. Klõpsake suvandit Arvetöölehed. 
6. Valige kreeditarve jaoks kanne. Märkige loendis valitud rida. 
7. Klõpsake suvandit Funktsioonid. 
8. Klõpsake käsku Vali kreeditarve jaoks. 
9. Kreeditarve jaoks kande valimiseks märkige ruut.
10. Klõpsake nuppu OK. 

Pärast seda toimingut saab on kreeditarve arve soovituse jaoks saadaval.
