# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="8091c-103">Projektiarvete jaoks kreeditarve loomine</span><span class="sxs-lookup"><span data-stu-id="8091c-103">Create a credit note on project invoices</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="8091c-104">Avage Projektihaldus ja -arvestus > Projektid > Kõik projektid.</span><span class="sxs-lookup"><span data-stu-id="8091c-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="8091c-105">Otsige loendist ja valige soovitud kirje.</span><span class="sxs-lookup"><span data-stu-id="8091c-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="8091c-106">Valige projekti ID, mida kreeditarve jaoks kasutatakse.</span><span class="sxs-lookup"><span data-stu-id="8091c-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="8091c-107">Klõpsake loendis valitud real olevat linki.</span><span class="sxs-lookup"><span data-stu-id="8091c-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="8091c-108">Klõpsake tegumiribal valikut Haldamine.</span><span class="sxs-lookup"><span data-stu-id="8091c-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="8091c-109">Klõpsake suvandit Arvetöölehed.</span><span class="sxs-lookup"><span data-stu-id="8091c-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="8091c-110">Valige kreeditarve jaoks kanne.</span><span class="sxs-lookup"><span data-stu-id="8091c-110">Select the transaction for credit note.</span></span> <span data-ttu-id="8091c-111">Märkige loendis valitud rida.</span><span class="sxs-lookup"><span data-stu-id="8091c-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="8091c-112">Klõpsake suvandit Funktsioonid.</span><span class="sxs-lookup"><span data-stu-id="8091c-112">Click Functions.</span></span> 
8. <span data-ttu-id="8091c-113">Klõpsake käsku Vali kreeditarve jaoks.</span><span class="sxs-lookup"><span data-stu-id="8091c-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="8091c-114">Kreeditarve jaoks kande valimiseks märkige ruut.</span><span class="sxs-lookup"><span data-stu-id="8091c-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="8091c-115">Klõpsake nuppu OK.</span><span class="sxs-lookup"><span data-stu-id="8091c-115">Click OK.</span></span> 

<span data-ttu-id="8091c-116">Pärast seda toimingut saab on kreeditarve arve soovituse jaoks saadaval.</span><span class="sxs-lookup"><span data-stu-id="8091c-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
