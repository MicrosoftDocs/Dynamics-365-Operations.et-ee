[!include [task guide banner](../../includes/task-guide-banner.md)]

    * <span data-ttu-id="2acd5-108">Filtreerida saab üksikväärtust või vahemikku.</span><span class="sxs-lookup"><span data-stu-id="2acd5-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="2acd5-109">Vahemiku määratlemisel veenduge, et kasutatakse õiget süntaksit.</span><span class="sxs-lookup"><span data-stu-id="2acd5-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="2acd5-110">Väärtused peaksid olema eraldatud topeltpunktiga (..).</span><span class="sxs-lookup"><span data-stu-id="2acd5-110">The values should be separated by a double period (..).</span></span>
