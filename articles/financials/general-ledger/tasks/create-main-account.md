    * <span data-ttu-id="dfc2a-114">Saate muuta deebet- või kreeditsaldo vaikeväärtust.</span><span class="sxs-lookup"><span data-stu-id="dfc2a-114">Change the default debit or credit balance.</span></span>

10. <span data-ttu-id="dfc2a-117">Klõpsake loendis valitud real olevat linki.</span><span class="sxs-lookup"><span data-stu-id="dfc2a-117">In the list, click the link in the selected row.</span></span>

14. <span data-ttu-id="dfc2a-121">Klõpsake vahekaarti Lisa.</span><span class="sxs-lookup"><span data-stu-id="dfc2a-121">Click Add.</span></span>
15. <span data-ttu-id="dfc2a-122">Märkige loendis valitud rida.</span><span class="sxs-lookup"><span data-stu-id="dfc2a-122">In the list, mark the selected row.</span></span>
