# <a name="create-advanced-rules-for-journals"></a>Täpsemate reeglite loomine töölehtede jaoks

[!include [task guide banner](../../includes/task-guide-banner.md)]

See protseduur hõlmab töölehtedele täpsemate reeglite loomise etappe. Muu hulgas hõlmab see töölehe juhtimise ja kasutaja sisestamispiirangute seadistamist. Protseduur kasutab demoettevõtte USMF andmeid.


## <a name="set-up-journal-control"></a>Töölehe juhtimise seadistamine
1. Minge jaotisesse Pearaamat > Töölehe seadistamine > Töölehtede nimed.
2. Otsige loendist ja valige soovitud kirje.
3. Klõpsake suvandit Töölehe juhtimine.
4. Klõpsake vahekaarti Lisa.
5. Klõpsake väljal Ettevõtte kontod otsingu avamiseks ripploendi nuppu.
6. Otsige loendist ja valige soovitud kirje.
7. Klõpsake loendis valitud real olevat linki.
8. Klõpsake vahekaarti Lisa.
9. Klõpsake väljal Konto struktuur otsingu avamiseks ripploendi nuppu.
10. Otsige loendist ja valige soovitud kirje.
11. Klõpsake loendis valitud real olevat linki.
12. Klõpsake väljal Segment otsingu avamiseks ripploendi nuppu.
13. Klõpsake loendis valitud real olevat linki.
14. Klõpsake väljal Alates väärtusest otsingu avamiseks ripploendi nuppu.
15. Otsige loendist ja valige soovitud kirje.
16. Klõpsake loendis valitud real olevat linki.
17. Klõpsake väljal Kuni väärtuseni otsingu avamiseks ripploendi nuppu.
18. Otsige loendist ja valige soovitud kirje.
19. Klõpsake loendis valitud real olevat linki.

## <a name="set-up-posting-restrictions"></a>Sisestuspiirangute seadistamine
1. Sulgege leht.
2. Klõpsake suvandit Sisestamispiirangud.
3. Valige jaotises Kuidas soovite seadistada sisestamispiiranguid? suvand Kasutajagrupi järgi.
4. Märkige puul ruut „grupp, millel lubate selle töölehe nime puhul sisestamist”.
5. Klõpsake nuppu OK.
