1. <span data-ttu-id="3f224-105">Avage Pearaamat > Töölehe sisestused > Päevaraamatud.</span><span class="sxs-lookup"><span data-stu-id="3f224-105">Go to General ledger > Journal entries > General journals.</span></span>
2. <span data-ttu-id="3f224-106">Leidke ja valige loendist soovitud tööleht või looge uus.</span><span class="sxs-lookup"><span data-stu-id="3f224-106">In the list, find and select the desired journal or create a new one.</span></span>
3. <span data-ttu-id="3f224-107">Klõpsake linki väljal Töölehe partiinumber.</span><span class="sxs-lookup"><span data-stu-id="3f224-107">Click to follow the link in the Journal batch number field.</span></span>
4. <span data-ttu-id="3f224-108">Märkige loendis valitud rida.</span><span class="sxs-lookup"><span data-stu-id="3f224-108">In the list, mark the selected row.</span></span>
5. <span data-ttu-id="3f224-109">Täpsustage soovitud väärtusi väljal Konto.</span><span class="sxs-lookup"><span data-stu-id="3f224-109">In the Account field, specify the desired values.</span></span>
    * <span data-ttu-id="3f224-110">Selles näites määratleme kindlustuse kulu.</span><span class="sxs-lookup"><span data-stu-id="3f224-110">In this example, we are defining the expense for the insurance.</span></span> <span data-ttu-id="3f224-111">See muutub perioodiliseks kulusummaks.</span><span class="sxs-lookup"><span data-stu-id="3f224-111">It will be come periodic expense amount.</span></span>  
6. <span data-ttu-id="3f224-112">Sisestage väljale Kirjeldus soovitud väärtus.</span><span class="sxs-lookup"><span data-stu-id="3f224-112">In the Description field, type a value.</span></span>
7. <span data-ttu-id="3f224-113">Sisestage arv väljale Deebet.</span><span class="sxs-lookup"><span data-stu-id="3f224-113">In the Debit field, enter a number.</span></span>
8. <span data-ttu-id="3f224-114">Täpsustage soovitud väärtusi väljal Vastaskonto.</span><span class="sxs-lookup"><span data-stu-id="3f224-114">In the Offset account field, specify the desired values.</span></span>
9. <span data-ttu-id="3f224-115">Klõpsake suvandit Funktsioonid.</span><span class="sxs-lookup"><span data-stu-id="3f224-115">Click Functions.</span></span>
10. <span data-ttu-id="3f224-116">Klõpsake suvandit Pearaamatu viitvõlad.</span><span class="sxs-lookup"><span data-stu-id="3f224-116">Click Ledger accruals.</span></span>
11. <span data-ttu-id="3f224-117">Klõpsake väljal Viitvõla ID otsingu avamiseks ripploendi nuppu.</span><span class="sxs-lookup"><span data-stu-id="3f224-117">In the Accrual identification field, click the drop-down button to open the lookup.</span></span>

13. <span data-ttu-id="3f224-119">Klõpsake loendis valitud real olevat linki.</span><span class="sxs-lookup"><span data-stu-id="3f224-119">In the list, click the link in the selected row.</span></span>
14. <span data-ttu-id="3f224-120">Sisestage kuupäev väljale Alguskuupäev.</span><span class="sxs-lookup"><span data-stu-id="3f224-120">In the Start date field, enter a date.</span></span>
15. <span data-ttu-id="3f224-121">Klõpsake suvandit Kanded.</span><span class="sxs-lookup"><span data-stu-id="3f224-121">Click Transactions.</span></span>
16. <span data-ttu-id="3f224-122">Sulgege leht.</span><span class="sxs-lookup"><span data-stu-id="3f224-122">Close the page.</span></span>
17. <span data-ttu-id="3f224-123">Klõpsake nuppu OK.</span><span class="sxs-lookup"><span data-stu-id="3f224-123">Click OK.</span></span>
18. <span data-ttu-id="3f224-124">Klõpsake valikut Sisesta.</span><span class="sxs-lookup"><span data-stu-id="3f224-124">Click Post.</span></span>
