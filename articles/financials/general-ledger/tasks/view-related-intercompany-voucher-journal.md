# <a name="view-related-intercompany-voucher-from-journal"></a>Seotud kontsernisisese kande kuvamine töölehelt

[!include [task guide banner](../../includes/task-guide-banner.md)]

Päevaraamatust kontsernisisese kande sisestamisel kuvatakse seotud kandeaknas vastaskonto ettevõttest pärinev kanne.


## <a name="post-an-intercompany-journal"></a>Kontsernisisese töölehe sisestamine
1. Avage jaotis Päevaraamatud.
2. Klõpsake valikut Uus.
3. Märkige loendis valitud rida.
4. Väljal Nimi sisestage või valige kontsernisisese töölehe nimi.
5. Klõpsake valikut Read.
6. Märkige loendis valitud rida.
7. Täpsustage soovitud väärtusi väljal Konto.
8. Valige või sisestage väärtus väljal Kirjeldus.
9. Sisestage väljale Kirjeldus soovitud väärtus.
10. Sulgege leht.
11. Sisestage arv väljale Deebet.
12. Väljal Vastaskonto ettevõte sisestage või valige vastaskonto ettevõte.
13. Sisestage või valige väärtus väljal Vastaskonto ettevõte.
14. Sulgege leht.
15. Täpsustage soovitud väärtusi väljal Vastaskonto.
16. Klõpsake valikut Sisesta.

## <a name="view-related-intercompany-voucher"></a>Seotud kontsernisisese kande kuvamine
1. Klõpsake suvandit Kanne.
2. Klõpsake suvandit Seotud kanded.
3. Märkige loendis valitud rida.
4. Klõpsake suvandit Kanne.
