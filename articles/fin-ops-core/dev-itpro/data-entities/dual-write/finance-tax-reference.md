# <a name="access-to-finance-and-tax-reference-data"></a>Juurdepääs finantsilistele ja maksude viiteandmetele

[!include [banner](../../includes/banner.md)]



Iga ettevõte töötab põhilise finantsandmete kogumiga, nt rahanduskalendriaastaga, ettevõtte tehingute sooritamise valuuta, ettevõtte sissetuleva ja väljamineva rahaga majandamiseks kasutatavad kontod, maksumäärad ja rahaülekanne. Neid andmeid säilitatakse rakendustes Finance and Operations. Kuid see on saadaval Common Data Service'is, et mudeljuhitud Microsoft Dynamics 365 rakendustes oleks ühine finants- ja maksuandmete allikas. Sel viisil on andmed ühtsed kogu äriökosüsteemis. 

Finants- ja maksuandmed integreeritakse järgmiste vastenduste abil:

+ [Integreeritud pearaamat](ledger-mapping.md)
+ [Integreeritud maksu koondandmed](tax-mapping.md)
