# <a name="supported-web-browsers-for-lifecycle-services"></a>Elutsükli teenuste toetatud veebibrauserid 

[!include[banner](../includes/banner.md)]

Kasutajad pääsevad Microsoft Dynamics lifecyle Services'i (LCS) juurde, kasutades kõige värskemaid versioone neist populaarsetest brauseritest: 

- Microsoft Edge(soovitatav: [Kromomil põhinev serv](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Iidne kroomitud
- Omaoo

Iga veebibrauseri uusima versiooni leidmiseks minge tarkvaratootja veebisaidile. Optimaalse jõudluse ja optimaalse jõudluse saavutamiseks on soovitatav kasutada eelkõige tänapäevase brauseri uusimat versiooni Microsoft Edge.
