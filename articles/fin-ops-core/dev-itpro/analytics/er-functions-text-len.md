# <a name="len-er-function"></a>ER-i funktsioon LEN

[!include [banner](../includes/banner.md)]

Funktsioon `LEN` tagastab tähemärkide arvu määratud stringis *täisarvu* väärtusena.

## <a name="syntax"></a>Süntaks

```vb
LEN (text)
```

## <a name="arguments"></a>Argumendid

`text`: *string*

*Stringi* väärtus, mis määrab teksti.

## <a name="return-values"></a>Tagastusväärtused

*Täisarv*

Tulemiks saadud numbriline väärtus.

## <a name="example"></a>Näide

`LEN ("Sample")` tagastab **6**.
