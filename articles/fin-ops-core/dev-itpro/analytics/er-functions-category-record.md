# <a name="list-of-er-functions-in-the-record-category"></a>ER-i funktsioonide loend kirje kategoorias

[!include [banner](../includes/banner.md)]

Elektroonilise aruandluse (ER) kirjete funktsioone saab kasutada andmete ekstraktimiseks ja toimingute tegemiseks andmetüübi *Konteiner (kirje)* andmeallikates. See teema sisaldab järgmiste funktsioonide kokkuvõtet.

## <a name="list-of-supported-functions"></a>Toetatud funktsioonide loend

| Funktsioon | Kirjeldus |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | See funktsioon tagastab *konteineri (kirje)* väärtuse null, millel on sama struktuur, kui määratud kirjete loendil või kirjel. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | See funktsioon tagastab *konteineri (kirje)* väärtuse null, millel on sama struktuur, kui määratud kirjete loendil või kirjel. |
