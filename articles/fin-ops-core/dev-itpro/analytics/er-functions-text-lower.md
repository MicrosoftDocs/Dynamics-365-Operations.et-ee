# <a name="lower-er-function"></a>ER-i funktsioon LOWER

[!include [banner](../includes/banner.md)]

Funktsioon `LOWER` tagastab *stringi* väärtusena määratud tekstistringi pärast selle teisendamist väiketäheliseks.

## <a name="syntax"></a>Süntaks

```vb
LOWER (text)
```

## <a name="arguments"></a>Argumendid

`text`: *string*

*Stringi* väärtus, mis määrab teksti.

## <a name="return-values"></a>Tagastusväärtused

*String*

Tulemiks saadud teksti väärtus.

## <a name="example"></a>Näide

`LOWER ("Sample")` tagastab tulemuse **„näidis”**.
