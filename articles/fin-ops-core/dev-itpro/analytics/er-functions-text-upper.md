# <a name="upper-er-function"></a>ER-i funktsioon UPPER

[!include [banner](../includes/banner.md)]

Funktsioon `UPPER` tagastab *stringi* väärtusena määratud tekstistringi pärast selle teisendamist suurtäheliseks.

## <a name="syntax"></a>Süntaks

```vb
UPPER (text )
```

## <a name="arguments"></a>Argumendid

`text`: *string*

Tüübi *String* andmeallika kehtiv tee.

## <a name="return-values"></a>Tagastusväärtused

*String*

Tulemiks saadud teksti väärtus.

## <a name="example"></a>Näide

`UPPER ("Sample")` tagastab tulemuse **„SAMPLE”**.
