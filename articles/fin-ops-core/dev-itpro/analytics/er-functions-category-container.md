# <a name="list-of-er-functions-in-the-container-category"></a>ER-i funktsioonide loend konteineri kategoorias

[!include [banner](../includes/banner.md)]

[Elektroonilise aruandluse (ER)](general-electronic-reporting.md) konteineri [funktsioone](er-formula-language.md#functions) saab kasutada toimingute tegemiseks, mis hõlmavad andmetüübi *Konteiner* andmeallikaid. Need toimingud leiavad aset, kui töötlemisandmed kajastavad binaarandmete kogumit binaarses suure objekti (BLOB) vormingus. See teema sisaldab järgmiste funktsioonide kokkuvõtet.

## <a name="list-of-supported-functions"></a>Toetatud funktsioonide loend

| Funktsioon | Kirjeldus |
|----------|-------------|
| [Base64StringToContainer](er-functions-container-base64stringtocontainer.md) | See funktsioon tagastab väärtuse *Konteiner*, mis koosneb binaarsest sisust, mis on dekodeeritud määratud ASCII-stringist, mis tähistab binaarsest tekstiks kodeerimisskeemide Base64 gruppi. |
