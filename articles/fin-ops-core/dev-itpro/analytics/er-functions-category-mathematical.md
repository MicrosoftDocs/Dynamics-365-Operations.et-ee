# <a name="list-of-er-functions-in-the-mathematical-category"></a>ER-i funktsioonide loend matemaatilises kategoorias

[!include [banner](../includes/banner.md)]

Elektroonilise aruandluse (ER) matemaatilisi funktsioone saab kasutada paljude tavaliste matemaatiliste arvutuste tegemiseks. See teema sisaldab järgmiste funktsioonide kokkuvõtet.

## <a name="list-of-supported-functions"></a>Toetatud funktsioonide loend

| Funktsioon | Kirjeldus |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | See funktsioon annab määratud numbri absoluutväärtuse *tegeliku* väärtusena. Teisisõnu tagastab see arvu ilma selle märgita. |
| [Võimsus](er-functions-mathematical-power.md)         | See funktsioon tagastab *tegeliku* väärtuse, mis kajastab määratud astmele määratud positiivse numbri tõstmise tulemi. |
| [Round](er-functions-mathematical-round.md)         | See funktsioon tagastab määratud numbri *tõese* väärtusena pärast selle ümardamist määratud arvu komakohtadeni. |
| [RoundDown](er-functions-mathematical-rounddown.md) | See funktsioon tagastab määratud numbri *tõese* väärtusena pärast selle ümardamist allapoole määratud arvu komakohtadeni. |
| [RoundUp](er-functions-mathematical-roundup.md)     | See funktsioon tagastab määratud numbri *tõese* väärtusena pärast selle ümardamist ülespoole määratud arvu komakohtadeni. |
