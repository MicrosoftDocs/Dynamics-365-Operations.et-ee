# <a name="change-the-banner-or-logo"></a><span data-ttu-id="02cce-103">Ribareklaami või logo muutmine</span><span class="sxs-lookup"><span data-stu-id="02cce-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="02cce-104">Järgmine protseduur loetleb toimingud, mida süsteemiadministraatorid saavad kasutada juriidilise isiku puhul kuvatava ribareklaami või logo pildi värskendamiseks.</span><span class="sxs-lookup"><span data-stu-id="02cce-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="02cce-105">Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid.</span><span class="sxs-lookup"><span data-stu-id="02cce-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="02cce-106">Avage **Navigeerimispaan > Moodulid > Organisatsiooni haldus > Organisatsioonid > Juriidilised isikud**.</span><span class="sxs-lookup"><span data-stu-id="02cce-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="02cce-107">Valige vasakul olevast loendist juriidiline isik, kelle ribareklaami või logo soovite uuendada.</span><span class="sxs-lookup"><span data-stu-id="02cce-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="02cce-108">Kui see on juba valitud, jätkake järgmise etapiga.</span><span class="sxs-lookup"><span data-stu-id="02cce-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="02cce-109">Laiendage vahekaart **Armatuurlaua pilt**.</span><span class="sxs-lookup"><span data-stu-id="02cce-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="02cce-110">Valige **Muuda**.</span><span class="sxs-lookup"><span data-stu-id="02cce-110">Select **Change**.</span></span>
    
    <span data-ttu-id="02cce-111">Ribareklaami ideaalne lahutusvõime on 1920 × 281 pikslit.</span><span class="sxs-lookup"><span data-stu-id="02cce-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="02cce-112">Logopildi ideaalne laius on 350 pikslit.</span><span class="sxs-lookup"><span data-stu-id="02cce-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="02cce-113">Valige käsk **Salvesta**.</span><span class="sxs-lookup"><span data-stu-id="02cce-113">Select **Save**.</span></span>
6. <span data-ttu-id="02cce-114">Avage **Navigeerimispaan > Moodulid > Üldine > Üldine > Vaikearmatuurlaud**.</span><span class="sxs-lookup"><span data-stu-id="02cce-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="02cce-115">Näete uut ribareklaami või logopilti armatuurlaual.</span><span class="sxs-lookup"><span data-stu-id="02cce-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="02cce-116">Võib-olla peate ettevõtte valija abiga valima praeguse ettevõtte asemel selle, mille jaoks ribareklaami üles laadisite.</span><span class="sxs-lookup"><span data-stu-id="02cce-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
