[!include [banner](../../includes/banner.md)]

<span data-ttu-id="ebf70-104">Kasutage seda protseduuri globaalse aadressiraamatu vaikeväärtuste ja turbepoliitikate määramiseks.</span><span class="sxs-lookup"><span data-stu-id="ebf70-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="ebf70-105">Selle tegevuse loomisel kasutati demoettevõtte USMF-i andmeid.</span><span class="sxs-lookup"><span data-stu-id="ebf70-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="ebf70-106">See ülesanne on mõeldud planeerimise ja konfigureerimise töörühmale.</span><span class="sxs-lookup"><span data-stu-id="ebf70-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="ebf70-107">Avage navigeerimispaanil **Moodulid > Organisatsiooni haldus > Globaalne aadressiraamat > Globaalse aadressiraamatu parameetrid**.</span><span class="sxs-lookup"><span data-stu-id="ebf70-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>

7. <span data-ttu-id="ebf70-113">Valige suvand väljal **Osapool** .</span><span class="sxs-lookup"><span data-stu-id="ebf70-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="ebf70-114">Valige suvand väljal **Klient** .</span><span class="sxs-lookup"><span data-stu-id="ebf70-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="ebf70-115">Valige suvand väljal **Hankija** .</span><span class="sxs-lookup"><span data-stu-id="ebf70-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="ebf70-116">Valige suvand väljal **Potentsiaalne klient** .</span><span class="sxs-lookup"><span data-stu-id="ebf70-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="ebf70-117">Valige suvand väljal **Konkurent** .</span><span class="sxs-lookup"><span data-stu-id="ebf70-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="ebf70-118">Klõpsake vahekaarti **Privaatse asukoha turve** .</span><span class="sxs-lookup"><span data-stu-id="ebf70-118">Click the **Private location security** tab.</span></span>
