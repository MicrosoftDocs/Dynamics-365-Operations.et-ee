# <a name="set-up-number-sequences-on-an-individual-basis"></a><span data-ttu-id="2603e-103">Individuaalsete numbriseeriate häälestamine</span><span class="sxs-lookup"><span data-stu-id="2603e-103">Set up number sequences on an individual basis</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="2603e-104">Selles teemas selgitatakse, kuidas seadistada individuaalseid numbriseeriaid.</span><span class="sxs-lookup"><span data-stu-id="2603e-104">This topic explains how to set up number sequences on an individual basis.</span></span> <span data-ttu-id="2603e-105">Numbriseeriaid kasutatakse loetavate ainuidentifikaatorite loomiseks koondandmetele ja kandekirjetele, mis neid nõuavad.</span><span class="sxs-lookup"><span data-stu-id="2603e-105">Number sequences are used to generate readable, unique identifiers for master data records and transaction records that require them.</span></span> <span data-ttu-id="2603e-106">Koondandmeid või kandekirjet, mis nõuavad identifikaatorit, nimetatakse viiteks.</span><span class="sxs-lookup"><span data-stu-id="2603e-106">A master data or transaction record that requires an identifier is referred to as a reference.</span></span> <span data-ttu-id="2603e-107">Enne kui saate viite jaoks uusi kirjeid luua, peate seadistama numbriseeria ja selle viitega siduma.</span><span class="sxs-lookup"><span data-stu-id="2603e-107">Before you can create new records for a reference, you must set up a number sequence and associate it with the reference.</span></span> <span data-ttu-id="2603e-108">Võite seadistada kõik vajalikud numbriseeriad samaaegselt, kasutades viisardit **Numbriseeriate seadistamine**, või võite luua või muuta üksikuid numbriseeriaid, kasutades lehte **Numbriseeriad**.</span><span class="sxs-lookup"><span data-stu-id="2603e-108">You can set up all required number sequences at the same time by using the **Set up number sequences** wizard, or you can create or modify individual number sequences by using the **Number sequences** page.</span></span>

1. <span data-ttu-id="2603e-109">Avage **Navigeerimispaan > Moodulid > Organisatsiooni haldus > Numbriseeriad > Numbriseeriad**.</span><span class="sxs-lookup"><span data-stu-id="2603e-109">Go to **Navigation pane > Modules > Organization administration > Number sequences > Number sequences**.</span></span>
2. <span data-ttu-id="2603e-110">Valige **Numbriseeria**.</span><span class="sxs-lookup"><span data-stu-id="2603e-110">Select **Number sequence**.</span></span>
3. <span data-ttu-id="2603e-111">Sisestage väärtus väljale **Numbriseeria kood**.</span><span class="sxs-lookup"><span data-stu-id="2603e-111">In the **Number sequence code** field, type a value.</span></span>
4. <span data-ttu-id="2603e-112">Sisestage väärtus väljale **Nimi**.</span><span class="sxs-lookup"><span data-stu-id="2603e-112">In the **Name** field, type a value.</span></span>

6. <span data-ttu-id="2603e-118">Laiendage jaotist **Segmendid**.</span><span class="sxs-lookup"><span data-stu-id="2603e-118">Expand the **Segments** section.</span></span>
    - <span data-ttu-id="2603e-119">Määratlege numbriseeriate vorming, lisades, eemaldades või teisaldades segmente.</span><span class="sxs-lookup"><span data-stu-id="2603e-119">Define the format for the number sequence by adding, removing, and rearranging segments.</span></span>  
    - <span data-ttu-id="2603e-120">Iga ulatusega numbriseeriad võivad sisaldada *Konstantseid segmente* ja *Tähtnumbrilisi segmente*.</span><span class="sxs-lookup"><span data-stu-id="2603e-120">Number sequences of all scopes can contain *Constant segments* and *Alphanumeric segments*.</span></span> <span data-ttu-id="2603e-121">Konstantsed segmendid sisaldavad muutumatut tähe- või numbrimärkide kogumit.</span><span class="sxs-lookup"><span data-stu-id="2603e-121">Constant segments contain a set of alphanumeric characters that do not change.</span></span> <span data-ttu-id="2603e-122">Selle segmendi tüübi abil saate lisada sidekriipsu või muu eraldaja numbriseeria segmentide vahele.</span><span class="sxs-lookup"><span data-stu-id="2603e-122">Use this segment type to add a hyphen or other separators between number sequence segments.</span></span> <span data-ttu-id="2603e-123">Tähtedest ja numbritest koosnevad segmendid sisaldavad numbrimärkide kombinatsiooni (#) ning ja-märke (&).</span><span class="sxs-lookup"><span data-stu-id="2603e-123">Alphanumeric segments contain a combination of number signs (#) and ampersands (&).</span></span> <span data-ttu-id="2603e-124">Need märgid tähistavad tähti ja numbreid, mis kasvavad iga kord, kui numbriseeria numbrit kasutatakse.</span><span class="sxs-lookup"><span data-stu-id="2603e-124">These characters represent letters and numbers that increment every time that a number from the sequence is used.</span></span> <span data-ttu-id="2603e-125">Kasutage trelle (#), et tähistada kasvavaid numbreid ning ja-märki (&), et tähistada kasvavaid tähti.</span><span class="sxs-lookup"><span data-stu-id="2603e-125">Use a number sign (#) to indicate incrementing numbers and an ampersand (&) to data-stu-id="2603e-126">For example, the format `#####_2014` creates the sequence `00001_2014`, `00002_2014`, and so on.</span></span> <span data-ttu-id="2603e-127">Vähemalt üks tähtedest ja data-ttu-id="2603e-128">Ulatuse segmendid, nagu ettevõte või juriidiline isik, ei ole nõutavad.</span><span class="sxs-lookup"><span data-stu-id="2603e-128">Scope segments, such as company or legal entity, are not required.</span></span> <span data-ttu-id="2603e-129">Kuid kui te ei kaasa ulatuse segmente vormingusse, luuakse iga ulatuse kohta ikkagi valitud viitele numbrid.</span><span class="sxs-lookup"><span data-stu-id="2603e-129">However, if you do not include scope segments in the format, numbers for the selected reference are still generated per scope.</span></span>  
7. <span data-ttu-id="2603e-130">Laiendage jaotist **Viited**.</span><span class="sxs-lookup"><span data-stu-id="2603e-130">Expand the **References** section.</span></span> <span data-ttu-id="2603e-131">Valige dokumendi tüüp või kirje, millele see numbriseeria määrata.</span><span class="sxs-lookup"><span data-stu-id="2603e-131">Select the document type or record to assign this number sequence to.</span></span> <span data-ttu-id="2603e-132">See etapp on valikuline järjestustele, mis on määratletud spetsiaalse rakendamise kasutusmustritele.</span><span class="sxs-lookup"><span data-stu-id="2603e-132">This step is optional for sequences that are defined for special application usage patterns.</span></span> <span data-ttu-id="2603e-133">Nende stsenaariumide puhul luuakse uus number, kasutades numbriseeriakoodi või ID väärtust, ilma viidet kasutamata.</span><span class="sxs-lookup"><span data-stu-id="2603e-133">In these scenarios, a new number is generated by using the value of a number sequence code or ID, without using a reference.</span></span> <span data-ttu-id="2603e-134">Näiteks spetsiaalse rakenduse kasutusmuster on kandeseeria, mida kasutatakse konkreetsete töölehenimede jaoks.</span><span class="sxs-lookup"><span data-stu-id="2603e-134">An example of a special application usage pattern is a voucher series that is used for specific journal names.</span></span> <span data-ttu-id="2603e-135">Aga me ei soovita teil selliseid mustreid kasutada.</span><span class="sxs-lookup"><span data-stu-id="2603e-135">However, we do not recommend that you use such patterns.</span></span>  
8. <span data-ttu-id="2603e-136">Laiendage jaotist **Üldine**.</span><span class="sxs-lookup"><span data-stu-id="2603e-136">Expand the **General** section.</span></span> <span data-ttu-id="2603e-137">Määrake kiirkaardil Üldine, kas numbriseeria on manuaalne ja pidev või katkev.</span><span class="sxs-lookup"><span data-stu-id="2603e-137">On the General FastTab, specify whether the number sequence is manual, and continuous or non-continuous.</span></span> <span data-ttu-id="2603e-138">Lisaks sisestage madalaimad ja kõrgeimad numbrid, mida numbriseerias saab kasutada.</span><span class="sxs-lookup"><span data-stu-id="2603e-138">In addition, enter the lowest and highest numbers that can be used in the number sequence.</span></span> <span data-ttu-id="2603e-139">Me ei soovita katkevat numbriseeriat muuta pidevaks numbriseeriaks.</span><span class="sxs-lookup"><span data-stu-id="2603e-139">We do not recommend changing a non-continuous number sequence to a continuous number sequence.</span></span> <span data-ttu-id="2603e-140">Numbriseeria ei saa olla täiesti pidev.</span><span class="sxs-lookup"><span data-stu-id="2603e-140">The number sequence will not be truly continuous.</span></span> <span data-ttu-id="2603e-141">See muutus võib põhjustada ka dublikaatvõtme rikkumisi andmebaasis.</span><span class="sxs-lookup"><span data-stu-id="2603e-141">This change may also cause duplicate key violations in the database.</span></span> <span data-ttu-id="2603e-142">Lisaks on pidevatel numbriseeriatel on suurem mõju tulemuslikkusele.</span><span class="sxs-lookup"><span data-stu-id="2603e-142">In addition, continuous number sequences have a larger effect on performance.</span></span>   
9. <span data-ttu-id="2603e-143">Klõpsake valikut **Salvesta**.</span><span class="sxs-lookup"><span data-stu-id="2603e-143">Click **Save**.</span></span>
