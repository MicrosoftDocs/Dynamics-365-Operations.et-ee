# <a name="set-a-users-preferred-time-zone"></a>Kasutaja eelistatud ajavööndi häälestus

[!include [banner](../../includes/banner.md)]


[!INCLUDE [PEAP](../../../../includes/peap-1.md)]

Järgmine artikkel selgitab, kuidas süsteemiadministraatori rollis kasutaja saab kasutajale ajavööndi seadistada.

1. Avage **Navigeerimispaan > Moodulid > Süsteemiadministraator > Kasutajad > Kasutajad**.
2. Otsige loendist ja valige soovitud kirje.
3. Valige **Kasutaja valikud**.
4. Valige vahekaart **Eelistused**.
5. Valige väljal **Ajavöönd** ripploendist suvand.
6. Valige käsk **Salvesta**.
