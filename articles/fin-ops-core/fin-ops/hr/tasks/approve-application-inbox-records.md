# <a name="approve-application-inbox-records"></a>Avalduste sisendkausta kirjete kinnitamine

[!include [banner](../../includes/banner.md)]

See protseduur näitab, kuidas vaadata üle lehtede Töötaja iseteenindus kaudu saadud avaldusi. Lisaks avalduste läbivaatamisele saate kinnitada valitud sisendkasti kirjete avaldusi. Avalduse sisendkausta kirjed tähistavad ettevõttele läbivaatamiseks edastatud kandideerimisavaldusi. Pärast kirje kinnitamist luuakse kandidaadi kirje isiku puhul, kes avalduse esitas. Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid.


## <a name="review-application-inbox-record"></a>Avalduste sisendkasti kirje ülevaatamine
1. Avage jaotis Inimressursid > Värbamine > Avaldused > Avalduste sisendkaust.
2. Otsige loendist ja valige soovitud kirje.
3. Laiendage jaotist Aadressid.
4. Laiendage jaotist Kontaktteave.
5. Laiendage jaotist Manused.

## <a name="approve-application-inbox-record"></a>Avalduste sisendkasti kirje kinnitamine
1. Märkige loendis valitud rida.
2. Märkige väljal Nimi olev väärtus edasiseks viiteks üles
3. Klõpsake nuppu Kinnita.
4. Klõpsake nuppu OK.
5. Sulgege leht.

## <a name="view-the-newly-created-application-record"></a>Äsja loodud avalduse kirje vaatamine
1. Avage Personaliarvestus > Värbamine > Avaldused > Avaldused.
2. Klõpsake loendis valitud real olevat linki.
3. Laiendage jaotist Manused.
