# <a name="configure-functionality-profiles-for-sales-representatives"></a>Müügiesindajate funktsiooniprofiilide konfigureerimine

[!include [task guide banner](../includes/task-guide-banner.md)]

See protseduur näitab, kuidas konfigureerida müügiesindajatele kehtivaid kaupluse funktsiooniprofiili sätteid. See protseduur kasutab demoettevõtte USRT andmeid.

1. Minge jaotisse Jaemüük ja kaubandus > Kanali häälestus > Kassa häälestus > Kassaprofiilid > Funktsiooniprofiilid.
2. Klõpsake nuppu Redigeeri.
3. Laiendage jaotist Funktsioonid.
    * Funktsiooniprofiili sätete abil saab konfigureerida kassat nii, et kassiiri vaike-müügigrupp lisatakse automaatselt müügigruppide viipade kuvamiseks ja müügigruppide nõudmiseks.  
4. Tehke valik väljal Müügiesindaja määramine.
5. Valige Jah väljal Nõua müügiesindajat.
