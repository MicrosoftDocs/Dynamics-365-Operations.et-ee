# <a name="create-financial-dimensions-for-retail-channels-and-configure-dimension-values-on-stores"></a>Jaemüügikanalite jaoks finantsdimensioonide loomine ja kauplustes dimensiooniväärtuste konfigureerimine

[!include [task guide banner](../includes/task-guide-banner.md)]

See protseduur selgitab jaemüügikanali finantsdimensiooni loomist dimensiooniväärtuste abil ja jaekauplustes finantsdimensiooni väärtuste konfigureerimise etappe. Teema ei sisalda muid seotud etappe, näiteks dimensioonikogumite ja kontostruktuuride loomist. Protseduur kasutab demoettevõtte USRT andmeid.

1. Avage Pearaamat > Kontoplaan > Dimensioonid > Finantsdimensioonid.
2. Klõpsake valikut Uus.
3. Valige väljal Kasuta väärtusi suvand Jaemüügikanalid.
4. Sisestage väärtus väljale Dimensiooni nimi.
5. Klõpsake käsku Aktiveeri.
6. Klõpsake valikut Sule.
7. Klõpsake käsku Aktiveeri.
8. Klõpsake suvandit Dimensiooniväärtused.
9. Sulgege leht.
10. Klõpsake nuppu Salvesta.
11. Sulgege leht.
12. Avage Jaemüük ja kaubandus > Kanalid > Jaemüügikauplused > Kõik jaemüügikauplused.
13. Klõpsake loendis valitud real olevat linki.
14. Laiendage jaotist Finantsdimensioonid.
15. Klõpsake nuppu Redigeeri.
16. Klõpsake väljal Retailchannel otsingu avamiseks ripploendi nuppu.
17. Leidke ja valige loendist värskendatava kaupluse dimensiooniväärtus.
18. Klõpsake loendis valitud real olevat linki.
19. Klõpsake väljal CostCenter otsingu avamiseks ripploendi nuppu.
20. Otsige loendist ja valige soovitud kirje.
21. Klõpsake loendis valitud real olevat linki.
22. Klõpsake väljal Osakond otsingu avamiseks ripploendi nuppu.
23. Otsige loendist ja valige soovitud kirje.
24. Klõpsake loendis valitud real olevat linki.
25. Klõpsake nuppu Salvesta.
