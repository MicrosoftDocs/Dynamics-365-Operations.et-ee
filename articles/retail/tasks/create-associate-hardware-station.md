# <a name="create-and-associate-a-hardware-station"></a> Riistvarajaama loomine ja seostamine

[!include[task guide banner](../includes/task-guide-banner.md)]

See protseduur selgitab uue riistvarajaama loomist. Luuakse uus riistvaraprofiil ja kasutatakse seda uute riistvarajaamade lisamiseks eelmääratletud kauplusse (kanalisse). Protseduur kasutab demoettevõtte USRT andmeid.

1. Avage Kaubanduse põhiandmed > Kanalid >... > .. > .. > Riistvarajaamade profiilid.
2. Klõpsake valikut Uus.
3. Sisestage väljale Riistvarajaama ID tekst „TestHWProfile”.
4. Sisestage väärtus väljale Nimi.
5. Sisestage number väljale Pordi number.
6. Klõpsake väljal Riistvaraprofiil otsingu avamiseks ripploendi nuppu.
7. Otsige loendist ja valige soovitud kirje.
8. Klõpsake loendis valitud real olevat linki.
9. Klõpsake väljal Paketi nimi otsingu avamiseks ripploendi nuppu.
10. Klõpsake loendis valitud real olevat linki.
    * See on uue keskkonnaga kaasnev standardne pakett. Versiooni seerianumber võib olla erinev.  
11. Klõpsake nuppu Salvesta.
12. Sulgege leht.
13. Avage Jaemüük ja kaubandus > Kanalid > Kõik jaekauplused.
14. Valige loendist rida 17.
    * Kui kasutate demoettevõtet USRT, on tegemist Houstoni kauplusega.  
15. Klõpsake loendis valitud real olevat linki.
16. Laiendage jaotist Riistvarajaamad.
17. Klõpsake vahekaarti Lisa.
18. Märkige loendis valitud rida.
19. Klõpsake väljal Profiili ID otsingu avamiseks ripploendi nuppu.
20. Otsige loendist ja valige soovitud kirje.
    * See peab olema eelmistes etappides loodud uue riistvarajaama profiil.  
21. Klõpsake loendis valitud real olevat linki.
22. Sisestage väärtus väljale Hosti nimi.
23. Sisestage väärtus väljale EFT terminal.
24. Klõpsake nuppu Salvesta.
