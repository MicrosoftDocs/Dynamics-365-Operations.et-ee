# <a name="manage-user-roles-in-microsoft-teams"></a>Kasutaja rollide haldamine rakenduses Microsoft Teams

[!include [banner](includes/banner.md)]

See teema kirjeldab Microsoft Dynamics 365 Commerce kasutajarollide haldamist rakenduses Microsoft Teams.

Teamsi iga grupi või kanali töörühma loomisel luuakse töörühmale vastav grupi liikmesus (nt `HOUSTON_D365@<YourTenantAzureADDomain>.com`). Kõigile töörühmagrupi liikmesuse alla määratud grupi töötajatele on määratud üks kahest kasutajarollist: **omanik** või **liige**. Grupi töötajad, kellel on **omaniku** kasutajaroll, saavad teha toiminguid, näiteks privaatse kanali lisamine ning liikmete lisamine või kustutamine. Tavaliselt on grupi juhil **omaniku** kasutajaroll.

Järgmine näide näitab töörühma liikmete ja nende kasutajarollide loendi näidet Microsoft Teams halduskeskuses.

![Töörühma liikmed ja kasutajarollid Microsoft Teams halduskeskuses](media/d365-commerce-teams-integration-user-roles.png)

Lisateavet vt jaotisest [Töörühma omanike ja liikmete Microsoft Teams määramine](https://docs.microsoft.com/microsoftteams/assign-roles-permissions).
