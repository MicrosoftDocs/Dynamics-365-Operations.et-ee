# <a name="set-up-rules-and-parameters-for-cross-docking-and-buyers-push"></a> Reeglite ja parameetrite seadistamine ristlaadimise ja kesklaost jaotuse jaoks

[!include [banner](../includes/banner.md)]

See protseduur tutvustab täiendamisreeglite loomise etappe. Täiendamisreegleid saab kasutada, et juhtida seda, kuidas tooted ristlaadimise ja kesklaost jaotusega kauplustesse jaotatakse. Täiendamisreegleid saab määrata kauplustele või kauplusegruppidele. Reegli igale reale määratud kaal määrab, kuidas toodete kogused kaupluste vahel jaotatakse, kui ristlaadimisel või kesklaost jaotamisel kasutatakse jaotamismeetodina täiendamisreegleid. See protsess kasutab demoettevõtte USRT-i andmeid.

1. Tehke valik Täiendamisreeglid.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Täiendamisreegel.
4. Sisestage väljale Kirjeldus soovitud väärtus.
5. Klõpsake nuppu Salvesta.
6. Klõpsake vahekaarti Lisa.
7. Märkige loendis valitud rida.
    * Saate selle tüübi jaoks valida Täiendamise hierarhia ja Kanali vahel. Väärtus juhib seda, kas Nime all tehtud valik on kanalite hierarhia või üks kindel kanal.  Selle näite puhul, jätke selle väärtuseks Täiendamise hierarhia.  
8. Valige väärtus väljal Nimi.
    * Kaalu vaikeväärtus tuletatakse laos täpsustatud kaalu põhjal.  Seda kaalu saab kasutada täiendamisreegli määramiseks või võite soovi korral sisestada väljale Kaal uue kaalu.  
9. Sisestage number väljale Kaal.
10. Klõpsake vahekaarti Lisa.
11. Märkige loendis valitud rida.
12. Tehke väljal Tüüp valik Kanal.
13. Sisestage või valige väärtus väljal Nimi.
14. Sisestage number väljale Kaal.
15. Klõpsake nuppu Salvesta.
