# <a name="configure-credit-card-processing"></a> Krediitkaardi töötlemise konfigureerimine

[!include [task guide banner](../includes/task-guide-banner.md)]

See protseduur selgitab, kuidas vaadata makseteenuste pakkujate loendit ja konfigureerida müügireskontro maksekontot. See protseduur kasutab demoettevõtte USRT andmed ning on mõeldud administraatoritele ja IT-spetsialistidele.


## <a name="view-a-list-of-payment-providers"></a>Makseteenuse pakkujate loendi kuvamine
1. Avage Müügireskontro > Maksete seadistus > Makseteenused.
2. Klõpsake suvandit Kuva saadaolevad pakkujad.

## <a name="configure-payment-account"></a>Maksekonto konfigureerimine
1. Klõpsake valikut Uus.
2. Sisestage väärtus väljale Makseteenus.
3. Valige suvand väljal Makse ülekandmine.
4. Laiendage jaotist Makseteenuse konto.
5. Sisestage väljale Keskkond: väärtus PROD.
6. Klõpsake suvandit Krediitkaarditüübid.
7. Klõpsake väljal Maksete tööleht otsingu avamiseks ripploendi nuppu.
8. Klõpsake loendis valitud real olevat linki.
9. Klõpsake vahekaarti Lisa.
10. Sisestage väärtus väljale Valuuta.
11. Otsige loendist ja valige soovitud kirje.
12. Klõpsake väljal Maksete tööleht otsingu avamiseks ripploendi nuppu.
13. Klõpsake loendis valitud real olevat linki.
14. Klõpsake vahekaarti Lisa.
15. Sisestage väärtus väljale Valuuta.
16. Otsige loendist ja valige soovitud kirje.
    * Saate neid etappe korrata nii paljude kaarditüüpide puhul kui vaja.  
17. Klõpsake väljal Maksete tööleht otsingu avamiseks ripploendi nuppu.
18. Klõpsake loendis valitud real olevat linki.
19. Klõpsake vahekaarti Lisa.
20. Sisestage väärtus väljale Valuuta.
21. Klõpsake nuppu Salvesta.
22. Sulgege leht.
23. Klõpsake suvandit Kinnita.
24. Märkige ruut Uute krediitkaartide vaiketöötleja.
25. Klõpsake nuppu Salvesta.
