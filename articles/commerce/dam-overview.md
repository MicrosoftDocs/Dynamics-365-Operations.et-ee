# <a name="digital-asset-management-overview"></a>Digitaalse varahalduse ülevaade

[!include [banner](includes/banner.md)]

Selle teema all antakse ülevaade digitaalsest varahaldusest rakenduses Microsoft Dynamics 365 Commerce saidiehitaja.

Dynamics 365 Commerce saidiehitaja meediumiteek sisaldab rikkalikku digitaalse varahalduse funktsionaalsust, sealhulgas järgmist tuge:
- Pildi varad
- Videovarad
- Muud tüüpi binaarsed varad, nt Office'i dokumendid ja PDF-failid
- Lokaliseeritud pildid
- Üleslaetud kaustad
- Piltide kärpimine
- Pildi fokaalpunktide kohandamine
- Omnikanali kujutisvarad (toode, toote variandid, kataloog, kategooria, töötaja ja kliendi pildid)

Digitaalsed varad on kanaliomased, välja arvatud omnikanali varad, mis kuuluvad vaikimisi kogu keskkonnale. 

Digitaalsed varad ja haldusfunktsioonid asuvad saidiehitaja **Meediumiteegis** saidi vasakpoolsel navigeerimispaanil.
