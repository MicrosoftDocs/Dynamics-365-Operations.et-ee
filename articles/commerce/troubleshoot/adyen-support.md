# <a name="troubleshoot-dynamics-365-payment-connector-for-adyen-issues"></a>Adyeni probleemide tõrkeotsing rakenduse Dynamics 365 Payment Connector

[!include [banner](../../includes/banner.md)]

See artikkel pakub tõrkeotsingu juhendeid, mis aitavad teil Microsoft Dynamics toe saada, kui teil on probleemid 365 Payment Connectoriga Puhvris.

## <a name="description"></a>Kirjeldus

Teil on Adyeni Dynamics 365 Payment Connectoriga probleeme järgmistes valdkondades ja vajate Adyeni meeskonna tuge:

- Müügikoha (POS), tänapäevase kassa (MPOS), kõnekeskuse või Dynamics 365 Commerce
- Adyeni makseteenuse pakkuja (PSP) viitenumber (nt, kui teil on küsimusi keeldumiste kohta, k.a käsitsi võtmesisestuse \[MKE\] kohta)
- Mis ei tööta katse- või reaalajas kaupmehekeskkonnas

## <a name="resolution"></a>Eraldusvõime

Kasutage järgmist e-kirja malli, et käivitada tugiprotsess Veebiteenuse meeskonnaga. Tõrkeotsingu kiirendamiseks veenduge, et meil sisaldab kõiki vajalikke üksikasju.

| Field        | Väärtus |
|--------------|-------|
| Sihtkoht:           | `support@adyen.com` |
| Koopia           | |
| Teemarida | Microsoft Dynamics toetaotlus |
| Keha         | <p>Tere tugi,</p><p>Esitage tugi järgmise probleemi puhul:</p><ul><li>Kaupmehe konto</li><li>Keskkond (Test/Prod)</li><li>Kanal (kassa/kõnekeskus/äri e-kaubandus)</li><li>KASSA viitenumber, kui väljaminek põhines konkreetsele kandele (SELLE viitenumbri leiate kviitungilt, Lülituse kliendialalt või kassaterminali kannete menüült.)</li><li>Veateate kuvatõmmis või foto(d), kui see on rakendatav</li><li>Sündmusevaaturi logid (.txt vormingus)</li><li>Proovitud väljastus- ja tõrkeotsinguetappide kirjeldus</li></ul> |
