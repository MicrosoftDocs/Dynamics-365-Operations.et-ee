# <a name="upload-files-other-than-images-and-videos"></a>Failide üleslaadimine, v.a pildid ja videod

[!include [banner](includes/banner.md)]

See artikkel kirjeldab, kuidas laadida üles muid faile kui pilte ja videoid saidikonstruktoris Microsoft Dynamics 365 Commerce.

Kaubanduse saidiehitaja meediumiteek toetab binaarvarade, v.a pildid ja videod, üleslaadimist. Näiteks võite soovida üles laadida Microsoft Excel, Microsoft Word, Microsoft PowerPointi faile või PDF-faile.

Toetatakse järgmisi dokumenditüüpe:
- 7Z
- AVI
- CS
- CSS
- DOC
- DOCX
- EPUB
- GIF
- INDD
- JAR
- JPG
- JPEG
- JS
- MP3
- MP4
- MPEG
- MPG
- ODP
- ODS
- ODT
- PDF
- PNG
- PPT
- PPTX
- PS
- QXP
- RAR
- RTF
- SVG
- TAR
- TGZ
- TXT
- WMV
- XLS
- XLSX
- XML
- ZIP

## <a name="upload-a-file"></a>Laadi fail üles

Faili üleslaadimiseks Kaubanduse saidiehitajasse toimige järgmiselt.

1. Valige vasakpoolsel navigeerimispaanil suvand **Meediumiteek**.
1. Valige tegumiriba käsk **Laadi üles \> Laadi üles meediumiüksused**.
1. Valige File Exploreris üks või mitu faili ja seejärel valige **Ava**.
1. Sisestage dialoogiboksis **Meediumiüksuse üleslaadimine** pealkiri, kirjeldus ja vajaduse korral märksõnade metaandmed.
1. Failide avaldamiseks kohe pärast üleslaadimist märkige ruut **Avalda meediumiüksused üksused pärast üleslaadimist**.
1. Valige nupp **OK**.
