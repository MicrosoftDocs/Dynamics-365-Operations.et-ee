Lisateavet klienditeeninduse KoguMChanneli kohta vt [klienditeeninduse klienditeeninduse sissejuhatusest](/dynamics365/customer-service/introduction-omnichannel). Lisateavet Selle kohta, kuidas Integreerida Dllchannel Dynamics 365 Commerce klienditeeninduse jaoks e-äri veebisaidiga, [vt Commerce’i vestlust klienditeeninduse mooduliga Koos Klienditeeninduse mooduliga](commerce-chat-module.md).

## <a name="power-virtual-agents"></a>Power Virtual Agents

Power Virtual Agents võimaldab teil luua hulgaks päringute vahemikku võimas aI-toidelised vestlusvestlused. Lisateavet vt Power Virtual Agents [Power Virtual Agents teemast Ülevaade](/power-virtual-agents/fundamentals-what-is-power-virtual-agents). Lisateavet selle kohta, kuidas integreerida Power Virtual Agents Dynamics 365 Commerce e-äri veebisaidiga, vt Commerce’i [vestlust Microsoftiga Power Virtual Agents](chat-module-pva.md).

> [!NOTE]
> Power Virtual Agents on juba kaasatud klienditeeninduse kanalisse Kliendi. Kui teil on vaja nii virtuaalset agenti (vestluskanalit) kui ka reaalajas agendi võimalusi, kasutage klienditeeninduse jaoks kliendikanalit.
