# <a name="crop-images"></a><span data-ttu-id="efce7-103">Piltide kärpimine</span><span class="sxs-lookup"><span data-stu-id="efce7-103">Crop images</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="efce7-104">Selle teema all kirjeldatakse, kuidas üles laadida videosid rakenduse Microsoft Dynamics 365 Commerce saidiehituses.</span><span class="sxs-lookup"><span data-stu-id="efce7-104">This topic describes how to crop images in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="efce7-105">Kaubanduse saidiehitaja meediumiteek võimaldab teil pilte kärpida optimeerimiseks erinevatele moodulitüüpidele ja vaateportidele.</span><span class="sxs-lookup"><span data-stu-id="efce7-105">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="efce7-106">Kärbi pilti</span><span class="sxs-lookup"><span data-stu-id="efce7-106">Crop an image</span></span>

<span data-ttu-id="efce7-107">Pildi kärpimiseks saidiehitajas toimige järgmiselt.</span><span class="sxs-lookup"><span data-stu-id="efce7-107">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="efce7-108">Valige Kaubanduse saidiehitaja vasakpoolsel navigeerimispaanil suvand **Meediumiteek**.</span><span class="sxs-lookup"><span data-stu-id="efce7-108">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="efce7-109">Põhiaknas valige pilt, mida soovite muuta.</span><span class="sxs-lookup"><span data-stu-id="efce7-109">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="efce7-110">Klõpsake käsuribal käsku **Redigeeri**.</span><span class="sxs-lookup"><span data-stu-id="efce7-110">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="efce7-111">Valige pilt, et siseneda **Redigeerimisrežiimi**.</span><span class="sxs-lookup"><span data-stu-id="efce7-111">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="efce7-112">Jaotises **Redigeerimisrežiim** valige **Redigeerimisvaade mooduli kaupa**.</span><span class="sxs-lookup"><span data-stu-id="efce7-112">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="efce7-113">Rippmenüüst **Moodul** valige mooduli tüüp.</span><span class="sxs-lookup"><span data-stu-id="efce7-113">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="efce7-114">Rippmenüüst **Vaatetüüp** valige vaate tüüp.</span><span class="sxs-lookup"><span data-stu-id="efce7-114">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="efce7-115">Rippmenüüst **Paigutus** valige pildi paigutus.</span><span class="sxs-lookup"><span data-stu-id="efce7-115">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="efce7-116">Rippmenüüst **Vaateport** valige vaatepordi suurus.</span><span class="sxs-lookup"><span data-stu-id="efce7-116">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="efce7-117">Pilt katab piirkonda, mis esindab kärbitavat ala.</span><span class="sxs-lookup"><span data-stu-id="efce7-117">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="efce7-118">Teisaldage ja muutke kärbitavat ala vastavalt vajadusele.</span><span class="sxs-lookup"><span data-stu-id="efce7-118">Move and resize the crop region as needed.</span></span> <span data-ttu-id="efce7-119">Kuvasuhe säilitatakse automaatselt.</span><span class="sxs-lookup"><span data-stu-id="efce7-119">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="efce7-120">Kui olete lõpetanud, valige tegumiribal käsk **Salvesta** ja seejärel valige **Lõpeta redigeerimine**.</span><span class="sxs-lookup"><span data-stu-id="efce7-120">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="efce7-121">Kui kohandatud kärpimine on lõpule viidud, jõustuvad pildimuudatused peaaegu kohe.</span><span class="sxs-lookup"><span data-stu-id="efce7-121">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
