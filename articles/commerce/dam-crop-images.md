## <a name="overview"></a><span data-ttu-id="0071e-105">Ülevaade</span><span class="sxs-lookup"><span data-stu-id="0071e-105">Overview</span></span>

<span data-ttu-id="0071e-106">Kaubanduse saidiehitaja meediumiteek võimaldab teil pilte kärpida optimeerimiseks erinevatele moodulitüüpidele ja vaateportidele.</span><span class="sxs-lookup"><span data-stu-id="0071e-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="0071e-107">Kärbi pilti</span><span class="sxs-lookup"><span data-stu-id="0071e-107">Crop an image</span></span>

<span data-ttu-id="0071e-108">Pildi kärpimiseks saidiehitajas toimige järgmiselt.</span><span class="sxs-lookup"><span data-stu-id="0071e-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="0071e-109">Valige Kaubanduse saidiehitaja vasakpoolsel navigeerimispaanil suvand **Meediumiteek**.</span><span class="sxs-lookup"><span data-stu-id="0071e-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="0071e-110">Põhiaknas valige pilt, mida soovite muuta.</span><span class="sxs-lookup"><span data-stu-id="0071e-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="0071e-111">Klõpsake käsuribal käsku **Redigeeri**.</span><span class="sxs-lookup"><span data-stu-id="0071e-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="0071e-112">Valige pilt, et siseneda **Redigeerimisrežiimi**.</span><span class="sxs-lookup"><span data-stu-id="0071e-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="0071e-113">Jaotises **Redigeerimisrežiim** valige **Redigeerimisvaade mooduli kaupa**.</span><span class="sxs-lookup"><span data-stu-id="0071e-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="0071e-114">Rippmenüüst **Moodul** valige mooduli tüüp.</span><span class="sxs-lookup"><span data-stu-id="0071e-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="0071e-115">Rippmenüüst **Vaatetüüp** valige vaate tüüp.</span><span class="sxs-lookup"><span data-stu-id="0071e-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="0071e-116">Rippmenüüst **Paigutus** valige pildi paigutus.</span><span class="sxs-lookup"><span data-stu-id="0071e-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="0071e-117">Rippmenüüst **Vaateport** valige vaatepordi suurus.</span><span class="sxs-lookup"><span data-stu-id="0071e-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="0071e-118">Pilt katab piirkonda, mis esindab kärbitavat ala.</span><span class="sxs-lookup"><span data-stu-id="0071e-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="0071e-119">Teisaldage ja muutke kärbitavat ala vastavalt vajadusele.</span><span class="sxs-lookup"><span data-stu-id="0071e-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="0071e-120">Kuvasuhe säilitatakse automaatselt.</span><span class="sxs-lookup"><span data-stu-id="0071e-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="0071e-121">Kui olete lõpetanud, valige tegumiribal käsk **Salvesta** ja seejärel valige **Lõpeta redigeerimine**.</span><span class="sxs-lookup"><span data-stu-id="0071e-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="0071e-122">Kui kohandatud kärpimine on lõpule viidud, jõustuvad pildimuudatused peaaegu kohe.</span><span class="sxs-lookup"><span data-stu-id="0071e-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
