# <a name="ways-to-add-content"></a>Sisu lisamise viisid

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

See teema annab teavet selle kohta, kuidas rakenduse Microsoft Dynamics 365 Commerce saidil lisada ja hallata sisu.

## <a name="overview"></a>Ülevaade

Saidi välimuse, olemuse ja sisu muutmiseks on mitmeid viise. Vajalikust kohandamise tasemest sõltuvalt saavad paljusid neid muudatusi rakendada mittearendajast kasutajad. Näiteks ei ole mallide koostamiseks, teemade valimiseks ja moodulite valimiseks ning konfigureerimiseks koodi kirjutamine vajalik. Seevastu on arendamise oskused vajalikud uue kujunduse või mooduli loomiseks, kuna kasutada tuleb e-kaubanduse tarkvara arendusekomplekti (SDK) ja Microsoft Dynamicsi teenuse Lifecycle Services (LCS) juurutamise töövoogu.

Järgmised teemad annavad üksikasjalikku teavet selle kohta, kuidas lisada ja hallata saidi sisu. Need keskenduvad teie saidi aladele, mille jaoks arendajat ei ole vaja. Vastavalt vajadusele viitavad need ülesannetele, mis nõuavad SDK tööd.

- Olemasoleva saidi lehe teksti, piltide või video muutmise kohta vaadake teemat [Moodulitega töötamine](work-with-modules.md).
- Veebisisu autorite eksituskindla kaubamärgipõhise autorluskogemuse tagamiseks vaadake teemat [Mallidega töötamine](work-with-templates.md).
- Saidi lehe jaotiste ümberkorraldamise kohta vaadake teemat [Paigutustega töötamine](work-with-layouts.md).
- Saidi lehtede fontide, värvide ja üldise välimuse muutmise kohta vaadake teemat [Saidi teema valimine](select-site-theme.md).
