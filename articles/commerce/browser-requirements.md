# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Brauserinõuded Dynamics 365 Commerce'i saitidele

[!include [banner](includes/banner.md)]

Selles teemas käsitletakse Microsoft Dynamics 365 Commerce'i saite puudutavaid brauseri nõudeid.

Dynamics 365 Commerce'i saite toetavad kõige paremini järgmised veebibrauserid. Iga veebibrauseri uusima versiooni leidmiseks minge tarkvaratootja veebisaidile.

- [Microsoft Edge](https://www.microsoft.com/edge) (kuni praeguse versioonini miinus 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (kuni praeguse versioonini miinus 1)
- [Google Chrome](https://www.google.com/chrome) (kuni praeguse versioonini miinus 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (kuni praeguse versioonini miinus 1)

> [!NOTE]
> Internet Explorerit ei toetata. Kui kasutate Internet Explorerit, ei pruugi mõni funktsioon toimida ja kasutamiskogemus võib olla puudulik.
