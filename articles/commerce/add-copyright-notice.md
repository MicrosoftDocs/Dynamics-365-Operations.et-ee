## <a name="add-a-copyright-notice"></a><span data-ttu-id="acfcd-109">Autoriõiguste teatise lisamine</span><span class="sxs-lookup"><span data-stu-id="acfcd-109">Add a copyright notice</span></span>

<span data-ttu-id="acfcd-110">Iga konkreetset malli kasutava lehe allossa autoriõiguse teatise lisamiseks toimige järgmiselt.</span><span class="sxs-lookup"><span data-stu-id="acfcd-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="acfcd-111">Avage **Fragmendid** ja valige seejärel **Uus**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-111">Go to **Fragments**, and then select **New**.</span></span>
1. <span data-ttu-id="acfcd-112">Valige dialoogiboksis **Uus fragment** moodul **Jalus** ja pange fragmendile nimi.</span><span class="sxs-lookup"><span data-stu-id="acfcd-112">In the **New fragment** dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="acfcd-113">Sisestage näiteks **Jalus-autoriõigus**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="acfcd-114">Valige nupp **OK**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-114">Select **OK**.</span></span>
1. <span data-ttu-id="acfcd-115">Navigeerimispaanil valige kolmikpunkti nupp (**...**) suvandi **Jalus** kõrval ja valige seejärel **Lisa moodul**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="acfcd-116">Dialoogiaknas valige suvand **Jaluse kategooria** ja seejärel valige nupp **OK**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="acfcd-117">Navigeerimispaanil valige kolmikpunkti nupp suvandi **Jaluse kategooria** kõrval ja valige seejärel **Lisa moodul**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="acfcd-118">Dialoogiaknas valige suvand **Tekstiplokk** ja seejärel valige nupp **OK**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-118">In the dialog box, select **Text block**, and then select **OK**.</span></span>
1. <span data-ttu-id="acfcd-119">Valige navigeerimispaanil **Tekstiplokk**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-119">In the navigation pane, select **Text block**.</span></span>
1. <span data-ttu-id="acfcd-120">Lisage paremal atribuutide paanil väljale **Lõik** oma autoriõiguse sõnum.</span><span class="sxs-lookup"><span data-stu-id="acfcd-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="acfcd-121">Sisestage näiteks **(C) Fabrikam 2019**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="acfcd-122">Valige **Salvesta**, valige **Lõpeta redigeerimine** ja valige seejärel **Avalda**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-122">Select **Save**, select **Finish editing**, and then select **Publish**.</span></span>
1. <span data-ttu-id="acfcd-123">Valige suvand **Mallid**, valige mall ja seejärel käsk **Redigeeri**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-123">Go to **Templates**, select the template, and then select **Edit**.</span></span>
1. <span data-ttu-id="acfcd-124">Jaotises **Lehe liigendus** laiendage suvandit **Kehatekst** ja laiendage seejärel suvandit **Vaikeleht**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="acfcd-125">Valige suvandi **Jaluse pesa** kõrval kolmikpunkti nupp ja seejärel valige käsk **Lisa fragment**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="acfcd-126">Valige varem loodud fragment ja seejärel valige käsk **Vali**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="acfcd-127">Valige malli registreerimiseks **Lõpeta redigeerimine** ja seejärel selle avaldamiseks **Avalda**.</span><span class="sxs-lookup"><span data-stu-id="acfcd-127">Select **Finish editing** to check in the template, and then select **Publish** to publish it.</span></span>

<span data-ttu-id="acfcd-128">Autoriõiguse teatist sisaldav jalus kuvatakse automaatselt kõigi valitud malli kasutavate lehtede allosas.</span><span class="sxs-lookup"><span data-stu-id="acfcd-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
