## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="50af0-108">Exceli töövihiku töölehele väljade lisamine</span><span class="sxs-lookup"><span data-stu-id="50af0-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="50af0-109">Jaemüügikannete redigeerimiseks Exceli töövihikusse väljade lisamiseks toimige järgmiselt.</span><span class="sxs-lookup"><span data-stu-id="50af0-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="50af0-110">Laadige alla ja avage Exceli fail lehelt **Väljavõtted**, lehelt **Jaemüügikanded** või tööruumi **Kaupluse finantsandmed** paanilt **Kande kinnitamise tõrked**.</span><span class="sxs-lookup"><span data-stu-id="50af0-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="50af0-111">Valige **Kujundus**.</span><span class="sxs-lookup"><span data-stu-id="50af0-111">Select **Design**.</span></span>
1. <span data-ttu-id="50af0-112">Valige soovitud tabeli pliiatsisümbol ning seejärel leidke ja valige saadaolevate väljade loendist väli, mida soovite lisada.</span><span class="sxs-lookup"><span data-stu-id="50af0-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>
