# <a name="axerd-retired"></a>AxErd eemaldatud

Kahjuks pidime eemaldama AxErdi veebilehe, kus esitati mitme Microsoft Dynamics AX 2012 tabeli kohta olemite seoste skeeme. Oleme järgmisesse GitHubi hoidlasse saanud sisestada nii projekti lähtekoodi, kui ka veebilehe HTML-failid: https://github.com/Microsoft/ax-2012-doc-tools

HTML-lehed on hoidlas, aadressil: [https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Default.htm](https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Default.htm)

Kirjeldus selle kohta, kuidas ise olemite seoste skeeme luua, on leitav aadressil: [https://github.com/Microsoft/ax-2012-doc-tools/blob/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Help-HowToUse-AxErd.htm](https://github.com/Microsoft/ax-2012-doc-tools/blob/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Help-HowToUse-AxErd.htm)

Lahendus on AxErdi kausta juurkataloogis: [https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd](https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd).
