# <a name="change-the-banner-or-logo"></a><span data-ttu-id="59170-103">Ribareklaami või logo muutmine</span><span class="sxs-lookup"><span data-stu-id="59170-103">Change the banner or logo</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="59170-104">Järgmine protseduur loetleb toimingud, mida süsteemiadministraatorid saavad kasutada juriidilise isiku puhul kuvatava ribareklaami või logo pildi värskendamiseks.</span><span class="sxs-lookup"><span data-stu-id="59170-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="59170-105">Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid.</span><span class="sxs-lookup"><span data-stu-id="59170-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="59170-106">Avage Organisatsiooni haldus > Organisatsioonid > Juriidilised isikud.</span><span class="sxs-lookup"><span data-stu-id="59170-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="59170-107">Valige vasakul olevast loendist juriidiline isik, kelle ribareklaami või logo soovite uuendada.</span><span class="sxs-lookup"><span data-stu-id="59170-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="59170-108">Kui see on juba valitud, jätkake järgmise etapiga.</span><span class="sxs-lookup"><span data-stu-id="59170-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="59170-109">Laiendage jaotist Pildid.</span><span class="sxs-lookup"><span data-stu-id="59170-109">Expand the Images section.</span></span> <span data-ttu-id="59170-110">Kui see on juba laiendatud, jätkake järgmise etapiga.</span><span class="sxs-lookup"><span data-stu-id="59170-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="59170-111">Klõpsake nuppu Redigeeri.</span><span class="sxs-lookup"><span data-stu-id="59170-111">Click Edit.</span></span>
5. <span data-ttu-id="59170-112">Valige väljal Armatuurlaua ettevõtte pildi tüüp suvand „Ribareklaam”.</span><span class="sxs-lookup"><span data-stu-id="59170-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="59170-113">Klõpsake käsku Muuda.</span><span class="sxs-lookup"><span data-stu-id="59170-113">Click Change.</span></span>
    * <span data-ttu-id="59170-114">Nõuanne: ribareklaami ideaalne lahutusvõime on 1920 × 281 pikslit.</span><span class="sxs-lookup"><span data-stu-id="59170-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="59170-115">Logopildi ideaalne laius on 350 pikslit.</span><span class="sxs-lookup"><span data-stu-id="59170-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="59170-116">Pärast üleslaadimise lõppu näete uut ribareklaami või logopilti jaotises Armatuurlaua pilt.</span><span class="sxs-lookup"><span data-stu-id="59170-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="59170-117">Klõpsake nuppu Salvesta.</span><span class="sxs-lookup"><span data-stu-id="59170-117">Click Save.</span></span>
8. <span data-ttu-id="59170-118">Avage jaotis Üldine > Üldine > Vaikearmatuurlaud.</span><span class="sxs-lookup"><span data-stu-id="59170-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="59170-119">Näete uut ribareklaami või logopilti armatuurlaual.</span><span class="sxs-lookup"><span data-stu-id="59170-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="59170-120">Võib-olla peate ettevõtte valija abiga valima praeguse ettevõtte asemel selle, mille jaoks ribareklaami üles laadisite.</span><span class="sxs-lookup"><span data-stu-id="59170-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
