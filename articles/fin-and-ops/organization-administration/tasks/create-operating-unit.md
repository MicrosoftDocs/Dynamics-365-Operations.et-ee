[!include [task guide banner](../../includes/task-guide-banner.md)]

    * <span data-ttu-id="1c291-112">Saate valida loodava tootmisüksuse tüübi.</span><span class="sxs-lookup"><span data-stu-id="1c291-112">Select the type of operating unit you want to create.</span></span>

    * <span data-ttu-id="1c291-115">Vajaduse korral laiendage jaotist Üldine.</span><span class="sxs-lookup"><span data-stu-id="1c291-115">Expand the General section, if necessary.</span></span>
