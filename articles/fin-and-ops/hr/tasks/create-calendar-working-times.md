# <a name="create-a-calendar-and-generate-working-times"></a>Kalendri loomine ja tööaegade genereerimine

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

Kalendrid kirjeldavad operatsiooniressursside võimsust ja tööaegu. Selle protseduuri abil saate määratleda töökalendri tööajamalli põhjal. Saate selle protseduuriga tutvuda demoettevõtte USMF-i või omaenda andmeid kasutades.

1. Avage Kõik tööruumid > Ressursi elutsükli haldus.
2. Klõpsake valikut Kalendrid.
3. Klõpsake valikut Uus.
4. Sisestage väärtus väljale Kalender.
    * See on selle kalendri ID, mida kasutatakse viitena kalendrite määramisel näiteks operatsiooniressursile või ressursigrupile.  
5. Sisestage väärtus väljale Nimi.
6. Sisestage number väljale Standardne tööpäev tundides.
7. Märkige loendis valitud rida.
8. Klõpsake valikut Tööajad.
9. Klõpsake valikut Koosta tööajad.
    * Looge töötunnid iga päeva kohta perioodil, millal on vaja tööd plaanida. Aja möödudes saate luua tööaegu täiendavatele perioodidele.  
10. Sisestage kuupäev väljale Alguskuupäev.
    * See on esimene päev, millal see kalender peab olema avatud.  
11. Sisestage kuupäev väljale Lõpukuupäev.
    * See on viimane päev, millal see kalender avatud on.  
12. Valige või sisestage väärtus väljal Tööajamall.
    * Tööajamall määratleb iga nädalapäeva töötunnid.  
13. Klõpsake nuppu OK.
14. Sulgege leht.
