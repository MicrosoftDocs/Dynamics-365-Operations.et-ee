# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>Töölehele lisatud uusi märkmeid ei saa näidata komplekteerimislehe aruandes

KB number: 4611660

## <a name="symptoms"></a>Sümptomid

Töölehele lisatud uusi märkmeid ei saa näidata komplekteerimislehe aruandes.

## <a name="resolution"></a>Eraldusvõime

Loendi aruanne näitab ainult müügitellimuse rea ja müügitellimuse päisega seotud märkuseid. See ei näita uusi arve töölehele lisatud märkusi.
