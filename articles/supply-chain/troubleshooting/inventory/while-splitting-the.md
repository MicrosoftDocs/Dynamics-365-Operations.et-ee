# <a name="when-a-catch-weight-quantity-is-split-minimum-quantity-is-used-instead-of-nominal-quantity"></a>Tegeliku kaalu koguse jaotamisel kasutatakse nimikoguse asemel miinimumkogust

KB-number: 4612073

## <a name="symptoms"></a>Sümptomid

Tegeliku kaaluga koguse tükeldamisel partiiks kasutab välja **Komplekteeritud kogus** kaubale määratud miinimumkogust, mitte nominaalkogust.

## <a name="resolution"></a>Eraldusvõime

Süsteem käitub kavandatud viisil. Süsteem kasutab komplekteerimiseks iga kauba minimaalse koguse seadistust.
