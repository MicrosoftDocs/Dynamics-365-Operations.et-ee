# <a name="the-last-tested-date-field-isnt-updated-when-multiple-quality-orders-are-created"></a>Välja Viimase testimise kuupäev ei uuendata mitme kvaliteettellimuse loomisel

KB number: 4612803

## <a name="symptoms"></a>Sümptomid

Välja **Viimase testimise** kuupäev ei uuendata mitme kvaliteettellimuse loomisel.

## <a name="resolution"></a>Eraldusvõime

Süsteem käitub kavandatud viisil. Viimane testitud kuupäev ei ole seotud kvaliteettellimustega. See säilitab kuupäeva, mil lõpetatud kaubad esimest korda osteti või toodeti. Seda kuupäeva kasutatakse kõlblikkusaja nõuande kuupäeva arvutamiseks.
