# <a name="cant-cancel-an-intercompany-purchase-order-thats-linked-to-a-sales-order"></a>Ei saa tühistada kontsernisisest ostutellimust, mis on seotud müügitellimusega

## <a name="symptoms"></a>Sümptomid

Kui proovite tühistada müügitellimusega seotud kontsernisisest ostutellimust, võite saada järgmise tõrketeate:

> Kogust ei saa vähendada, kuna järelejääv uuendamiskogus muudab märki.

## <a name="resolution"></a>Lahendus

See probleem parandati rakenduse Microsoft Dynamics 365 Supply Chain Management versioonis 10.0.13. Selles ja uuemates versioonides saate nüüd tühistada müügitellimusega seotud kontsernisiseseid ostutellimusi.
