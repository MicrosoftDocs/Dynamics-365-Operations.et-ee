# <a name="shipment-for-load-has-been-confirmed-but-no-lines-are-posted"></a>Koorma saadetis on kinnitatud, kuid ühtegi rida ei sisestata

## <a name="symptoms"></a>Sümptomid

Väljaminevate laotoimingutega töötamisel võite saada järgmise tõrketeate:

> Saadetis 1% koormuse jaoks on kinnitatud.

Edasist sisestamist siiski ei toimunud.

## <a name="resolution"></a>Lahendus

Selline käitumine on nii kavandatud. Saatmise kinnitus ei mõjuta sisestamist. See lihtsalt uuendab saadetise ja koorma olekut. Sisestamine peab toimuma eraldi protsessis.
