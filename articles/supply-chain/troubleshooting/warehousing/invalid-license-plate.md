# <a name="invalid-license-plate-error-when-scanning-license-plate-id-in-the-mobile-app"></a>Kehtetu litsentsiplaadi tõrge ID skannimisel mobiilirakenduses

## <a name="symptoms"></a>Sümptomid

Kui skaneerite identifitseerimisnumbri mobiilirakenduses Warehouse Management, võidakse kuvada see tõrketeade:

> Kehtetu litsentsiplaat.

## <a name="resolution"></a>Lahendus

Veenduge, et identifitseerimisnumbri ID on olemas identifitseerimisnumbrite tabelis ja et identifitseerimisnumbril olevad kaubad ja kogused on saadaval (ehk need ei ole blokeeritud).
