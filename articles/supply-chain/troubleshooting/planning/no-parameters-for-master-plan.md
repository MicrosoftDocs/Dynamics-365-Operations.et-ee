# <a name="parameters-for-the-master-plan-dont-exist"></a>Koondplaani parameetreid pole olemas

Tõrkekood: SYS25368

## <a name="symptoms"></a>Sümptomid

Kui proovite plaanitud tellimust kindlustada, kuvatakse järgmine tõrketeade:

> Parameetreid koondplaanile %1 pole olemas.

## <a name="cause"></a>Põhjus

Konfiguratsiooniprobleemide tõttu ei leia süsteem määratud plaani sätteid.

## <a name="resolution"></a>Lahendus

- Minge **Koondplaneerimine \> Seadistamine \> Plaanid \> Kloonplaanid** ja veenduge, et määratud nimega plaan oleks olemas.
