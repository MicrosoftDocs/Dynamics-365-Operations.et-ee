# <a name="report-as-finished-update-fails-with-a-missing-quantity-error"></a>Lõpetatuna kinnitamise värskendus nurjub puuduva koguse tõrkega

## <a name="symptoms"></a>Sümptomid

Te üritate esitada tootmistellimuse kui valmis, teatades samas veakoguse, kuid mitte aja või materjali koguse. Aruanne kui lõplik ajakohastamine ebaõnnestub, kui üritate tootmiskorraldust lõpetada, ja kuvatakse järgmine veateade:

> Puuduv lõpetatuna kinnitatud kogus.

## <a name="resolution"></a>Lahendus

Kui te esitate veakoguse tootmiskorraldusel, peate märkima ka kaupade koguse.
