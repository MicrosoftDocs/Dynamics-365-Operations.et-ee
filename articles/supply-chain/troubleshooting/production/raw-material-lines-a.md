# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Kasutajad saavad lõpetatuna kinnitatud toodete materjaliridade komplekteerimise tühistada

KB number: 4614721

## <a name="symptoms"></a>Sümptomid

Kasutajad saavad lõpetatuna kinnitatud toodete materjaliridade komplekteerimise tühistada.

## <a name="resolution"></a>Eraldusvõime

Süsteem käitub kavandatud viisil. Toormaterjali tarbimist saab korrigeerida seni, kuni tootmistellimuse olekuks saab *Lõpetatud*.
