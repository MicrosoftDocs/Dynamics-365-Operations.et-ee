# <a name="late-selection-isnt-respected-when-production-orders-are-reset-via-a-batch-job"></a>Hilist valikut ei arvestata tootmistellimuste lähtestamisel töö-paketi kaudu

KB number: 4614634

## <a name="symptoms"></a>Sümptomid

Kui kasutate tootmistellimuse oleku lähtestamiseks korduvat pakett-tööd, ei järgida hiljutisi valikuid.

## <a name="resolution"></a>Eraldusvõime

Praegune kujundus ei toeta hilinenud valikute kasutamist *Lähtesta olek* protsessis.
