# <a name="the-warehouse-in-the-picking-list-journal-isnt-updated-on-a-bom-line"></a>Komplekteerimislehe töölehel olevat ladu ei ole koosluse real uuendatud

KB number: 4614848

## <a name="symptoms"></a>Sümptomid

Komplekteerimislehe töölehel olevat ladu ei ole materjalide koosluse real uuendatud.

## <a name="resolution"></a>Eraldusvõime

Süsteem käitub kavandatud viisil. Kui luuakse komplekteerimislehe töölehe rida, mis sisaldab viiteid (saatelehe ID kaudu) tootmise koosluse reale, ei uuendata tootmiskoosluse rea ladu, kui tootmise komplekteerimislehe töölehe rea laodimensiooni muudetakse hiljem.
