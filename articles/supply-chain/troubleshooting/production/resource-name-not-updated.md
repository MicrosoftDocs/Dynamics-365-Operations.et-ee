# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Ressursi nime ei värskendata pärast töötaja nime muutmist aadressiraamatus

## <a name="symptoms"></a>Sümptomid

Kui muudate töötaja nime globaalses aadressiraamatus, ei uuendata vastava ressursi nime ressursigrupi koondandmetes.

## <a name="resolution"></a>Lahendus

Vastavat ressursinime ei uuendata praegu automaatselt. Probleemi lahendamiseks peate ressursi nime käsitsi värskendama.
