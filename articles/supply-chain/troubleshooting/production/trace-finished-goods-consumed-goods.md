# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>Valmistoodete seerianumbreid ei saa võrrelda tarbitud kaupade seerianumbritega

## <a name="symptoms"></a>Sümptomid

Lõpetatud kaupade seerianumbrit ei saa jälgida materjali seerianumbrite suhtes, mida tootmistellimus tarbib nende lõpetatud kaupade valmistamiseks.

## <a name="workaround"></a>Vastukaal

Probleemi lahendamiseks looge tootmistellimused kogusele 1.
