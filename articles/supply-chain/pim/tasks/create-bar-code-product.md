[!include [banner](../../includes/banner.md)]

<span data-ttu-id="53333-104">Selles teemas näidatakse, kuidas käsitsi luua vöötkoodi, kasutades näitena numbrit M0001.</span><span class="sxs-lookup"><span data-stu-id="53333-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="53333-105">Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid.</span><span class="sxs-lookup"><span data-stu-id="53333-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="53333-106">Valige kodulehel **Väljastatud toote hooldus**.</span><span class="sxs-lookup"><span data-stu-id="53333-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="53333-107">Avage **Tooted > Väljastatud tooted** jaotises **Lingid**.</span><span class="sxs-lookup"><span data-stu-id="53333-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="53333-108">Otsige loendist ja valige soovitud kirje.</span><span class="sxs-lookup"><span data-stu-id="53333-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="53333-109">Selle näite puhul valige kaubakood **M0001.**</span><span class="sxs-lookup"><span data-stu-id="53333-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="53333-110">Klõpsake toimingupaanil suvandit **Halda varusid**.</span><span class="sxs-lookup"><span data-stu-id="53333-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="53333-111">**Vöötkoodi** valimine</span><span class="sxs-lookup"><span data-stu-id="53333-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="53333-112">Valige suvand **Uus**.</span><span class="sxs-lookup"><span data-stu-id="53333-112">Select **New**.</span></span>
7. <span data-ttu-id="53333-113">Märkige valitud rida, mis on loodud allolevas loendis.</span><span class="sxs-lookup"><span data-stu-id="53333-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="53333-114">Valige või sisestage väärtus väljal **Vöötkoodi häälestus**.</span><span class="sxs-lookup"><span data-stu-id="53333-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="53333-115">Sisestage või valige väärtus väljal **Vöötkoo**.</span><span class="sxs-lookup"><span data-stu-id="53333-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="53333-116">**Tippige väärtus väljale Vöötkood.**</span><span class="sxs-lookup"><span data-stu-id="53333-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="53333-117">Sisestage arv väljale **Kogus**.</span><span class="sxs-lookup"><span data-stu-id="53333-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="53333-118">Valige käsk **Salvesta**.</span><span class="sxs-lookup"><span data-stu-id="53333-118">Select **Save**.</span></span>
13. <span data-ttu-id="53333-119">Sulgege leht.</span><span class="sxs-lookup"><span data-stu-id="53333-119">Close the page.</span></span>
