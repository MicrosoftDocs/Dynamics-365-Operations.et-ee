[!include [banner](../../includes/banner.md)]

<span data-ttu-id="e0096-104">Selles teemas näidatakse, kuidas käsitsi luua vöötkoodi, kasutades näitena numbrit M0001.</span><span class="sxs-lookup"><span data-stu-id="e0096-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="e0096-105">Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid.</span><span class="sxs-lookup"><span data-stu-id="e0096-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="e0096-106">Avage **Tooteteabe haldus \> Tooted \> Väljastatud tooted**.</span><span class="sxs-lookup"><span data-stu-id="e0096-106">Go to **Product information management \> Products \> Released products**.</span></span>
1. <span data-ttu-id="e0096-107">Otsige loendist ja valige soovitud kirje.</span><span class="sxs-lookup"><span data-stu-id="e0096-107">In the list, find and select the desired record.</span></span> <span data-ttu-id="e0096-108">Selle näite puhul valige kaubakood **M0001.**</span><span class="sxs-lookup"><span data-stu-id="e0096-108">For this example, select item number **M0001**.</span></span>
1. <span data-ttu-id="e0096-109">Klõpsake toimingupaanil suvandit **Halda varusid**.</span><span class="sxs-lookup"><span data-stu-id="e0096-109">On the Action Pane, select **Manage inventory**.</span></span>

1. <span data-ttu-id="e0096-111">Valige suvand **Uus**.</span><span class="sxs-lookup"><span data-stu-id="e0096-111">Select **New**.</span></span>

1. <span data-ttu-id="e0096-118">Sulgege leht.</span><span class="sxs-lookup"><span data-stu-id="e0096-118">Close the page.</span></span>
