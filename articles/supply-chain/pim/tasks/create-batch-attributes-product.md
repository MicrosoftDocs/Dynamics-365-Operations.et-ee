# <a name="create-batch-attributes-for-a-product"></a>Toote jaoks partiiatribuutide loomine

[!include [banner](../../includes/banner.md)]

See protseduur näitab, kuidas luua partii atribuuti, määrata vaikeväärtuse vahemikke ja kaasata atribuudi gruppi. Selle protseduuri loomiseks kasutati demoettevõtte USP2 andmeid.

1. Avage Varude haldus > Seadistus > Partii > Partii atribuudid.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Atribuut.
4. Sisestage väljale Kirjeldus soovitud väärtus.
5. Valige väljalt Atribuudi tüüp suvand Murd.
    * See protseduur kasutab kümnendarvude võimaldamiseks tüüpi Murd. Saate valida muid atribuudi tüüpe. Tüübi Loetelu valimisel peate sisestama väärtused loetelu loendisse enne, kui saate väärtuse väljale Siht sisestada.  
6. Sisestage number väljale Miinimum.
7. Sisestage number väljale Maksimum.
8. Sisestage number väljale Juurdekasv.
9. Sisestage väärtus väljale Siht.
10. Klõpsake nuppu Salvesta.
11. Sulgege leht.
12. Avage Varude haldus > Seadistus > Partii > Partii atribuudigrupid.
13. Klõpsake valikut Uus.
14. Sisestage väärtus väljale Atribuudigrupp.
15. Sisestage väljale Kirjeldus soovitud väärtus.
16. Klõpsake nuppu Salvesta.
17. Klõpsake käsku Grupeeri atribuudid.
18. Klõpsake valikut Uus.
19. Klõpsake väljal Atribuut otsingu avamiseks ripploendi nuppu.
20. Otsige loendist ja valige soovitud kirje.
21. Klõpsake loendis valitud real olevat linki.
    * Atribuut võib olla kaasatud mis tahes gruppi.  
22. Klõpsake nuppu Salvesta.
23. Sulgege leht.
