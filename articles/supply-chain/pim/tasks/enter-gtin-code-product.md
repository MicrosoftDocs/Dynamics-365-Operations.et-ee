# <a name="enter-the-gtin-code-for-a-product"></a>Toote GTIN-koodi sisestamine

[!include [task guide banner](../../includes/task-guide-banner.md)]

GTIN-koodid määrab GS1 organisatsioon. Selles näites sisestate demoettevõtte USMF tootenumbri M0001.

1. Klõpsake valikut Väljastatud toodete hooldus.
2. Klõpsake valikut Väljastatud tooted.
3. Laiendage toote pildi kiirinfot.
4. Otsige loendist ja valige soovitud kirje.
5. Klõpsake toimingupaanil suvandit Halda varusid.
6. Klõpsake valikut GTIN-koodid.
7. Klõpsake valikut Uus.
8. Märkige loendis valitud rida.
9. Sisestage või valige väärtus väljal Ühik.
10. Tehke valik väljal GTIN-kood.
11. Sisestage väärtus väljale GTIN.
12. Sulgege leht.
