# <a name="create-a-new-product"></a>Uue toote loomine

[!include [banner](../../includes/banner.md)]

See teema kirjeldab uue konfiguratsiooni loomist. Tavaliselt teeb seda toote koostaja. Selle tegevuse loomisel kasutati demoettevõtte USMF-i andmeid.


## <a name="create-a-product"></a>Toote loomine
1. Avage navigeerimispaanil **Moodulid > Tooteteabe haldus > Tooted > Tooted**.
2. Valige suvand **Uus**.
3. Sisestage väärtus väljale **Toote number**. Kui toote koodile ei ole numbriseeriat seadistatud, tuleb see sisestada käsitsi.  
4. Sisestage väärtus väljale **Toote nimi**. Toote nimi on vaikimisi otsingu nimi. Saate seda vajaduse korral muuta.  
5. Valige nupp **OK**.

## <a name="set-up-dimension-groups"></a>Dimensioonigruppide seadistamine
1. Klõpsake rippdialoogi avamiseks suvandit **Dimensiooni rühmad**.
2. Sisestage või valige väärtus väljal **Ladustamisdimensiooni rühm**. Laoala dimensiooni grupp määrab, millised laoala dimensioonid peate sisestama iga toote kande kohta ja kuidas seda varudes jälgitakse.  
3. Sisestage või valige väärtus väljal **Jälgimisdimensiooni rühm**. Jälgimisdimensiooni grupp määrab, millised jälgimisdimensioonid peate sisestama iga toote kande kohta ja kuidas seda varudes käsitletakse.  
4. Valige nupp **OK**.
