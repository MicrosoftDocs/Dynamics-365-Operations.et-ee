# <a name="add-an-image-to-a-product"></a>Tootele pildi lisamine

[!include [task guide banner](../../includes/task-guide-banner.md)]

See protseduur näitab, kuidas tootele pilti lisada. Selle toimingu loomiseks on kasutatud demoettevõtet USMF.

1. Avage Tooteteabe haldus > Tooted > Väljastatud tooted.
2. Laiendage toote pildi kiirinfot.
3. Otsige loendist ja valige soovitud kirje.
    * Näiteks valige kaubakood C0002.  
4. Klõpsake nuppu Pildi muutmine.
5. Klõpsake valikut Uus.
    * Klõpsake käsku Sirvi. Leidke pilt, mida kasutada soovite, valige see ja sulgege siis dialoogiboks.
