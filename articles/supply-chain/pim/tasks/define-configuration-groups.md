# <a name="define-configuration-groups"></a>Konfiguratsioonigruppide määratlemine

[!include [banner](../../includes/banner.md)]

See protseduur selgitab toote konfiguratsioonigruppide määratlemist, luues demoettevõttes USMF Company kaks gruppi. See on neljas protseduur kaheksast, mis selgitab kombinatsioonide loomist dimensioonipõhise konfiguratsiooni jaoks.

1. Minge jaotisse Tooteteabe haldus > Seadistus > Kooslused ja valemid > Konfiguratsioonigrupid.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Konfiguratsioonigrupp.
    * Sisestage koosluseridade puhul kasutatava konfiguratsioonigrupi nimi.  
4. Sisestage väärtus väljale Nimi.
5. Klõpsake valikut Uus.
6. Sisestage väärtus väljale Konfiguratsioonigrupp.
    * Sisestage koosluseridade puhul kasutatava konfiguratsioonigrupi nimi.  
7. Sisestage väärtus väljale Nimi.
8. Sulgege leht.
