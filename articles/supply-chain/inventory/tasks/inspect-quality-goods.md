# <a name="inspect-the-quality-of-goods"></a>Kaupade kvaliteedi kontrollimine

[!include [banner](../../includes/banner.md)]

Selles teemas selgitatakse, kuidas kvaliteettellimust töödelda. Saate selle juhendi käitada demoettevõtte USMF andmetega. Enne selle näidisprotseduuri käivitamist peate ostutellimuse 000016 kinnitama ja toote sissetuleku sisestama. See loob kvaliteettellimuse automaatselt. Kvaliteedikontrolle teeb üldjuhul kvaliteediametnik.


## <a name="select-a-quality-order"></a>Kvaliteettellimuse valimine
1. Navigeerimispaanil avage **Moodulid > Varud > Perioodilised ülesanded > Kvaliteethaldus > Kvaliteettellimus**.
2. Valige kvaliteettellimus, mis loodi enne selle protseduuri käivitamist.  

## <a name="record-test-results"></a>Testitulemuste salvestamine
1. Valige **Tulemid**.
2. Valige suvand **Redigeeri**.
3. Sisestage number väljale **Tulemuse kogus**.
4. Valige otsingu avamiseks väljal **Väljundteade** ripploendi nupp.  
- Selles näites põhineb tulemus eelmääratletud tulemusel. Üldjuhul saate salvestada täpsema katse tulemuse, nt suuruse või muu dimensiooni.  
5. Valige käsk **Salvesta**.
6. Sulgege leht.

## <a name="validate-the-quality-order"></a>Kontrolli kvaliteettellimuse õigsust
1. Valige **Kinnita**.
2. Väljal **Kinnitaja** valige kasutaja, kes teostab kontrolli rippmenüüs.  
3. Klõpsake **Vali**.
4. Valige nupp **OK**.
5. Sulgege leht.
