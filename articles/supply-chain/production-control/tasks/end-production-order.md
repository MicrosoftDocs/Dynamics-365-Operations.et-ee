# <a name="end-a-production-order"></a>Tootmistellimuse lõpetamine

[!include [banner](../../includes/banner.md)]

See protseduur näitab, kuidas tootmistellimust lõpetada. Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid. See on viimane protseduur seitsmest, mis selgitab tootmistellimuse elutsüklit.


## <a name="end-a-production-order"></a>Tootmistellimuse lõpetamine
1. Avage Tootmise juhtimine > Tootmistellimused > Kõik tootmistellimused.
    * Valige tootmistellimus, mille olek on Lõpetatuna kinnitatud.  
2. Klõpsake toimingupaanil valikut Tootmistellimus.
3. Klõpsake suvandit Lõpeta.
    * Sellel lehel saate kinnitada tootmistellimuse lõpetamise.  
4. Klõpsake vahekaarti Üldine.
5. Sisestage kuupäev väljale Kuupäev.
6. Valige väljal Praagimeetod suvand Eraldamine.
    * Kui valite suvandi Eraldamismeetod, lisatakse lõpetatud kaupadele kulud mahakantud materjalide kulud.  
7. Klõpsake nuppu OK.

## <a name="validate-calculation-results"></a>arvutustulemuste kinnitamine
1. Klõpsake toimingupaanil valikut Kulude haldamine.
2. Klõpsake käsku Kuva kulu võrdlus.
    * Pärast seda, kui tootmistellimus on lõpetatud, saate võrrelda eeldatavat omahinda realiseeritud omahinnaga, et saada ülevaade tootmishälvetest.
