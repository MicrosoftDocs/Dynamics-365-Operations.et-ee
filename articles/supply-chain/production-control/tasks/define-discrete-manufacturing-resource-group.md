# <a name="define-discrete-manufacturing-resource-group"></a>Diskreetse tootmise ressursigrupi määratlemine

[!include [banner](../../includes/banner.md)]

Ressursigrupp on operatsiooniressursside kogum, mis tavaliselt vastab tootmistöödes kollaste joontega määratletud töörakkude füüsilisele organiseerimisele. Selles protseduuris näitlikustatakse, kuidas määratleda diskreetses tootmises kasutatavat ressursigruppi. Saate selle toiminguga tutvuda demoettevõtte USMF-i või omaenda andmeid kasutades.

1. Avage Ressursigrupid.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Ressursigrupp.
4. Sisestage väljale Kirjeldus soovitud väärtus.
5. Sisestage või valige väärtus väljal Koht.
6. Sisestage või valige väärtus väljal Tootmisüksus.

## <a name="define-default-operational-parameters"></a>Toimingu vaikeparameetrite määratlemine
1. Laiendage jaotist Toiming.
2. Sisestage arv väljale Praagi protsent.
3. Sisestage või valige väärtus väljal Seadistuse kategooria.
4. Sisestage või valige väärtus väljal Käitusaja kategooria.
5. Sisestage arv väljale Operatsioonide planeerimise protsent.

## <a name="define-operating-hours"></a>Tööaja määratlemine
1. Laiendage jaotist Kalendrid.
2. Klõpsake vahekaarti Lisa.
3. Sisestage või valige väärtus väljal Kalender.

## <a name="add-operations-resources"></a>Operatsiooniressursside lisamine
1. Laiendage jaotist Ressursid.
2. Klõpsake vahekaarti Lisa.
3. Sisestage või valige väärtus väljal Ressurss.
4. Klõpsake vahekaarti Lisa.
5. Sisestage või valige väärtus väljal Ressurss.
6. Otsige loendist ja valige soovitud kirje.
7. Klõpsake loendis valitud real olevat linki.
