# <a name="move-scheduled-kanban-jobs"></a>Ajastatud kanban-tööde teisaldamine

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

See protseduur keskendub plaanitud protsessi kanban-tööde teise perioodi teisaldamisele. Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid. See protseduur on mõeldud kanbanidega töötavale tööde järelevaatajale või tootmise planeerijale.


## <a name="select-scheduled-kanban-jobs"></a>Plaanitud kanban-tööde valimine
1. Avage Tootmine > Kanban > Kanban-töö planeerimine.
2. !MtCMR!Klõpsake väljal Töörakk otsingu avamiseks ripploendi nuppu. áçêìõý!
3. Märkige loendis valitud rida.
    * Valige töörakk 1250.  
4. Klõpsake suvandit Vali.
5. Valige väljalt Plaanitud suvand Töö oleku kuvamine.
    * See filtrib tööloendi kuvama ainult plaanitud kanban-töid.  

## <a name="move-kanban-jobs-to-a-different-period"></a>Kanban-tööde teisaldamine teise perioodi
1. Leidke ja valige loendist vajalik kirje.
    * Valige töö, mille olekuks on Plaanitud, näiteks töö, mis on väljal Plaanitud periood kavandatud 20. detsembrile 2012. Seejärel teisaldage töö eelmisesse perioodi.  
2. Klõpsake suvandit Eelmine periood.
3. Klõpsake suvandit Lõpp.
    * See teisaldab töö tööde loendi lõppu viimase tööna eelmises perioodis.  
4. Leidke ja valige loendist vajalik kirje.
    * Valige töö, mille olekuks on Plaanitud, näiteks töö, mis on väljal Plaanitud periood kavandatud 18. detsembrile 2012. Seejärel teisaldage töö järgmisesse perioodi.  
5. Klõpsake suvandit Järgmine periood.
6. Klõpsake suvandit Algus.
    * See teisaldab töö tööde loendi algusesse esimese tööna eelmises perioodis.  

## <a name="task-move-a-job-within-a-period"></a>Ülesanne: perioodi töö teisaldamine
1. Leidke ja valige loendist vajalik kirje.
    * Valige töö, mille olekuks on Plaanitud, näiteks teine töö, mis on väljal Plaanitud periood kavandatud 19. detsembrile 2012. Seejärel liigutage tööd plaanitud perioodi sees.  
2. Klõpsake suvandit Edasi.
    * Pange tähele, et tööd teisaldatakse loendis üks rida alla.  
3. Klõpsake suvandit Tagasi.
    * Pange tähele, et tööd teisaldatakse loendis üks rida üles.
