# <a name="remove-a-kanban-job-from-the-schedule"></a>Kanban-töö eemaldamine graafikust

[!include [task guide banner](../../includes/task-guide-banner.md)]

See protseduur keskendub plaanitud protsessi kanban-töö eemaldamisele graafikust, taastades töö oleku väärtusele Plaanimata. Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid. See protseduur on mõeldud tööde järelevaatajale või tootmise planeerijale.


## <a name="find-a-planned-kanban-job"></a>Plaanitud kanban-töö leidmine
1. Avage Tootmise juhtimine > Kanban > Kanban-tööde plaanimine.
2. Klõpsake väljal Töörakk otsingu avamiseks ripploendi nuppu.
3. Klõpsake loendis valitud real olevat linki.
    * Valige töörakk 1250.  
4. Klõpsake Vali.
5. Valige väljalt Töö oleku kuvamine suvand Plaanitud.

## <a name="remove-the-planned-kanban-job-from-the-schedule"></a>Plaanitud kanban-töö eemaldamine graafikust
1. Otsige loendist ja valige soovitud kirje.
    * Valige töö, mille olekuks on Plaanitud, näiteks töö 19. detsembrist 2012.  
2. Klõpsake toimingupaanil valikut Plaan.
3. Klõpsake suvandit Töö oleku taastamine.
4. Klõpsake nuppu OK.
    * See taastab töö praeguse oleku suvandilt Plaanitud suvandile Plaanimata ja eemaldab selle protsessitahvlilt.
