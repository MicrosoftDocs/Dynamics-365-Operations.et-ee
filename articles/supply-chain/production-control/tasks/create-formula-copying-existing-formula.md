# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Valemi loomine, kopeerides olemasolevast valemist

[!include [task guide banner](../../includes/task-guide-banner.md)]

See protseduur näitab, kuidas kiiresti uut valemit koostada, kopeerides olemasolevalt valemilt. Selle protseduuri koostamiseks võite kasutada demoettevõtete USMF, USP2 või USPI andmeid.


## <a name="create-a-new-formula"></a>Uue valemi koostamine
1. Minge jaotisse Valemid.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Valem.
4. Sisestage väärtus väljale Nimi.
5. Sisestage või valige väärtus väljal Koht.
6. Sisestage või valige väärtus väljal Kaubagrupp.
7. Klõpsake nuppu Salvesta.

## <a name="copy-from-an-existing-formula"></a>Olemasolevast valemist kopeerimine
1. Klõpsake käsku Kopeeri.
2. Sisestage või valige väärtus väljal Kaubakood.
3. Valige või sisestage väärtus väljal Valemi versioon.
4. Klõpsake nuppu OK.

## <a name="revise-the-copied-formula"></a>Kopeeritud valemi muutmine
1. Otsige loendist ja valige soovitud kirje.
2. Klõpsake  Kustuta.
3. Klõpsake nuppu Jah.

## <a name="approve-formula"></a>Kinnita valem
1. Klõpsake suvandit Kinnita valem.
2. Valige või sisestage väärtus väljal Kinnitaja.
3. Klõpsake nuppu OK.
4. Sulgege leht.
