# <a name="schedule-a-production-order"></a>Tootmistellimuse plaanimine

[!include [task guide banner](../../includes/task-guide-banner.md)]

See protseduur näitab, kuidas tootmistellimust plaanida. Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid. See on kolmas protseduur seitsmest, mis selgitab tootmistellimuse elutsüklit.


## <a name="schedule-a-production-order"></a>Tootmistellimuse plaanimine
1. Avage Tootmise juhtimine > Tootmistellimused > Kõik tootmistellimused.
    * Valige tootmistellimus, mille olek on Hinnanguline.  
2. Klõpsake tegumiribal valikut Graafik.
3. Klõpsake valikut Tööde planeerimine.
    * Plaanimise parameetrid häälestatakse sel lehel. Saate häälestada konkreetsete kasutajate või kõigi kasutajate parameetrid.  
4. Valige väljal Plaanimissuund suvand „Edasi tänasest”.
5. Sisestage kuupäev väljale Plaanimiskuupäev.
6. Märkige või tühjendage ruut Piiratud võimsus.
7. Märkige või tühjendage ruut Limiteeritud materjal.
8. Klõpsake nuppu OK.

## <a name="view-the-scheduling-results"></a>Plaanimise tulemuste kuvamine
1. Klõpsake toimingupaanil valikut Tootmistellimus.
2. Klõpsake valikut Kõik tööd.
    * Lehel kuvatakse äsja loodud plaanitud tööd.  
3. Laiendage või ahendage jaotist Plaanimine.
    * Kiirinfos Plaanimine näete plaanitud kuupäeva ja kellaaega.  
4. Klõpsake suvandit Päringud.
5. Klõpsake suvandit Täiskoormus.
    * Lehel Täiskoormus kuvatakse töö plaanimise kaudu reserveeritud võimekus, ressursi jaoks praegu reserveeritud tundide koguarv ja ressursi töö plaanimise jaoks järelejäänud tundide arv.  
6. Sulgege leht.
7. Sulgege leht.
