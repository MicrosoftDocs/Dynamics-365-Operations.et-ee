# <a name="create-a-value-stream"></a>Väärtusevoo loomine

[!include [banner](../../includes/banner.md)]

Väärtuse vood on tootmisüksused, mis grupeerivad ühe või mitu tootmisvoogu. Tootmisvoo määratlemiseks peate esmalt määratlema väärtuse voo.


## <a name="open-the-form"></a>Avage vorm
1. Minge jaotisse Tootmise juhtimine > Seadistus > Kulusäästlik tootmisvoog > Väärtuse vood.

## <a name="create-a-new-value-stream"></a>Looge uus väärtuse voog
1. Rippdialoogi avamiseks klõpsake valikut Uus.
2. Sisestage väärtus väljale Nimi.

## <a name="select-a-value-stream-manager"></a>Valige väärtuse voo haldur
1. Klõpsake väljal Haldur otsingu avamiseks ripploendi nuppu.
    * Väärtuse voo haldur määratletakse organisatsiooni dokumenteerimiseks.  
2. Otsige loendist ja valige soovitud kirje.
3. Klõpsake loendis valitud real olevat linki.

## <a name="save-and-close"></a>Salvesta ja sule
1. Klõpsake nuppu Salvesta.
2. Sulgege leht.
3. Värskendage lehte.
