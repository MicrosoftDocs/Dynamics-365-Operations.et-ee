# <a name="view-current-wip-status-on-a-production-order"></a>Tootmistellimuse WIP praeguse oleku kuvamine

[!include [banner](../../includes/banner.md)]

See protseduur näitab, kuidas vaadata tootmistellimuse lõpetamata toodangu (WIP) aruannet. Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid. See protseduur on mõeldud kulukontrollerile.

1. Klõpsake valikut Kuluhaldus.
2. Klõpsake suvandit Tootmistellimused.
3. Kasutage kiirfiltrit, et filtreerida välja Tootmine väärtuse p000153 järgi.
4. Klõpsake toimingupaanil valikut Kulude haldamine.
5. Klõpsake suvandit Tootmise WIP-aruanne.
6. Määrake väljal Alates kuupäevaks 01.12.2012.
7. Määrake väljal Kuni kuupäevaks 31.12.2012.
