# <a name="set-up-an-appointment-for-a-load"></a>Koorma laadimise seadistamine

[!include [banner](../../includes/banner.md)]

See protseduur näitab, kuidas seadistada ja plaanida koormale laadimissilla määramist. Seda teeb üldjuhul transpordikoordinaator. Saate seda protseduuri kasutada demoandmete ettevõttes USMF.


## <a name="select-the-load"></a>Valige koorem
1. Avage Transpordihaldus > Plaanimine > Koorma plaanimise töölaud.
2. Tühjendage märkeruut Peida saadetud ja vastuvõetud.
3. Valige loendist koorem, mille olek on Saadetud.
4. Klõpsake nuppu Transport.
5. Klõpsake nuppu Laadimiste planeerimine.

## <a name="create-an-appointment"></a>Laadimise loomine
1. Klõpsake valikut Uus.
2. Valige või sisestage väärtus väljal Laadimisreegel.
3. Klõpsake nuppu Salvesta.
4. Klõpsake valikut Uuenda olekut.
5. Klõpsake nuppu Kinnita.
6. Klõpsake nuppu Salvesta.
7. Sulgege leht.
