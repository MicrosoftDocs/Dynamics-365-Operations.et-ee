# <a name="set-up-hub-accessorial-charges-and-accessorial-masters"></a>Keskuse lisatasude ja lisade koondandmete seadistamine

[!include[task guide banner](../../includes/task-guide-banner.md)]

See protseduur kirjeldab keskuse lisade koondandmete loomist ja nende koondandmete kasutamist keskuse lisade tasu loomiseks. Protseduur kasutab USMF-i andmekomplekti. Seda seadistust teeb üldjuhul transpordikoordinaator.


## <a name="set-up-a-hub-master"></a>Keskuse koondandmete seadistamine
1. Avage Transpordihaldus > Seadistus > Hinnang > Lisade koondandmed.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Lisade koondandmed.
4. Sisestage väärtus väljale Nimi.
5. Valige väljal Lisatasu tüüp suvand Keskus.
6. Klõpsake nuppu Salvesta.
7. Sulgege leht.

## <a name="set-up-a-hub-accessorial-charge"></a>Keskuse lisatasu seadistamine
1. Avage Transpordihaldus > Seadistus > Hinnang > Keskuse lisatasud.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Keskuse lisade ID.
4. Klõpsake väljal Keskus otsingu avamiseks ripploendi nuppu.
5. Otsige loendist ja valige soovitud kirje.
6. Valige suvand väljalt Keskuse asend.
    * Saate luua tasu kui vastuvõtu või kohaleviimisena. Teie valikust olenevalt rakendatakse tasu teie marsruudi asjakohasele transpordisegmendile.  
7. Klõpsake väljal Lisade koondandmed otsingu avamiseks ripploendi nuppu.
8. Klõpsake loendis valitud real olevat linki.
    * Valige äsja loodud koondandmed.  
9. Klõpsake nuppu Salvesta.
10. Sulgege leht.
