# <a name="set-up-shipping-carriers"></a>Kättetoimetamisettevõtte seadistus

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

See protseduur näitab, kuidas seadistada tarne vedaja ja määratleda üksikasjad, nt teenus, saadetise režiim, transpordi maksevahend, transpordi piirangud ja tarnekulu. Transpordi koordinaator saab seejärel määrata saadetise vedaja sissetulevale või väljaminevale koormale.


## <a name="create-a-new-shipping-carrier"></a>Uue kättetoimetaja loomine
1. Tehke valik Transpordi haldus > Seadistus > Vedajad > Kättetoimetajad.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Kättetoimetaja.
4. Sisestage väärtus väljale Nimi.
5. Klõpsake väljal Režiim otsingu avamiseks ripploendi nuppu.
6. Otsige loendist ja valige soovitud kirje.
7. Klõpsake loendis valitud real olevat linki.

## <a name="fill-in-the-general-information-for-the-shipping-carrier"></a>Kättetoimetaja üldise teabe sisestamine
1. Laiendage jaotist Ülevaade.
2. Märkige või tühjendage ruut Aktiveeri saadetise vedaja.
3. Klõpsake väljal Hankija otsingu avamiseks ripploendi nuppu.
    * Valige hankija konto, millele tarne vedaja määrata.  
4. Otsige loendist ja valige soovitud kirje.
5. Klõpsake loendis valitud real olevat linki.
6. Valige suvand väljalt Transpordi maksevahendi tüüp.
    * Valige leht Transpordi maksevahendi kasutamise juhend või valige EDI pakkumise värskendamiseks elektroonilise andmevahetuse (EDI) abil.  
7. Märkige või tühjendage ruut Aktiveeri vedaja hinnang.

## <a name="create-the-necessary-services-for-the-shipping-carrier"></a>Kättetoimetaja vajalike teenuste loomine
1. Laiendage jaotist Teenused.
2. Klõpsake valikut Uus.
3. Märkige loendis valitud rida.
4. Sisestage väärtus väljale Vedaja teenus.
5. Sisestage väärtus väljale Nimi.
6. Klõpsake väljal Transpordiviis otsingu avamiseks ripploendi nuppu.
7. Otsige loendist ja valige soovitud kirje.
8. Klõpsake loendis valitud real olevat linki.

## <a name="set-up-the-address-for-the-carrier-optional"></a>Vedaja aadressi seadistamine (valikuline)
1. Lülitage ümber jaotis Aadressid.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Nimi või kirjeldus.
4. Klõpsake väljal Riik/regioon otsingu avamiseks ripploendi nuppu.
5. Klõpsake loendis valitud real olevat linki.
6. Klõpsake väljal Sihtnumber otsingu avamiseks ripploendi nuppu.
7. Otsige loendist ja valige soovitud kirje.
8. Klõpsake loendis valitud real olevat linki.
9. Sisestage väärtus väljale Tänav.
10. Klõpsake nuppu OK.

## <a name="set-up-the-rating-profile-for-the-shipping-carrier"></a>Kättetoimetaja hinnanguprofiili seadistamine
1. Laiendage jaotist Hinnanguprofiilid.
2. Klõpsake valikut Uus.
3. Märkige loendis valitud rida.
4. Sisestage väärtus väljale Hinnanguprofiil.
5. Sisestage väärtus väljale Nimi.
6. Klõpsake väljal Koht otsingu avamiseks ripploendi nuppu.
7. Otsige loendist ja valige soovitud kirje.
8. Klõpsake loendis valitud real olevat linki.
9. Klõpsake väljal Ladu otsingu avamiseks ripploendi nuppu.
10. Otsige loendist ja valige soovitud kirje.
11. Klõpsake loendis valitud real olevat linki.
12. Klõpsake väljal Määra mootor otsingu avamiseks ripploendi nuppu.
    * Valige Määra mootor, mis on kooskõlas vedaja lepinguga.  
13. Otsige loendist ja valige soovitud kirje.
14. Klõpsake loendis valitud real olevat linki.
15. Klõpsake väljal Koondmäär otsingu avamiseks ripploendi nuppu.
16. Otsige loendist ja valige soovitud kirje.
17. Klõpsake loendis valitud real olevat linki.
18. Klõpsake väljal Transiitaja mootor otsingu avamiseks ripploendi nuppu.
19. Klõpsake loendis valitud real olevat linki.
20. Klõpsake nuppu Salvesta.
