# <a name="set-up-a-transportation-tender"></a>Transpordi maksevahendi seadistamine

[!include [task guide banner](../../includes/task-guide-banner.md)]

See protseduur näitab, kuidas seadistada transpordi maksevahendit. Seda teeb üldjuhul transpordikoordinaator. Saate seda protseduuri kasutada demoandmete ettevõttes USMF.


## <a name="select-a-route"></a>Protsessi valimine
1. Avage Transpordihaldus > Plaanimine > Koorma plaanimise töölaud.
2. Tühjendage märkeruut Peida saadetud ja vastuvõetud.
3. Valige rida koorma ID-ga 00006.
4. Klõpsake suvandit Hindamine ja marsruudivalik.
5. Klõpsake valikut Protsessid.

## <a name="create-the-transportation-tender"></a>Transpordi maksevahendi loomine
1. Klõpsake valikut Transpordi maksevahendid.
2. Klõpsake valikut Uus.
3. Laiendage jaotis Üldine.
4. Sisestage number väljale Nõutavad määrad.
5. Klõpsake nuppu Salvesta.
6. Klõpsake valikut Uuenda olekut.
7. Klõpsake Edasta.
8. Valige protsess.
