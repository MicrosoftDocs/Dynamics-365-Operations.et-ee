# <a name="maintenance-request-types"></a>Hooldustaotluste tüübid

[!include [banner](../../includes/banner.md)]

 

Hooldustaotluse tüüpe kasutatakse hooldustaotluste kategoriseerimiseks. Näiteks võib teil olla hooldustaotluse tüüpe, mis on seotud ennetava hoolduse ja parandava hooldusega. Või teil võib olla spetsiaalne hooldustaotluse tüüp, mida kasutatakse varade remondi haldamiseks (depooparandus).

’Hooldustaotluse tüüp määratleb kuuluvuse hooldustaotluse elutsükli oleku rühma (hooldustaotluse elutsükli mudelisse). Hooldustaotluse elutsükli mudelid määratlevad elutsükli olekud, mida saab hooldustaotlusele seada. (Hooldustaotluse elutsükli oleku näited: **Loodud**, **Aktiivne** ja **Lõpetatud**.)

1. Valige **Varahaldus**\>**Häälestus**\>**Hooldustaotlused**\>**Hooldustaotluse tüübid**.
2. Uue hooldustaotluse tüübi loomiseks valige **Uus**.
3. Sisestage väljale **Hooldustaotluse tüüp** hooldustaotluse tüübi ID.
4. Väljale **Nimi** sisestage nimi.
5. Kiirkaardil **Üldine** väljal **Hooldustaotluse elutsükli mudel** väljal valige hooldustaotluse elutsükli mudel.
6. Väljal **Töökäsu tüüp** valige töökäsu tüüp. Kui hooldustaotlus teisendatakse töökäsuks, saab töökäsu automaatselt töökäsu tüübi, mis on seotud hooldustaotluse tüübiga.
