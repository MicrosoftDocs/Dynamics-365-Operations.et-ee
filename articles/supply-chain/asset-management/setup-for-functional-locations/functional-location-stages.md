# <a name="functional-location-lifecycle-states"></a><span data-ttu-id="77e95-103">Funktsionaalse asukoha elutsükli olekud</span><span class="sxs-lookup"><span data-stu-id="77e95-103">Functional location lifecycle states</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="77e95-104">Selles teemas kirjeldatakse, kuidas seadistada funktsionaalseid asukoha elutsükli olekuid ja elutsükli mudeleid varahalduses.</span><span class="sxs-lookup"><span data-stu-id="77e95-104">This topic describes how to set up functional location lifecycle states and lifecycle models in Asset Management.</span></span> <span data-ttu-id="77e95-105">Funktsionaalse asukoha elutsükli olekud määratlevad, et funktsionaalne asukoht võib läbida näiteks olekud loodud, aktiivne ja lõpetatud.</span><span class="sxs-lookup"><span data-stu-id="77e95-105">Functional location lifecycle states define the states that a functional location can go through, for example, created, active, and ended.</span></span> <span data-ttu-id="77e95-106">Kõiki funktsionaalseid asukohti, olenemata nende elutsükli olekust, saate vaadata loendilehel **Kõik funktsionaalsed asukohad**.</span><span class="sxs-lookup"><span data-stu-id="77e95-106">You are able to view all functional locations, regardless of their lifecycle state, in the **All functional locations** list page.</span></span> <span data-ttu-id="77e95-107">Funktsionaalse asukoha olekut saate muuta valides selle loendilehel **Kõik funktsionaalsed asukohad** ja valides **Värskenda funktsionaalse asukoha olek**.</span><span class="sxs-lookup"><span data-stu-id="77e95-107">You can change the state of a functional location by selecting it in the **All functional locations** list page and selecting **Update functional location state**.</span></span>

## <a name="set-up-functional-location-lifecycle-states"></a><span data-ttu-id="77e95-108">Seadke funktsionaalse asukoha elutsükli olekud</span><span class="sxs-lookup"><span data-stu-id="77e95-108">Set up functional location lifecycle states</span></span>

1. <span data-ttu-id="77e95-109">Valige **Varahaldus** > **Sätted** > **Funktsionaalsed asukohad** > **Elutsükli olekud**.</span><span class="sxs-lookup"><span data-stu-id="77e95-109">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle states**.</span></span>
2. <span data-ttu-id="77e95-110">Uue funktsionaalse asukoha oleku loomiseks valige **Uus**.</span><span class="sxs-lookup"><span data-stu-id="77e95-110">Select **New** to create a new functional location state.</span></span>
3. <span data-ttu-id="77e95-111">Sisestage oleku ID väljale **Elutsükli olek** ja funktsionaalse asukoha oleku nimi väljale **Nimi**.</span><span class="sxs-lookup"><span data-stu-id="77e95-111">Insert the state ID in the **Lifecycle state** field and a name for the functional location state in the **Name** field.</span></span> <span data-ttu-id="77e95-112">Väljal **Elutsükli mudelid** saate vaadata funktsionaalse asukoha olekut kasutavate funktsionaalse asukoha oleku elutsükli mudelite arvu.</span><span class="sxs-lookup"><span data-stu-id="77e95-112">In the **Lifecycle models** field, you can see the number of functional location lifecycle models that uses the functional location state.</span></span>
4. <span data-ttu-id="77e95-113">Kiirkaardil **Üldine** valige „Jah“ tumblernupul **Aktiivne**, kui funktsionaalne asukoht peaks selles olekus aktiivne olema.</span><span class="sxs-lookup"><span data-stu-id="77e95-113">On the **General** FastTab, select "Yes" on the **Active** toggle button if the functional location should be active at this state.</span></span>
5. <span data-ttu-id="77e95-114">Valige „Jah“ tumblernupul **Loo varad**, kui varade loomine peaks olema automaatselt võimalik sama nimega nagu funktsionaalne asukoht ja installige see funktsionaalse asukoha selle olekuga.</span><span class="sxs-lookup"><span data-stu-id="77e95-114">Select "Yes" on the **Create assets** toggle button if it should be possible to automatically create an asset with the same name as the functional location and install it on the functional location at this state.</span></span>  
>[!NOTE]
><span data-ttu-id="77e95-115">See tumblernupp on seotud väljaga **Vara tüüp** kiirkaardil **Üldine** vormil **Funktsionaalse asukoha tüübid** (**Varahaldus** > **Sätestamine** > **Funktsionaalsed asukohad** > **Funktsionaalse asukoha tüübid**).</span><span class="sxs-lookup"><span data-stu-id="77e95-115">This toggle button relates to the **Asset type** field on the **General** FastTab in the **Functional location types** form (**Asset management** > **Setup** > **Functional locations** > **Functional location types**).</span></span>
6. <span data-ttu-id="77e95-116">Valige „Jah“ tumblernupul **Nimeta asukoht ümber**, kui selles olekus on võimalik muuta funktsionaalse asukoha nime.</span><span class="sxs-lookup"><span data-stu-id="77e95-116">Select "Yes" on the **Rename location** toggle button if it should be possible to change the name of the functional location at this state.</span></span>
7. <span data-ttu-id="77e95-117">Valige „Jah“ tumblernupul **Uued alamasukohad**, kui selles olekus on võimalik lisada uusi alamasukohti funktsionaalsele asukohale.</span><span class="sxs-lookup"><span data-stu-id="77e95-117">Select "Yes" on the **New sub locations** toggle button if it should be possible to add new sub locations to the functional location at this state.</span></span>
8. <span data-ttu-id="77e95-118">Valige „Jah“ tumblernupul **Installi varad**, kui selles olekus on võimalik installida varasid funktsionaalsele asukohale.</span><span class="sxs-lookup"><span data-stu-id="77e95-118">Select "Yes" on the **Install assets** toggle button if it should be possible to install assets on the functional location at this state.</span></span>
9. <span data-ttu-id="77e95-119">Valige „Jah“ tumblernupul **Kustuta funktsionaalne asukoht**, kui selles olekus on võimalik kustutada funktsionaalne asukoht.</span><span class="sxs-lookup"><span data-stu-id="77e95-119">Select "Yes" on the **Delete functional location** toggle button if it should be possible to delete the functional location at this state.</span></span>
10. <span data-ttu-id="77e95-120">Valige vara olek väljal **Elutsükli olek**, kui soovite, et vara elutsükli olekut kõigi funktsionaalse asukoha varade kohta värskendataks selles olekus automaatselt.</span><span class="sxs-lookup"><span data-stu-id="77e95-120">Select an asset state in the **Lifecycle state** field if you want the asset lifecycle state for all assets installed on the functional location to be automatically updated at this state.</span></span> <span data-ttu-id="77e95-121">Näide: kui sulete funktsionaalse asukoha ja määrate funktsionaalse asukoha elutsükli oleku väärtuseks „Lõpetatud“, võite soovida automaatselt muuta selle funktsionaalse asukoha installitud varade elutsükli olekuks „Pole kasutuses“.</span><span class="sxs-lookup"><span data-stu-id="77e95-121">Example: If you close down a functional location, and set the functional location lifecycle state to "Ended", you may want to automatically change the lifecycle state of the assets installed on that functional location to "Not in use".</span></span>


>[!NOTE]
><span data-ttu-id="77e95-122">Funktsionaalsed asukoha elutsükli olekud, elutsükli mudelid ja tüübid on seotud ja neid kasutatakse samamoodi nagu töötellimuse elutsükli olekud, töökäsu elutsükli mudelid ja töötellimuse tüübid.</span><span class="sxs-lookup"><span data-stu-id="77e95-122">Functional location lifecycle states, lifecycle models, and types are related and used in the same way as work order lifecycle states, work order lifecycle models, and work order types.</span></span> 

## <a name="set-up-functional-location-lifecycle-models"></a><span data-ttu-id="77e95-123">Funktsionaalse asukoha elutsükli mudelite sätestamine</span><span class="sxs-lookup"><span data-stu-id="77e95-123">Set up functional location lifecycle models</span></span>

2. <span data-ttu-id="77e95-128">Valige uue elutsükli mudeli loomiseks **Uus**.</span><span class="sxs-lookup"><span data-stu-id="77e95-128">Select **New** to create a new lifecycle model.</span></span>
