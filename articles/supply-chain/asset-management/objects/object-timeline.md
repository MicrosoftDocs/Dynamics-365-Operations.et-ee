# <a name="asset-event-history"></a>Vara sündmuse ajalugu

[!include [banner](../../includes/banner.md)]

 

See artikkel selgitab, kuidas pääseda vara sündmuse ajalukku varahalduses. Lehel **Vara sündmuste ajalugu** kuvatakse vara kasutusaja jooksul tehtud registreerimisajalugu. Sellele lehele pääsete juurde menüü-üksustest **Kõik varad**, **Aktiivsed varad** ja **Minu aktiivsed varad**. Valige vara ja seejärel valige **Sündmuste ajalugu**.

Kiirkaardil **Üksikasjad** lehel **Vara sündmuste ajalugu** saate otsida kogu saadaolevat teavet. Näiteks saate kasutada vara sündmuste ajalugu leida järgmine teave:

- Millal töö tüüpi viimati varal kasutati
- Millal sisestas konkreetne töötaja töökäsu töö kohta märkuse

Ajaskaala värskendatakse iga kord, kui leht avatakse. See sisaldab järgmisi teemasid.

- Varale tehtud muudatused: vara asukoht, vara ID, nimi ja hankija garantii
- Emavara
- Vara kooslus (BOM)
- Vara elutsükli oleku logi
- Funktsionaalsed asukohad
- Puudumistaotluste loomine
- Töökäsud, sh konteeritud kaup ja märkmed
- Tõrge
- Seisundi hindamine
