2. <span data-ttu-id="cd4e2-107">Klõpsake loendis valitud real olevat linki.</span><span class="sxs-lookup"><span data-stu-id="cd4e2-107">In the list, click the link in the selected row.</span></span>

4. <span data-ttu-id="cd4e2-109">Klõpsake Kauba toiming.</span><span class="sxs-lookup"><span data-stu-id="cd4e2-109">Click Item task.</span></span>

9. <span data-ttu-id="cd4e2-115">Klõpsake nuppu OK.</span><span class="sxs-lookup"><span data-stu-id="cd4e2-115">Click OK.</span></span>
10. <span data-ttu-id="cd4e2-116">Märkige loendis valitud rida.</span><span class="sxs-lookup"><span data-stu-id="cd4e2-116">In the list, mark the selected row.</span></span>
