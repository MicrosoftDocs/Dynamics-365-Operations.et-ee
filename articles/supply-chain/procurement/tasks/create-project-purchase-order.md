<span data-ttu-id="aeccb-104">See protseduur näitab, kuidas luua projekti ostutellimust.</span><span class="sxs-lookup"><span data-stu-id="aeccb-104">This procedure shows you how to create a project purchase order.</span></span> <span data-ttu-id="aeccb-105">Ülesandes kasutatakse USSI andmekomplekti.</span><span class="sxs-lookup"><span data-stu-id="aeccb-105">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="aeccb-106">Avage Projektihaldus ja -arvestus > Projektid > Kõik projektid.</span><span class="sxs-lookup"><span data-stu-id="aeccb-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="aeccb-107">Klõpsake loendis valitud real olevat linki.</span><span class="sxs-lookup"><span data-stu-id="aeccb-107">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="aeccb-108">Klõpsake tegumiribal valikut Haldamine.</span><span class="sxs-lookup"><span data-stu-id="aeccb-108">On the Action Pane, click Manage.</span></span>
4. <span data-ttu-id="aeccb-109">Klõpsake Kauba toiming.</span><span class="sxs-lookup"><span data-stu-id="aeccb-109">Click Item task.</span></span>
5. <span data-ttu-id="aeccb-110">Klõpsake valikut Ostutellimus.</span><span class="sxs-lookup"><span data-stu-id="aeccb-110">Click Purchase order.</span></span>
6. <span data-ttu-id="aeccb-111">Sisestage väärtus väljale Hankija konto või valige see.</span><span class="sxs-lookup"><span data-stu-id="aeccb-111">In the Vendor account field, enter or select a value.</span></span>

    * <span data-ttu-id="aeccb-113">Need toimingud ei ole kohustuslikud, kuid need lihtsustavad ostutellimust, häälestades ostutellimuste jaoks vaikimisi tegevuskoha või lao.</span><span class="sxs-lookup"><span data-stu-id="aeccb-113">These steps aren't required, but they do simplify the purchase order by setting up a default site and warehouse for the purchase order lines.</span></span>  
8. <span data-ttu-id="aeccb-114">Sisestage või valige väärtus väljal Ladu.</span><span class="sxs-lookup"><span data-stu-id="aeccb-114">In the Warehouse field, enter or select a value.</span></span>
9. <span data-ttu-id="aeccb-115">Klõpsake nuppu OK.</span><span class="sxs-lookup"><span data-stu-id="aeccb-115">Click OK.</span></span>
10. <span data-ttu-id="aeccb-116">Märkige loendis valitud rida.</span><span class="sxs-lookup"><span data-stu-id="aeccb-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="aeccb-117">Sisestage või valige väärtus väljal Kaubakood.</span><span class="sxs-lookup"><span data-stu-id="aeccb-117">In the Item number field, enter or select a value.</span></span>
    * <span data-ttu-id="aeccb-118">See saab olla kaubakood või hankekategooria.</span><span class="sxs-lookup"><span data-stu-id="aeccb-118">This can be the item number or a procurement category.</span></span>

13. <span data-ttu-id="aeccb-120">Klõpsake vahekaart Projekt.</span><span class="sxs-lookup"><span data-stu-id="aeccb-120">Click the Project tab.</span></span>
    * <span data-ttu-id="aeccb-121">Kontrollige, kas müügi- ja kuluhinnad on saadaval.</span><span class="sxs-lookup"><span data-stu-id="aeccb-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="aeccb-122">Kui need pole saadaval, kuid on vajalikud, sisestage teave.</span><span class="sxs-lookup"><span data-stu-id="aeccb-122">If they are not available but needed, enter the information.</span></span>  
14. <span data-ttu-id="aeccb-123">Klõpsake nuppu Salvesta.</span><span class="sxs-lookup"><span data-stu-id="aeccb-123">Click Save.</span></span>
