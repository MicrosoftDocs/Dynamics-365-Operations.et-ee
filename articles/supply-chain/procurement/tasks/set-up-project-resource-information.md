# <a name="set-up-project-resource-information"></a>Projekti ressursiteabe seadistamine

[!include [banner](../../includes/banner.md)]

See protseduur näitab, kuidas häälestada projekti ressurssi. Selles ülesanded kasutatakse USMF andmekogumit.

1. Avage Projektihaldus ja raamatupidamine > Projekti ressursid > Ressursside loend.
2. Klõpsake nuppu Uus ressurss.
3. Valige või sisestage väärtus väljal Töötaja.
4. Sisestage või valige väärtus väljal Kalender.
5. Sisestage või valige väärtus väljal Perioodi kood.
6. Klõpsake käsku Loo.
7. Klõpsake nuppu Omahinnad.
8. Klõpsake nuppu Omahind – tund.
9. Klõpsake valikut Uus.
10. Sisestage kuupäev väljale Jõustumiskuupäev.
11. Sisestage arv väljale Omahind.
12. Klõpsake nuppu Salvesta.
13. Sulgege leht.
14. Klõpsake nuppu Müügihinnad.
15. Klõpsake nuppu Müügihind – tund.
16. Klõpsake valikut Uus.
17. Sisestage kuupäev väljale Jõustumiskuupäev.
18. Väljale Hinnakujundus sisestage number.
19. Valige suvand väljal Kehtiv.
20. Klõpsake nuppu Salvesta.
21. Sulgege leht.
