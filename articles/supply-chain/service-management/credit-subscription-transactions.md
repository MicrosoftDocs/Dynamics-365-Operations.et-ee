## <a name="credit-subscription-transactions"></a><span data-ttu-id="c751e-104">Kordustellimuste kannete krediteerimine</span><span class="sxs-lookup"><span data-stu-id="c751e-104">Credit subscription transactions</span></span>

1.  <span data-ttu-id="c751e-105">Klõpsake valikuid **Teenuste haldus** \> **Üldine** \> **Teenuse kordustellimused** \> **Kõik teenuse kordustellimused**.</span><span class="sxs-lookup"><span data-stu-id="c751e-105">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="c751e-106">Valige kordustellimuse kandele lisatud kordustellimus, mille jaoks soovite luua kreeditarve.</span><span class="sxs-lookup"><span data-stu-id="c751e-106">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="c751e-107">Valige vahekaart **Analüüs** ja klõpsake siis toimingupaanil nuppu **Tasukanded**.</span><span class="sxs-lookup"><span data-stu-id="c751e-107">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="c751e-108">Valige vormilt **Tasukanded** kanne, mille jaoks soovite luua kreeditarve.</span><span class="sxs-lookup"><span data-stu-id="c751e-108">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="c751e-109">Klõpsake valikuid **Funktsioonid** \> **Vali kreeditarve jaoks**.</span><span class="sxs-lookup"><span data-stu-id="c751e-109">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="c751e-110">Valige vormilt **Kreeditarve jaoks valimine** kanne, mida soovite krediteerida ja seejärel klõpsake valikut **OK**.</span><span class="sxs-lookup"><span data-stu-id="c751e-110">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="c751e-111">Kui loote kreeditarve, veenduge, et valiksite <STRONG>Kreeditarved</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="c751e-111">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="c751e-112">Selle leiate loendist <STRONG>Arveldusviis</STRONG> dialoogiboksist <STRONG>Loo arve</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="c751e-112">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="c751e-113">Kui väli **Tühista viitvõlad krediteerimisel** vormis **Teenuste halduse parameetrid** on seatud valikule **Käsitsi**, peate tühistama iga viittulu kande ükshaaval, enne kui saate kandele luua kreeditarve soovituse.</span><span class="sxs-lookup"><span data-stu-id="c751e-113">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
