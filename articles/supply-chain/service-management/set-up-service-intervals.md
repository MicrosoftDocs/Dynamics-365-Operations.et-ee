# <a name="set-up-service-intervals"></a><span data-ttu-id="1776f-103">Hooldusintervallide seadistamine</span><span class="sxs-lookup"><span data-stu-id="1776f-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="1776f-104">Hooldusintervall näitab sagedust, millega hooldustellimuse ridu hooldusleppe ridade jaoks hooldustellimuste automaatsel loomisel luuakse.</span><span class="sxs-lookup"><span data-stu-id="1776f-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="1776f-105">Klõpsake valikut **Hooldushaldus** \> **Häälestus** \> **Hoolduslepped** \> **Hooldusintervallid**.</span><span class="sxs-lookup"><span data-stu-id="1776f-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="1776f-106">Looge uus hooldusintervall.</span><span class="sxs-lookup"><span data-stu-id="1776f-106">Create a new service interval.</span></span>
3. <span data-ttu-id="1776f-107">Sisestage hooldusintervalli kood ja kirjeldus.</span><span class="sxs-lookup"><span data-stu-id="1776f-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="1776f-108">Valige soovitud vahemik väljal **Vahemik**.</span><span class="sxs-lookup"><span data-stu-id="1776f-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="1776f-109">Väljale **Sagedus** tippige sagedus.</span><span class="sxs-lookup"><span data-stu-id="1776f-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="1776f-110">Sagedus on tegur, millega peate vahemiku soovitud hooldusleppe intervalli saamiseks korrutama.</span><span class="sxs-lookup"><span data-stu-id="1776f-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="1776f-111">Hooldusintervalli salvestamiseks vajutage klahve **Alt+S**.</span><span class="sxs-lookup"><span data-stu-id="1776f-111">Press **Alt+S** to save the service interval.</span></span>
