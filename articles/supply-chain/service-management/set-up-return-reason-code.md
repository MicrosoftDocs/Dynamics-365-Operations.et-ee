# <a name="set-up-return-reason-codes"></a>Tagastuspõhjuste koodide häälestus   

[!include [banner](../includes/banner.md)]


## <a name="set-up-return-reason-code-groups"></a>Tagastamise põhjuste koodigruppide seadistamine 

1.  Avage **Müük ja turundus** \> **Häälestamine** \> **Müügitellimused** \> **Tagastused** \> **Tagastuspõhjuste koodigrupid**.

2.  Uue tagastuspõhjuse koodi grupi rea loomiseks valige **Uus**.

3.  Tippige väljale **Tagastuspõhjuste koodigrupid** selle grupi jaoks identifikaator.

4.  Tippige väljale **Kirjeldus** lühike selgitav fraas selle grupi kasutamise kohta.

## <a name="set-up-return-reason-codes"></a>Tagastuspõhjuste koodide seadistamine 

See protseduur aitab teil seadistada põhjusekoodid, mille abil saate näidata, miks klient toote tagastas.

1.  Avage **Müük ja turundus** \> **Häälestamine** \> **Müügitellimused** \> **Tagastused** \> **Tagastuspõhjuste koodid**.

2.  Vajutage CTRL + N, et luua uus tagastuse põhjusekoodi rida.

3.  Sisestage väljale **Tagastuspõhjuste koodid** tekst koodi tuvastamiseks.

4.  Sisestage välja **Kirjeldus** tekst, et kirjeldada seda tagastuspõhjuse koodi.

5.  Kasutage ripploendit **Tagastuspõhjuste koodigrupp** koodi tagastuspõhjuse koodigruppi lisamiseks.
