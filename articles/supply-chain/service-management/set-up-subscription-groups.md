# <a name="set-up-subscription-groups"></a><span data-ttu-id="b8320-103">Saate häälestada kordustellimuste gruppe.</span><span class="sxs-lookup"><span data-stu-id="b8320-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="b8320-104">Klõpsake valikuid **Teenuste haldus** \> **Seadistus** \> **Teenuse kordustellimused** \> **Kordustellimuste grupid**.</span><span class="sxs-lookup"><span data-stu-id="b8320-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="b8320-105">Uue kordustellimuse grupi loomiseks vajutage klahve CTRL + N või klõpsake käsku **Uus**.</span><span class="sxs-lookup"><span data-stu-id="b8320-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="b8320-106">Sisestage väljale **Kordustellimuse grupp** uue kordustellimuse grupi nimi.</span><span class="sxs-lookup"><span data-stu-id="b8320-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="b8320-107">Valige väljal **Arveldamine** kordustellimuse arve esitamise sagedus.</span><span class="sxs-lookup"><span data-stu-id="b8320-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="b8320-108">Arve esitamise sagedust näitab perioodi kood, mis on määratud vormis **Perioodi tüübid**.</span><span class="sxs-lookup"><span data-stu-id="b8320-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="b8320-109">Tulu lisamiseks kordustellimuse kannetele valige märkeruut **Lisatulu** ja seejärel valige sagedus väljas **Lisandumine**.</span><span class="sxs-lookup"><span data-stu-id="b8320-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="b8320-110">Lisandumise sagedus põhineb vormil **Perioodid** seadistatud perioodi koodil.</span><span class="sxs-lookup"><span data-stu-id="b8320-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
