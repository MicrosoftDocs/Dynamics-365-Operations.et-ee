# <a name="move-service-order-lines"></a><span data-ttu-id="5bc87-103">Hooldustellimuse ridade teisaldamine</span><span class="sxs-lookup"><span data-stu-id="5bc87-103">Move service order lines</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="5bc87-104">Klõpsake valikuid **Teenusehaldus** \> **Üldine** \> **Hooldustellimused** \> **Hooldustellimused**.</span><span class="sxs-lookup"><span data-stu-id="5bc87-104">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="5bc87-105">Topeltklõpsake hooldustellimust, mille ridu soovite teisaldada teisele hooldustellimusele.</span><span class="sxs-lookup"><span data-stu-id="5bc87-105">Double-click the service order that contains the lines that you want to move to another service order.</span></span>

3.  <span data-ttu-id="5bc87-106">Klõpsake vormi **Hooldustellimused** toimingupaanil nuppu **Teisalda read**.</span><span class="sxs-lookup"><span data-stu-id="5bc87-106">In the **Service orders** form, on the Action Pane, click the **Move lines** button.</span></span>

4.  <span data-ttu-id="5bc87-107">Valige hooldustellimus, kuhu soovite read teisaldada, vormi **Hooldustellimuse ridade teisaldamine** väljal **Hooldustellimus**.</span><span class="sxs-lookup"><span data-stu-id="5bc87-107">In the **Move service order lines** form, in the **Service order** field, select the service order that you want to move the lines to.</span></span>

5.  <span data-ttu-id="5bc87-108">Kui soovite kustutada tühja hooldustellimuse, mille hooldustellimuse read teisaldasite, märkige ruut **Kustuta tühi hooldustellimus**.</span><span class="sxs-lookup"><span data-stu-id="5bc87-108">Select the **Delete empty service order** check box to delete the service order that you moved the service order lines from.</span></span>

6.  <span data-ttu-id="5bc87-109">Klõpsake nupul **OK**.</span><span class="sxs-lookup"><span data-stu-id="5bc87-109">Click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="5bc87-110">Hooldustellimuse ridu saate teisaldada ainult sama hoolduslepinguga seotud hooldustellimustele.</span><span class="sxs-lookup"><span data-stu-id="5bc87-110">You can move service order lines only between service orders that are attached to the same service agreement.</span></span></P>
