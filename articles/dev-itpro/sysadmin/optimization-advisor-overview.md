<span data-ttu-id="c7d04-116">Võimalused võivad olla ettevõttekohased või ettevõtteülesed, olenevalt seadistuse tüübist ja valideeritavatest andmetest.</span><span class="sxs-lookup"><span data-stu-id="c7d04-116">Opportunities can be company-specific or cross-company, depending on the type of setup and data that is being validated.</span></span> <span data-ttu-id="c7d04-117">Ettevõtteüleseid võimalusi saab vaadata kõikides ettevõtetes.</span><span class="sxs-lookup"><span data-stu-id="c7d04-117">Cross-company opportunities can be viewed from all companies.</span></span> <span data-ttu-id="c7d04-118">Konkreetse ettevõtte võimaluste vaatamiseks peate esmalt valima ettevõtte.</span><span class="sxs-lookup"><span data-stu-id="c7d04-118">To view the opportunities for a specific company, you must first select the company.</span></span>
