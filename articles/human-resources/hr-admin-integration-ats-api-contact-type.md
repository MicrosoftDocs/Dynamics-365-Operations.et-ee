# <a name="contact-type"></a>Kontakti tüüp

Selles teemas kirjeldatakse rakenduse Dynamics 365 Human Resources suvandikomplekti Kontakti tüüp.

Füüsiline nimi: mshr_logisticselectronicaddressmethodtype

See loetelu annab kandidaatide kontakti tüüpide väärtuste suvandikomplekti. 

| Väärtus | Silt | Kirjeldus |
| --- | --- | --- |
| 200000000 | None | Tüüpi pole valitud. |
| 200000001 | Telefon | Telefoninumber. |
| 200000002 | Meiliaadress | Meiliaadress. |
| 200000003 | URL | Veebisaidi URL. |
| 200000004 | Teleks | Teleksinumber. |
| 200000005 | Faks | Faksinumber. |
| 200000006 | Facebook | Facebooki konto. Tuvastanud kasuta ID. |
| 200000007 | Twitter | Twitteri konto. Tuvastatud @username järgi. |
| 200000008 | LinkedIn | LinkedIni konto. Tuvastatud kasutajanime järgi. |
