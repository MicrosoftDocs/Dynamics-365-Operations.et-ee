# <a name="create-custom-links-in-manager-self-service"></a>Kohandatud linkide loomine halduri iseteeninduses

[!include [banner](includes/preview-feature.md)]

Saate lisada kohandatud linke halduri iseteeninduse vahekaardil **Minu meeskond**. See funktsioon võimaldab teil pakkuda kiiret juurdepääsu olulisele teabele. See sarnaneb kohandatud linkide lisamisega töövõtja iseteeninduse vahekaardil **Minu teave**.

## <a name="enable-the-preview-feature"></a>Eelvaate funktsiooni lubamine

Selle funktsiooni kasutamiseks lubage **(Eelversioon) Kohandatud lingid halduri iseteeninduses** tööruumis **Funktsioonide haldus**. Lisateavet eelvaatefunktsioonide lubamise kohta vt [Funktsioonide haldus](hr-admin-manage-features.md).

## <a name="set-up-custom-links"></a>Kohandatud linkide seadistamine

1. Valige jaotises **Human Resourcesi parameetrid** suvand **Halduri iseteenindus**.

2. Jaotises **Halduritele linkide seadistamine** saate lisada, redigeerida või eemaldada linke. Saate ka grupeerida linke, et neid kuvataks grupina halduri iseteeninduses.

   ![Kohandatud linkide seadistamine halduri iseteeninduses](./media/hr-employee-manager-self-service-custom-links-setup.png)

3. Linkide kuvamiseks avage töövõtja iseteeninduse vahekaart **Minu meeskond**.

   ![Kohandatud linkide kuvamine halduri iseteeninduses](./media/hr-employee-manager-self-service-custom-links-view.png)
