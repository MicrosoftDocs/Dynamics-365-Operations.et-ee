# <a name="create-custom-links-in-manager-self-service"></a>Kohandatud linkide loomine juhi iseteeninduses


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Saate lisada kohandatud linke **halduri iseteeninduse** vahekaardil **Minu meeskond**. See funktsioon võimaldab teil pakkuda kiiret juurdepääsu olulisele teabele. See sarnaneb kohandatud linkide lisamisega **töövõtja iseteeninduse** vahekaardil **Minu teave**.

## <a name="enable-the-feature"></a>Funktsiooni lubamine

Selle funktsiooni kasutamiseks lubage **Kohandatud lingid halduri iseteeninduses** tööruumis **Funktsioonide haldus**. Lisateavet funktsioonide eelversioonide lubamise kohta vt [Funktsioonide haldamine](hr-admin-manage-features.md).

## <a name="set-up-custom-links"></a>Kohandatud linkide seadistamine

1. Valige jaotises **Human Resourcesi parameetrid** suvand **Halduri iseteenindus**.

2. Jaotises **Halduritele linkide seadistamine** saate lisada, redigeerida või eemaldada linke. Linke saate grupeerida ka nii, et need kuvatakse halduri iseteeninduse **grupis**.

   ![Kohandatud linkide seadistamine halduri iseteeninduses.](./media/hr-employee-manager-self-service-custom-links-setup.png)

3. Linkide näha, minge töötaja **iseteeninduse** vahekaardile **Minu meeskond**.

   ![Kohandatud linkide kuvamine halduri iseteeninduses.](./media/hr-employee-manager-self-service-custom-links-view.png)
