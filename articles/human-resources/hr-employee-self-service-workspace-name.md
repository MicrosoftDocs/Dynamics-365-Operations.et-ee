[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="e337b-104">Kui teil on vabatahtlikke või teisi mitte-töövõtjaid, võite soovida muuta tööruumi **Töövõtja iseteenindus** nime.</span><span class="sxs-lookup"><span data-stu-id="e337b-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="e337b-105">Saate muuta selle tööruumi nimeks **Iseteenindus**.</span><span class="sxs-lookup"><span data-stu-id="e337b-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="e337b-106">Tööruumi **Töövõtja iseteenindus** nime muutmine muudab ka Dynamics 365 Human Resourcesi sisemiselt kasutatavat menüükäsku.</span><span class="sxs-lookup"><span data-stu-id="e337b-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="e337b-107">Kui olete eelnevalt rakendanud menüü-üksusele **HcmEmployeeSelfServiceWorkspace** turvalisuse kohandusi, soovitame rakendada samad muudatused ka üksusele **HcmSelfServiceWorkspace**, et säilitada paarsust.</span><span class="sxs-lookup"><span data-stu-id="e337b-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

2. <span data-ttu-id="e337b-109">Valige vahekaart **Töövõtja iseteenindus**.</span><span class="sxs-lookup"><span data-stu-id="e337b-109">Select the **Employee self-service** tab.</span></span>

3. <span data-ttu-id="e337b-110">Valige jaotises **Kuvatav nimi** suvand **Iseteenindus**.</span><span class="sxs-lookup"><span data-stu-id="e337b-110">Under **Display name**, select **Self service**.</span></span>

   ![Töövõtja iseteeninduse tööruumi nime muutmine Iseteeninduseks](./media/hr-employee-self-service-workspace-name.png)
