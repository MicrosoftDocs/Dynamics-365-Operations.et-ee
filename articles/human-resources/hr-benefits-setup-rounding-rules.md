# <a name="set-up-rounding-rules"></a>Ümardamisreeglite seadistamine

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Saate seadistada ümardamise reeglid rakenduses Dynamics 365 Human Resources oma organisatsiooni vajaduste rahuldamiseks.

1. Tööruumis **Soodustuste haldus** jaotises **Seadistus** valige suvand **Ümardamise reeglid**.

2. Valige suvand **Uus**.

3. Määratlege suvandite **Ümarda lähima väärtuseni**, **Stsenaarium** ja **Ümardamisvorm**.

4. Valige käsk **Salvesta**.
