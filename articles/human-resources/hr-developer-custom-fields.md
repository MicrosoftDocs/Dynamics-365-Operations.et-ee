# <a name="custom-fields"></a>Kohandatud väljad

Saate rakenduses Dynamics 365 Human Resources lisada lehtedele kohandatud väljasid. Lisateavet lehel kohandatud väljade loomise kohta leiate teemast [Kohandatud väljade loomine ja nendega töötamine](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

Järgmised rakenduse Common Data Service üksused toetavad rakenduses Human Resources loodud kohandatud kliendiväljasid.

- Töötaja
- Etniline päritolu
- Veterani staatus
- Keelekood
- Amet
- Töö tüüp
- Tööfunktsioon
- Ametikoht
- Positsiooni tüüp

Lubage kindlasti kõik loodavad kohandatud väljad, et neid rakenduses Common Data Service kasutada. Lisateavet vt teemast [Kohandatud väljade esitamine andmeüksustele](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).
