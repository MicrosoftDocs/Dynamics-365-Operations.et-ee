# <a name="applicant-integration-result"></a>Kandidaadi integratsiooni tulemus


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

See artikkel kirjeldab kandidaadi integratsiooni tulemuste valikukomplekti Dynamics 365 Human Resources.

Füüsiline nimi: mshr_hcmapplicantintegrationresult

See loetelu annab kandidaadi kirjele oleku.

| Väärtus | Silt | Kirjeldus |
| --- | --- | --- |
| 200000000 | Töötlemata | Kandidaat on endiselt kaalumisel. |
| 200000001 | Palgatud | Kandidaat on palgatud. |
| 200000002 | Palkamata | Otsustati kandidaati mitte palgata. |
| 200000003 | Lõpetatud | Kandidaat jäeti kaalumiselt välja. |
