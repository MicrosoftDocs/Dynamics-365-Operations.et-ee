# <a name="generate-payroll-entities"></a>Palgaarvestuse üksuste loomine

Selle funktsiooni OData abil saate luua palgaarvestuse integreerimiseks vajalikud olemid. Kui neid olemeid muudetakse personaliosakonnas (nt kohandatud väljade lisamine), saab seda funktsiooni uuesti kutsuda iga olemi metaandmete värskendamiseks. Vastus sisaldab toimingu ID-d, mida saate jälgida, et teaksite, kui genereerimisprotsess on lõppenud.

**Taotlus**

```http
GET [Organizaton URI]/api/data/v9.1/RefreshHumanResourcesVirtualEntities
```

**keha**

```json
{
    "PhysicalNames" : ["PayrollEmployeeEntity", "HcmWorkerBaseEntity", "PayrollPositionEntity", "PayrollPositionJobEntity", "PayrollWorkerAddressEntity", "HcmJobDetailEntity", "HcmCompFixedPlanTableEntity", "PayrollFixedCompensationPlanEntity", "HcmEmploymentDetailEntity"]
}
```

**Vastus**

```json
{
    "AsyncOperationId": "8b98d338-f939-4c86-9a91-80b76b6ab2ea"
}
```
