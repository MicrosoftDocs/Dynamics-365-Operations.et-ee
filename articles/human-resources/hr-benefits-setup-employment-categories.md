# <a name="create-employment-categories"></a>Töötajate kategooriate loomine


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Seadistage rakenduses Dynamics 365 Human Resources teie töötajatega ühtivad tööhõive kategooriad.

1. Tööruumis **Soodustuste haldus** jaotises **Seadistus** valige suvand **Tööhõive kategooriad**.

2. Valige suvand **Uus**.

3. Määrake suvandite **Töötaja tüüp**, **Tööhõive kategooria** ja **Kirjeldus** väärtus.

4. Valige käsk **Salvesta**.
