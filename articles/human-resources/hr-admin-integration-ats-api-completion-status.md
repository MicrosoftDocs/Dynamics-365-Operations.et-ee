# <a name="completion-status"></a>Lõpuleviimise olek

Selles teemas kirjeldatakse rakenduse Dynamics 365 Human Resources suvandikomplekti Lõpuleviimise olek.

Füüsiline nimi: mshr_hcmcompletionstatus

See loetelu annab kandidaatide skriiningute oleku väärtuste suvandikomplekti. 

| Väärtus | Silt | Kirjeldus |
| --- | --- | --- |
| 200000000 | Lõpetamata | Kandidaat ei ole skriiningut veel lõpule viinud. |
| 200000001 | Edasta | Kandidaat on skriiningu läbinud. |
| 200000002 | Nurjumine | Kandidaat on skriiningu läbikukkunud. |
