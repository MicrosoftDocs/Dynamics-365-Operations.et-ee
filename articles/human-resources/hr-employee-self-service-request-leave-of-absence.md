# <a name="request-a-leave-of-absence"></a>Puhkuse taotlemine

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Saate esitada puhkusetaotluse ja näha oma puhkusetaotluste olekut rakenduses Dynamics 365 Human Resources.

## <a name="request-a-leave-of-absence"></a>Puhkuse taotlemine

1. Tööruumis **Töövõtja iseteenindus** valige suvand **Lisateave** (…) paanil **Vaba aja saldod**.

2. Puhkusetaotluse esitamiseks valige **Puhkuse taotlemine**.

3. Sisestage teave suvandite **Puhkusetüüp**, **Alguskuupäev** ja **Lõppkuupäev** kohta.

4. Kui peate esitama mis tahes lisadokumente, valige käsk **Laadi üles** jaotises **Manused**.

5. Vajadusel sisestage teave väljale **Kommentaar**.

6. Kui olete taotluse esitamiseks valmis, valige suvand **Esita**. Muidu valige suvand **Salvesta mustand**.


## <a name="view-leave-of-absence-request-status"></a>Puhkusetaotluse oleku vaatamine

1. Tööruumis **Töövõtja iseteenindus** valige suvand **Lisateave** (…) paanil **Vaba aja saldod**.

2. Puhkusetaotluste vaatamiseks valige **Puhkuse kuvamine**.
