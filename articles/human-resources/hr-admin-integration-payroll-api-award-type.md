## <a name="option-set-award-type"></a>Valikukomplekt: preemia tüüp

Füüsiline nimi: mshr_HrmCompVarAwardEmplType

See loetelu pakub inimressursside virtuaalsetes üksustes preemia tüübi jaoks seatud valiku.

| Väärtus | Silt | Kirjeldus |
| --- | --- | --- |
| 200000000 | Põhipreemia | Põhipreemia. |
| 200000001 | Korraga makstav fikseeritud tasu | Korraga makstav fikseeritud tasu. |
