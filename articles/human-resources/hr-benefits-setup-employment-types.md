# <a name="set-up-employment-types"></a>Töösuhte tüübi häälestamine


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Seadistage rakenduses Dynamics 365 Human Resources teie töötajatega ühtivad tööhõive tüübid.

1. Tööruumis **Personalihaldus** jaotises **Seadistus** valige suvand **Tööhõive tüübid**.

2. Valige suvand **Uus**.

3. Määrake järgmiste väljade väärtused.

   | Väli | Kirjeldus |
   | --- | --- |
   | **Töölevõtu tüüp** | Tööhõive tüübi kordumatu ID. |
   | **Kirjeldus** | Tööhõive tüübi kirjeldus. |

4. Valige käsk **Salvesta**.
