# <a name="assign-workers-to-a-leave-plan"></a>Puhkuseplaanile töötajate määramine

Kui olete rakenduses Dynamics 365 Human Resources loonud puhkuse ja puudumise plaani, määrake see töötajatele.

## <a name="assign-a-plan-to-a-group-of-employees"></a>Plaani määramine töötajate rühmale

1. Lehel **Puhkused ja puudumine** valige plaan, mille soovite määrata.

2. Valige suvand **Määra töövõtjatele**.

3. Muutke vastavalt vajadusele järgmisi sätteid.

   - **Registreerumine töösuhte alguskuupäeval**
   - **Registreerimise kuupäev**
   - **Lõppkuupäev**
   - **Taseme alus**

4. Jaotises **Registreeritavad töötaja** valige kõik plaanile määratavad tööd.

5. Valige lehe allosas käsk **Määra plaan**.

## <a name="assign-a-plan-to-an-employee"></a>Töövõtjale plaani määramine

1. Töövõtja kirjes valige suvand **Puhkus**.

2. Valige suvand **Puhkused ja puudumised**.

3. Valige suvand **Uus**.

4. Sisestage registreerimiskuupäev ja valige puhkuseplaan.
