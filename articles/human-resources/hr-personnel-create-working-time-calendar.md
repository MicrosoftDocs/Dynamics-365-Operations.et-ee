# <a name="create-calendars-and-generate-working-times"></a>Kalendrite loomine ja tööaegade genereerimine

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

> [!IMPORTANT]
> Selles artiklis märgitud funktsioon on praegu saadaval Inimressursside klientidele finants infrastruktuuris.  


Kalendrid kirjeldavad operatsiooniressursside võimsust ja tööaegu. Selles artiklis selgitatakse, kuidas määratleda tööajamalli põhjal töökalendrit. Saate selle protseduuriga tutvuda demoettevõtte USMF-i või omaenda andmeid kasutades.

1. Valige kodulehel **Ressursi elutsükli haldus**.
2. Valige **Kalendrid**.
3. Valige suvand **Uus**.
4. Väljal **Kalender** liigitage oma kalender. See on selle kalendri ID, mida kasutatakse viitena kalendrite määramisel näiteks operatsiooniressursile või ressursigrupile.  
5. **Sisestage** väljale Nimi oma kalendri nimi.
6. Sisestage number väljale **Standardne tööpäev tundides**.
7. Veenduge, et rida on valitud ja seejärel valige **tegevuspaanilt** tööajad **·**.
8. Valige **Koosta tööajad**. Looge töötunnid iga päeva kohta perioodil, millal on vaja tööd plaanida. Aja möödudes saate luua tööaegu täiendavatele perioodidele.  
9. Sisestage kuupäev väljale **Alates kuupäevast**. See on esimene päev, millal see kalender peab olema avatud.  
10. Sisestage kuupäev väljale **Lõppkuupäev**. See on viimane päev, millal see kalender avatud on.  
11. Valige või sisestage väärtus väljal **Tööajamall**. Tööajamall määratleb iga nädalapäeva töötunnid.  
12. Valige nupp **OK**.
13. Sulgege leht.
