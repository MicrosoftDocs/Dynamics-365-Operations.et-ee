# <a name="screening-frequency"></a>Skriiningu sagedus

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Selles teemas kirjeldatakse skriiningu sageduse suvandikomplekti rakendusele Dynamics 365 Human Resources.

Füüsiline nimi: mshr_hcmfrequencyunit

See loetelu annab skriiningu sageduse väärtuste suvandikomplekti. 

| Väärtus | Silt | Kirjeldus |
| --- | --- | --- |
| 200000000 Ainult ühekordne | Skriining on nõutav ainult üks kord. |
| 200000001 Igapäevane | Skriiningu sagedust arvutatakse päevades. |
| 200000002 Iganädalane | Skriiningu sagedust arvutatakse nädalates. |
| 200000003 Igakuine | Skriiningu sagedust arvutatakse kuudes. |
| 200000004 Aastane | Skriiningu sagedust arvutatakse aastates. |
