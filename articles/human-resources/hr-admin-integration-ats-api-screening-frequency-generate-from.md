Selles teemas kirjeldatakse skriiningu sageduse loomise vormi suvandikomplekti rakendusele Dynamics 365 Human Resources.

Füüsiline nimi: mshr_hcmfrequencygeneratefrom

See loetelu annab järgmise vajaliku skriiningu alguskuupäeva arvestuse määramise väärtuste suvandikomplekti.

| Väärtus | Silt | Kirjeldus |
| --- | --- | --- |
| 200000000 Pole valitud | Väärtust pole valitud. |
| 200000001 Lõpetamiskuupäev | Arvestus tehakse alates viimase lõpuleviidud skriiningu kuupäevast. |
