# <a name="suspend-leave"></a><span data-ttu-id="18cfc-103">Puhkuse katkestamine</span><span class="sxs-lookup"><span data-stu-id="18cfc-103">Suspend leave</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="18cfc-104">Saate katkestada töötaja puhkuse, et lõpetada puhkuse viitvõlgade töötlemine valitud puhkuse tüüpide korral.</span><span class="sxs-lookup"><span data-stu-id="18cfc-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="18cfc-105">Ühe töötaja puhkuse ja puudumise katkestamine</span><span class="sxs-lookup"><span data-stu-id="18cfc-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="18cfc-106">Töövõtja kirjes valige suvand **Puhkus**.</span><span class="sxs-lookup"><span data-stu-id="18cfc-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="18cfc-107">Valige **Puhkuse katkestamine**.</span><span class="sxs-lookup"><span data-stu-id="18cfc-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="18cfc-108">Valige suvand **Uus**.</span><span class="sxs-lookup"><span data-stu-id="18cfc-108">Select **New**.</span></span>

4. <span data-ttu-id="18cfc-109">Tehke dialoogiboksis **Puhkuse viitvõla katkestamine** valik **Puhkuse tüüp** ja seejärel valige peatamise **Alguskuupäev** ja **Lõppkuupäev**.</span><span class="sxs-lookup"><span data-stu-id="18cfc-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

5. <span data-ttu-id="18cfc-110">Soovi korral saate katkestamisele lisada **kommentaari**.</span><span class="sxs-lookup"><span data-stu-id="18cfc-110">Optionally, you can add a **Comment** for the suspension.</span></span> 

<span data-ttu-id="18cfc-111">Kui viitvõlgu töödeldakse töötaja puhkuse katkestamise ajal, ei arvestata viitvõlgu katkestatud puhkuse tüüpide kohta.</span><span class="sxs-lookup"><span data-stu-id="18cfc-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
