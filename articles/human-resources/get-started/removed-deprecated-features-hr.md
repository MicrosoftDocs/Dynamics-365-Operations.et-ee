# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>Dynamics 365 inimressurssidest on eemaldatud või taunitud funktsioone.

See artikkel kirjeldab funktsioone, mis on eemaldatud või planeeritud Microsoft Dynamics 365 inimressurssidest eemaldamiseks.

- *Eemaldatud* funktsioon pole tootes enam saadaval.
- *Aegunud* funktsioon ei ole aktiivses arenduses ja vee võidakse tulevases värskenduses eemaldada.

See loend peaks aitama teil neid eemaldusi ja aegumisi oma plaanides arvesse võtta.

## <a name="features-removed-or-deprecated-in-human-resources"></a>Inimressurssides eemaldatud või taunitud funktsioonid 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>Uus LCS-i projekti loomine inimressurssides (eraldiseisev) 

[Inimressursside ettevalmistamine](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Aegumise/eemaldamise põhjus** | Asendatud projekti loomisega finantside ja toimingute infrastruktuuris Lisateavet infrastruktuuri ühendamise kohta vt Dynamics [365 Inimressursside infrastruktuuri ühendamise KKK-st](/hr-infrastructure-merge-faq). |
| **Asendatud teise funktsiooniga?**   | Jah |
| **Mõjutatud tootealad**         | Avaldus |
| **Juurutamissuvand**              | Kõik |
| **Olek**                         | Taunitud: alates juulist 2022 ei toeta me enam uute projektide loomist Inimressurssides (eraldiseisev). |

### <a name="production-environment-provisioning-on-human-resources-stand-alone"></a>Tootmiskeskkonna ressursi ressursistamine inimressurssides (eraldiseisev) 

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Aegumise/eemaldamise põhjus** | Asendatud keskkonna ettevalmistamisega finantside ja toimingute infrastruktuuris Lisateavet infrastruktuuri ühendamise kohta vt Dynamics [365 Inimressursside infrastruktuuri ühendamise KKK-st](/hr-infrastructure-merge-faq.md). |
| **Asendatud teise funktsiooniga?**   | Jah |
| **Mõjutatud tootealad**         | Avaldus |
| **Juurutamissuvand**              | Kõik |
| **Olek**                         | Taunitud: alates juulist 2022 ei toeta me enam inimressursside tootmiskeskkondade ressursina kasutamist (eraldiseisev). |
