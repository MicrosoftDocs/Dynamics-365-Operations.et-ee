# <a name="get-the-onboard-app"></a>Rakenduse Onboard hankimine

[!include [banner](includes/banner.md)]

Saate vaadata demo ja proovida rakendust Microsoft Dynamics 365 Talent: Onboard tasuta lehelt [Onboardi tooteleht.](https://dynamics.microsoft.com/talent/onboard/)

> [!NOTE]
> Tasuta prooviversioon nõuab ettevõtte meilikontot.

Saate osta Onboardi tellimuse, mis on kas eraldiseisev rakendus või osa rakendusest Dynamics 365 Talent. Talent on terviklik personalijuhtimise süsteem, mis sisaldab rakendust Dynamics 365 Talent: Attract, Onboard ja Core HR. Lisateavet selle kohta, kuidas Onboardi osta, leiate lehelt [Onboardi tooteleht](https://dynamics.microsoft.com/talent/onboard/).

Prooviversiooni või ostu protsessi käigus seadistate oma Microsoft 365 meiliaadressi ja parooli. Märkige kindlasti need väärtused.

> [!WARNING]
> Te ei saa oma prooviversiooni andmeid tasulisse tellimiskeskkonda üle viia. <!--Reviewers: please verify.-->

Teavet Talenti uute funktsioonide kohta vt teemast [Mis on uut või mida on muudetud rakenduses Dynamics 365 Talent](./whats-new.md) ja [Dynamics 365 ja Power Platformi väljalaskemärkmed](https://docs.microsoft.com/business-applications-release-notes/index). Kui soovite näha Onboardi uute funktsioonide eelvaadet, vt [Juurdepääs rakenduse Microsoft Dynamics 365 Talent eelvaatefunktsioonidele](./access-preview-feature.md).

Kui olete IT-spetsialist ja soovite lisateavet rakenduse Onboard kahe versiooni ettevalmistamise kohta, lugege teemat [Dynamics 365 Talent – rakenduse Onboard ettevalmistamine](./modular-app-tech-faq.md).

## <a name="get-started-with-onboard"></a>Onboardi kasutamise alustamine

Kui te esimest korda Onboardi avate, olete oodatud alustama Microsoft 365 halduskeskuse ringkäiku. Admin centeris saate seadistada organisatsiooni, hallata kasutajaid ja hallata oma tellimusi. (Üks neist tellimustest on teie Onboard tellimus.) Lisateavet Microsoft 365 halduskeskuse kohta vt [Microsoft 365 halduskeskusest](https://docs.microsoft.com/office365/admin/admin-overview/about-the-admin-center?view=o365-worldwide)

Rakenduse Onboard avamiseks toimige järgmiselt.

1. Avage [Microsofti sisselogimise leht](https://portal.office.com/).
2. Kui teilt küsitakse, sisestage oma Microsoft 365 meiliaadress ja parool.
3. Valige ülemises vasakpoolses rakendusekäiviti ja seejärel valige **Dynamics 365**.

    [![Dynamics 365 juurdepääs](./media/onboard-start-dynamics365.png)](./media/onboard-start-dynamics365.png)

4. Valige **Talent: Onboard** paanil.

    [![Onboardi avamine](./media/onboard-start-onboard.png)](./media/onboard-start-onboard.png)

Teie esimene sisselogimine võib kesta mõne minuti, sest keskkond tuleb lähtestada.

## <a name="try-the-walkthrough"></a>Proovige juhendavat tutvustust

Kui te esimest korda Onboardi avate, saate valida **Käivita kiirtutvustus**, et alustada töömalliga.

Kui jätate kiirtutvustuse vahele, saate sellele hiljem juurde pääseda, kui valite nupu **Spikker** (**?**) ja seejärel valite **Alusta**.

![[Onboardi kiirtutvustusega alustamine](./media/onboard-start-walkthrough.png)](./media/onboard-start-walkthrough.png)

## <a name="change-the-domain-name"></a>Domeeninime muutmine

Kui te aktsepteerite vaikimisi domeeninime, kui Onboardiga sisse logisite, saate selle hiljem teiseks domeenks muuta. (Domeeni vaikenimi lõpeb **onmicrosoft.com**.)

1. Avage [Microsofti sisselogimise leht](https://portal.office.com/).
2. Kui teilt küsitakse, sisestage oma Microsoft 365 meiliaadress ja parool.
3. Kui näete soovitust, et lisada enda domeen valikus **Teile soovitatud**, valige **Kuva soovitus** ja järgige viipasid. Kui te soovitust ei näe, valige vasakul asuvas menüüs käsk **Kuva kõik**, valige **Häälestus**, valige **Domeenid** ja seejärel valige kas **Lisa domeen** või **Osta domeen**. Seejärel järgige viipasid.
