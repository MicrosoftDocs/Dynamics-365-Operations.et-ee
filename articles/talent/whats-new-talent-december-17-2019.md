# <a name="whats-new-or-changed-in-dynamics-365-talent-december-17-2019"></a>Mis on Dynamics 365 Talent-is uut või mida on muudetud (17. detsember 2019)

Selles artiklis kirjeldatakse Dynamics 365 Talenti uusi või muutunud funktsioone.

## <a name="changes-in-attract"></a>Muutused Attractis

See väljaanne sisaldab väiksemaid vigadeparandusi rakendusele Dynamics 365 Talent: Attract.

## <a name="changes-in-onboard"></a>Muutused Onboardis

See väljaanne sisaldab väiksemaid vigadeparandusi rakendusele Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Core HR-i muudatused

Jaotises kirjeldatud muudatused rakenduvad järgunumbrile 8.1.2674. Sulgudes olevad numbrid mõnedes pealkirjades viitavad toenumbritele teenuses Microsoft Dynamics Lifecycle Services (LCS).

### <a name="print-performance-reviews"></a>Jõudluse ülevaadete printimine

Printimisjõudluse ülevaated on nüüd saadaval kõikides keskkondades. Selle funktsiooni lubamiseks kasutage tööruumi **Funktsioonihaldus**. Lisateavet funktsioonihalduse kohta vt [Funktsioonihalduse ülevaade](https://docs.microsoft.com/dynamics365/fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview). Lisateavet printimisjõudluse ülevaadete kohta vt teemat [Jõudluse ülevaated](https://docs.microsoft.com/dynamics365/talent/performance-management-overview#performance-reviews).

### <a name="retired-position-with-direct-reports-floats-in-org-diagram-as-an-open-position-385436"></a>Kustutatud asend otseste aruannete nn ujukitega avatud asendis organisatsiooni diagrammis (385436)

Selle nädala väljaanne parandab viisi, kuidas kustutatud asendeid otseste aruannetega aruandluse hierarhias kuvatakse.
 
### <a name="system-unable-to-generate-calendar-days-381195"></a>Süsteem ei saa luua kalendripäevi (381195) 

Selle nädala muudatused parandavad probleemi, kus kalendripäevi ei õnnestunud luua.

### <a name="current-balance-report-in-leave-pbi-doubles-employee-balance-385581"></a>Puhkuse PBI jooksva saldo aruanne kahekordistab töötaja saldot (385581)

See muudatus parandab selle stsenaariumi jooksva saldo aruande.
