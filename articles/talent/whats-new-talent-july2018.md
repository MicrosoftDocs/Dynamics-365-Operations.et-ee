# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-july-2018"></a>Mis on uus või muudetud Dynamics 365 for Talent Core HR-is (juuli 2018)

[!include [banner](includes/banner.md)]

Teema kirjeldab funktsioone, mis on Microsoft Dynamics 365 for Talent Core HR-is kas uued või muudetud.

## <a name="powerapps-personalization"></a>PowerAppsi isikupärastamine

Microsoft Dynamics 365 for Talent toetab integreerimist Microsoft PowerAppsi teenusega. PowerApps võimaldab nii arendajatel kui ka mittetehnilistel kasutajatel luua kohandatud ärirakendusi mobiiliseadmete, tahvelarvutite ja veebi jaoks ilma koodi kirjutamata. Rakendusi, mida teie, teie organisatsioon või laiem ettevõtete „ökosüsteem” PowerAppsi abil arendab, saab lisada rakenduse Talent klienti, et võimendada toote funktsionaalsust. Näiteks võite luua rakenduse, mis täiendab Talenti teisest süsteemist toodud teabega.

Lisateabe saamiseks vt [PowerAppsi rakenduste manustamine](../fin-and-ops/get-started/embed-power-apps.md).

## <a name="ceridian-payroll-integration"></a>Ceridiani palgaarvestuse integreerimine

Integratsioon Talenti ja Ceridian Dayforce’i vahel on nüüd saadaval USAs, Kanadas ja Mehhikos. Integratsioon kasutab üldisi andmekategooriaid. Järgmisena on toodud mõned näited.

- Inimressursside andmed
- Kompensatsiooni andmed
- Palgaandmed, näiteks palgatsüklid, makseperioodid ja tulukoodid
- Töötaja andmed

Lisateabe saamiseks vt [Palgaarvestuse integratsiooni toetamine](configure-payroll-integration.md).

## <a name="worker-tax-regions-have-been-expanded-beyond-the-us"></a>Töötaja maksuregioone on laiendatud USAst kaugemale

Lisati väljaspool Ameerika Ühendriike olevate maksuregioonide tugi. Kui töötajatele määratakse maksuregioone, saavad nad reguleerida maksuarvutusi ja neid saab kasutada integratsioonides väliste palgaarvestuse lahendustega.

## <a name="the-title-field-has-been-expanded-in-talent"></a>Talentis laiendati tiitlivälja

Selles värskenduses laiendati tiitleid. Välja pikkus on nüüd 65 tähemärki. See muudatus juurutati kõikjal, kus valitud on tiitel, nt töötajate, ametikohtade ja tööde juures.

## <a name="benefit-enrollment-status-report"></a>Soodustuse registreerimise olekuaruanne

Integreeritud aruandlus soodustuste saamiseks avatud registreerimise kohta annab teile selge ülevaate, kus teie töövõtjad avatud registreerimise protsessis on. Saate teada, mitu töövõtjad on protsessi lõpule viinud, mitu töövõtjat seda praegu lõpule viivad ja mitu töövõtjat pole seda alustanud. Lisaks saate kiiresti vaadata võimalikke probleeme, mis tekitavad töövõtja registreerimise ajal, ja täielikku logi kõigi töövõtjate edastustest. Seetõttu saate hõlpsalt kinnitada ja auditeerida töövõtjate edastusi.
