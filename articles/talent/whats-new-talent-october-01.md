# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-october-1-2018"></a>Mis on Dynamics 365 for Talent Core HR-is uut või mida on muudetud (1. oktoober 2018)

[!include [banner](includes/banner.md)]

**Järk 8.1.1035.0**

Teema kirjeldab funktsioone, mis on Core HR-is kas uued või muudetud.

## <a name="turn-off-electronic-payment-validation"></a>Elektroonilise makse kinnitamise väljalülitamine

Elektroonilise makse teabe kinnitamine toimub, kui seadistate otsedeposiidi väljamaksed kas tööruumi **Töövõtja iseteenindus** (ESS) kaudu või otse töövõtja vormil. See suvand annab paindlikkuse eemaldada selle kinnituse, kui te summade ja jääkväärtuste kinnituse kontrolle enam ei vaja. Sellest funktsioonist on abi, kui integreerite teistsuguste nõuetega välise palgasüsteemiga.

## <a name="manager-self-service---add-goals-for-team-members-through-the-team-performance-goals-tile"></a>Juhataja iseteenindus – töörühma liikmetele eesmärkide lisamine paani Töörühma jõudluseesmärgid kaudu

Enne seda väljalaset saavad juhatajad lisada eesmärke oma töövõtjatele individuaalselt otse iga töövõtjaga seostatud jõudluseesmärkide kaudu. Selle värskendusega pääsevad juhatajad juurde paanile **Töörühma jõudluseemärgid** ja luua uusi eesmärke ilma otseseid alluvaid valimata.

## <a name="manager-self-service---add-reviews-for-team-members-through-the-team-performance-reviews-tile"></a>Juhataja iseteenindus – töörühma liikmetele ülevaadete lisamine paani Töörühma jõudlusülevaated kaudu

Enne seda väljalaset saavad juhatajad lisada ülevaateid oma töövõtjatele individuaalselt otse iga töövõtjaga seostatud jõudlusülevaadete kaudu. Selle värskendusega pääsevad juhatajad juurde paanile **Töörühma jõudlusülevaated** ja luua uusi ülevaateid ilma otseseid alluvaid valimata.

## <a name="configure-proration-options-by-leave-plan"></a>Proportsionaalse arvestuse suvandite konfigureerimine puhkuseplaani abil

Organisatsioonid tasustavad puhkeaega erinevalt töövõtjate organisatsiooniga liitumise ja sealt lahkumise põhjal. Mõne organisatsiooni puhul eraldatakse töötajatele alustamiskuupäeval preemia täisulatuses, teiste puhul toimub proportsionaalne jaotus. Selles väljaandes on saadaval uued võimalused preemiate ja arvestuste proportsionaalseks eraldamiseks organisatsiooniga liitujatele ning sellest lahkujatele. Suvandid on järgmised: Proportsionaalne arvestus, Täielik arvestus ja Arvestus puudub.

## <a name="other-changes"></a>Muud muudatused

-   Üksust Töötaja on värskendatud – pealkirja **Personal** saab nüüd värskendada Exceli lisandmooduli/andmehaldusega.

-   Turbemuudatus, mis võimaldab eemaldada nupud **Kustuta** ja **Inaktiveeri** töövõtja iseteeninduse makseteabe jaotisest.

## <a name="known-issue"></a>Teadaolev probleem

-   **Probleem:** töötajale uue manuse lisamisel on nupud **Uus** ja **Redigeeri** hallid. **Lahendus:** enne manuselehe avamist veenduge, et kiirinfod lehel **Töötaja** oleksid suletud. Kui kiirinfod on lehe **Töötaja** laadimisel suletud, on nupud **Manused** lubatud. (See probleem lahendatakse järgmisel platvormivärskendusel.)
