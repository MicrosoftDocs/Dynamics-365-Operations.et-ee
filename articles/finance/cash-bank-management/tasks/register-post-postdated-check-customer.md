# <a name="register-and-post-a-postdated-check-for-a-customer"></a>Kliendi hilisema kuupäevaga tšeki registreerimine ja sisestamine

[!include [banner](../../includes/banner.md)]

Saate registreerida kliendilt saadud hilisema kuupäevaga tšeki üksikasjad. Saate ka sisestada hilisema kuupäevaga dateeritud tšeki ja luua finantskanded.   Viige lõpule enne kliendilt saadud hilisema kuupäevaga tšeki registreerimist ja sisestamist järgmised ülesanded.   * Seadistage lehel Sularaha ja panga haldus hilisema kuupäevaga tšekid * Määrake hilisema kuupäevaga tšekkide jaoks maksemeetod   Seda toimingut tehakse rollis Laekur. See protsess kasutab demoettevõtte USMF-i andmeid.

1. Avage Müügireskontro > Maksed > Makse tööleht.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Nimi.
4. Klõpsake valikut Read.
5. Märkige loendis valitud rida.
6. Täpsustage soovitud väärtusi väljal Konto.
7. Sisestage arv väljale Kreedit.
    * Sisestage hilisema kuupäevaga dateeritud tšekis määratud summa.  
8. Klõpsake vahekaarti Makse.
9. Sisestage väärtus väljale Makseviis.
    * Valige makseviis hilisema kuupäevaga dateeritud tšekile.  
10. Klõpsake vahekaarti Hilisema kuupäevaga dateeritud tšekid.
11. Sisestage kuupäev väljale Tähtaja kuupäev.
    * Sisestage kuupäev, millal hilisema kuupäevaga dateeritud tšekk kuulub maksmisele.  
12. Sisestage väärtus väljale Väljaandva panga filiaal.
    * Sisestage hilisema kuupäevaga dateeritud tšeki panga üksikasjad.  
13. Sisestage väärtus väljale Tšeki number.
14. Sisestage väärtus väljale Väljaandva panga nimi.
    * Sisestage hilisema kuupäevaga dateeritud tšeki panga üksikasjad.  
15. Klõpsake valikut Sisesta.
16. Sulgege leht.
