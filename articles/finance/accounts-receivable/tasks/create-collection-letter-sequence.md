3. <span data-ttu-id="291a0-108">Sisestage seeriat tähistav seeria ID väljale **Märgukirja seeria**.</span><span class="sxs-lookup"><span data-stu-id="291a0-108">In the **Collection letter sequence** field, enter a sequence ID that will represent the sequence.</span></span> <span data-ttu-id="291a0-109">Seda kasutatakse sisestusreeglite seadistamiseks.</span><span class="sxs-lookup"><span data-stu-id="291a0-109">It will be used when you set up a posting profile.</span></span>

6. <span data-ttu-id="291a0-115">Sisestage väärtus väljale **Kirjeldus**.</span><span class="sxs-lookup"><span data-stu-id="291a0-115">In the **Description** field, type a value.</span></span> <span data-ttu-id="291a0-116">Tasu valuuta on vaikimisi kliendi valuutas.</span><span class="sxs-lookup"><span data-stu-id="291a0-116">The currency for the fee defaults to the customer currency.</span></span> <span data-ttu-id="291a0-117">Valuutakood võib arve valuutast erineda.</span><span class="sxs-lookup"><span data-stu-id="291a0-117">This currency code can be different than the invoice currency.</span></span>

8. <span data-ttu-id="291a0-121">Valige märgukirja koodi väljalt seerias järgmisena saadetav märgukiri.</span><span class="sxs-lookup"><span data-stu-id="291a0-121">In the collection letter code field, select the next collection letter that will be sent in the sequence.</span></span>
9. <span data-ttu-id="291a0-122">Sisestage väärtus väljale **Kirjeldus**.</span><span class="sxs-lookup"><span data-stu-id="291a0-122">In the **Description** field, type a value.</span></span>

13. <span data-ttu-id="291a0-127">Klõpsake loendis valitud real olevat linki.</span><span class="sxs-lookup"><span data-stu-id="291a0-127">In the list, click the link in the selected row.</span></span>
14. <span data-ttu-id="291a0-128">Väljale **Minimaalne tähtaja ületamise saldo** sisestage enne märgukirja saatmist nõutav minimaalne tähtaja ületamise saldo.</span><span class="sxs-lookup"><span data-stu-id="291a0-128">In the **Minimum overdue balance** field, enter the minimum overdue balance required before a collection letter is sent.</span></span>
15. <span data-ttu-id="291a0-129">Väljale **Päevad** sisestage ajapikenduspäevade arv, mida lubate.</span><span class="sxs-lookup"><span data-stu-id="291a0-129">In the **Days** field, enter the number of grace days that you will allow.</span></span> <span data-ttu-id="291a0-130">See on päevade arv pärast maksetähtaega, mil saab luua märgukirja.</span><span class="sxs-lookup"><span data-stu-id="291a0-130">This is the number of days after the due date that a collection letter can be generated.</span></span> <span data-ttu-id="291a0-131">Arvutamisel kasutatav maksetähtaeg oleneb märgukirja asukohast märgukirjaseerias järgmiselt: ⦁</span><span class="sxs-lookup"><span data-stu-id="291a0-131">The due date that is used for the calculation depends on the position of the collection letter in the collection letter sequence:</span></span>

    - <span data-ttu-id="291a0-133">2. ja edasiste märgukirjade ajapikenduse periood on seotud eelmise märgukirja sisestamise või printimise kuupäevaga, olenevalt välja Värskenda märgukirja koodi valikust lehel Müügireskontro parameetrid.</span><span class="sxs-lookup"><span data-stu-id="291a0-133">The grace period for collection letters 2 and higher is relative to the date that the previous collection letter is posted or printed, depending on the selection in the Update collection letter code field in the Accounts receivable parameters page.</span></span>

18. <span data-ttu-id="291a0-137">Sisestage väärtus väljale **Kirjeldus**.</span><span class="sxs-lookup"><span data-stu-id="291a0-137">In the **Description** field, type a value.</span></span>

22. <span data-ttu-id="291a0-141">Klõpsake loendis valitud real olevat linki.</span><span class="sxs-lookup"><span data-stu-id="291a0-141">In the list, click the link in the selected row.</span></span>
23. <span data-ttu-id="291a0-142">Sisestage number väljale **Minimaalne tähtaja ületanud saldo**.</span><span class="sxs-lookup"><span data-stu-id="291a0-142">In the **Minimum overdue balance** field, enter a number.</span></span>
24. <span data-ttu-id="291a0-143">Sisestage number väljale **Päevad**.</span><span class="sxs-lookup"><span data-stu-id="291a0-143">In the **Days** field, enter a number.</span></span>
