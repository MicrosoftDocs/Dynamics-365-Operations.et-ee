# <a name="view-related-intercompany-voucher-from-journal"></a>Seotud kontsernisisese kande kuvamine töölehelt

[!include [banner](../../includes/banner.md)]

Päevaraamatust kontsernisisese kande sisestamisel kuvatakse seotud kandeaknas vastaskonto ettevõttest pärinev kanne.


## <a name="post-an-intercompany-journal"></a>Kontsernisisese töölehe sisestamine
1. Avage üldine **tööleht, klõpsake** nuppu **Uus**.
2. Märkige loendis valitud rida.
3. Sisestage **või** valige väljal Nimi kontsernisisese töölehe nimi.
4. **Märkige** valitud rida loendis oleval real.
5. Täpsustage soovitud väärtusi väljal **Konto**.
6. Sisestage **või** valige väärtus väljal Kirjeldus.
7. Sulgege leht.
8. Sisestage arv väljale **Deebet**.
9. Tippige või **valige vastaskonto** ettevõte väljal Vastasettevõte.
10. Määratlega väljal **Vastaskonto** soovitud väärtused.
11. Klõpsake käsku **Sisesta**.

## <a name="view-related-intercompany-voucher"></a>Seotud kontsernisisese kande kuvamine
1. Klõpsake suvandit **Kanne**.
2. Valige **loendist seotud** kanded ja märkige valitud rida.
3. Klõpsake suvandit **Kanne**.
