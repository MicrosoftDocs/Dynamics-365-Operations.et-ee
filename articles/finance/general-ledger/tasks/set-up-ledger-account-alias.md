# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="be75e-103">Pearaamatukonto pseudonüümi seadistamine</span><span class="sxs-lookup"><span data-stu-id="be75e-103">Set up a ledger account alias</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="be75e-104">Selle protseduuriga kirjeldatakse kontopseudonüümi loomist, mis pakub otseteed kontonumbri sisestamiseks.</span><span class="sxs-lookup"><span data-stu-id="be75e-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="be75e-105">Protseduur kasutab demoettevõtte USMF andmeid.</span><span class="sxs-lookup"><span data-stu-id="be75e-105">This procedure uses demo data company USMF.</span></span>

1. <span data-ttu-id="be75e-106">Minge jaotisse Pearaamat > Kontoplaan > Kontod > Pearaamatukonto pseudonüüm.</span><span class="sxs-lookup"><span data-stu-id="be75e-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="be75e-107">Klõpsake valikut Uus.</span><span class="sxs-lookup"><span data-stu-id="be75e-107">Click New.</span></span>
3. <span data-ttu-id="be75e-108">Sisestage väärtus väljale Pearaamatukonto pseudonüüm.</span><span class="sxs-lookup"><span data-stu-id="be75e-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="be75e-109">Valige väljal Konto struktuur see struktuur, millesse konto ja dimensioonid kuuluvad.</span><span class="sxs-lookup"><span data-stu-id="be75e-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="be75e-110">Klõpsake väljal Ettevõte otsingu avamiseks ripploendi nuppu.</span><span class="sxs-lookup"><span data-stu-id="be75e-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="be75e-111">Leidke ja valige loendist ettevõte, millele pseudonüüm rakendub.</span><span class="sxs-lookup"><span data-stu-id="be75e-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="be75e-112">Klõpsake loendis valitud real olevat linki.</span><span class="sxs-lookup"><span data-stu-id="be75e-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="be75e-113">Määrake konto ja dimensioonid väljal Pearaamatukonto pseudonüümi määratlus.</span><span class="sxs-lookup"><span data-stu-id="be75e-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="be75e-114">Otsetee kasutamisel asustatakse konto ja dimensioonid.</span><span class="sxs-lookup"><span data-stu-id="be75e-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="be75e-115">Valige väljal Algne fookus see dimensioon, millele liigub fookus pseudonüümi kasutamisel.</span><span class="sxs-lookup"><span data-stu-id="be75e-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="be75e-116">Pärast otsetee sisestamist ning konto ja dimensioonide asustamist liigub kursor või fookus väljale Algne fookus.</span><span class="sxs-lookup"><span data-stu-id="be75e-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
