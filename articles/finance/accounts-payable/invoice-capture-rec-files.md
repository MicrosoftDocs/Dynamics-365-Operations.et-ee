# <a name="invoice-capture-received-files"></a>Arve hõivamise vastuvõetud failid

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

Arve hõivamise lehekülg Saadud **failid on** keskne koht, kus arvefailid on erinevatest allikatest vastu võetud.

## <a name="upload-invoice-files"></a>Laadi arvefailid üles

Ostureskontro (AP) ametnikud saavad arve pilte üles laadida, valides **faili üleslaadimise dialoogiakna** **avamiseks**. Pärast seda, kui AP-ametnik valib faili, muutub **nupp** Üleslaadimine kättesaadavaks.

## <a name="include-or-obsolete-invoice-files"></a>Kaasa või aegunud arvefailid

Kasutajad saavad valida vigade või hoiatustega arveid ning seejärel arveid kaasata või aegunuks muuta, valides tegevuspaanil vastava nupu. Aegunuks märgitud arvet ei kuvata vaates Saadud **failid (Aegunud**).

## <a name="view-captured-invoices"></a>Kuva hõivatud arved

Kui vastuvõetud fail on edukalt tuvastatud, tagastab hõivatud arve teave AI-mudeli ja sisestab selle kohta Microsoft Dataverse. Seejärel saab AP-ametnik arve üksikasju kontrollida, valides käsu **Kuva hõivatud arve**. Kasutajad saavad vajadusel algse arvefaili alla laadida.
