[!include [banner](../includes/banner.md)]

<span data-ttu-id="344de-104">Pärast kliendile projektipakkumise loomist ja saatmist peate saama enne pakkumise uuendamist olekusse **Saadetud** kliendilt vastava kinnituse.</span><span class="sxs-lookup"><span data-stu-id="344de-104">After you create and send a project quotation to a customer, you must get confirmation from the customer before you update the quotation status to **Sent**.</span></span> <span data-ttu-id="344de-105">Kõiki kliendi palutud muudatusi saab pakkumises uuendada.</span><span class="sxs-lookup"><span data-stu-id="344de-105">Any modifications requested by the customer can be updated in the quotation.</span></span> <span data-ttu-id="344de-106">Kui olete muutnud pakkumise olekusse **Saadetud**, ei saa te seda muuta.</span><span class="sxs-lookup"><span data-stu-id="344de-106">After you update the quotation status to **Sent**, you can’t modify it.</span></span> <span data-ttu-id="344de-107">Järgmisena kirjeldatakse valikuid, mis on seotud pakkumise kinnitamiseks saatmisega, tagasiside põhjal uuenduste tegemisega ja pakkumise saatmisega.</span><span class="sxs-lookup"><span data-stu-id="344de-107">The following procedure describes the options to send a quotation for confirmation, make updates based on feedback, and then send the quotation.</span></span>

## <a name="send-a-project-quotation-confirmation"></a><span data-ttu-id="344de-108">Projektipakkumise kinnituse saatmine</span><span class="sxs-lookup"><span data-stu-id="344de-108">Send a project quotation confirmation</span></span>  

<span data-ttu-id="344de-109">Te saate saata olemasoleva projektipakkumise kinnitamiseks e-posti teel või prinditud koopia näol.</span><span class="sxs-lookup"><span data-stu-id="344de-109">You can send an existing project quotation for confirmation through email or as a printed copy.</span></span> 

1. <span data-ttu-id="344de-110">Avage **Projektihaldus ja raamatupidamine** > **Pakkumised** > **Projektipakkumine.**</span><span class="sxs-lookup"><span data-stu-id="344de-110">Go to **Project management and accounting** > **Quotations** > **Project quotation.**</span></span> 
2. <span data-ttu-id="344de-111">Valige lehel **Projektipakkumine** pakkumine, mille soovite saata kinnitamiseks.</span><span class="sxs-lookup"><span data-stu-id="344de-111">On the **Project quotation** page, select the quotation you want to send for confirmation.</span></span> 
3. <span data-ttu-id="344de-112">Valige vahekaardil **Järeltegevus** grupis **Loomine** suvand **Kinnita**.</span><span class="sxs-lookup"><span data-stu-id="344de-112">On the **Follow up** tab, in the **Generate** group, select **Confirm**.</span></span> 
4. <span data-ttu-id="344de-113">Seadke lehel **Pakkumise kinnitamine** nõutud parameetrid.</span><span class="sxs-lookup"><span data-stu-id="344de-113">On the **Confirm quotation** page, set the required parameters.</span></span> <span data-ttu-id="344de-114">Näiteks kinnituse printimiseks lülitage suvandites **Printimine** sisse **Prindi kinnitus** ja seejärel valige **OK**.</span><span class="sxs-lookup"><span data-stu-id="344de-114">For example, to print the confirmation, under **Print** options, turn on **Print confirmation**, and then select **OK**.</span></span>
5. <span data-ttu-id="344de-115">Valige lehel **Saada pakkumine** **Suvandid** ning valige lehel **Prindisihtkoha sätted**, kas soovite saata pakkumise meetodil **Ekraan**, **E-post**, **Fail**, **Prindiarhiiv** või **Printer** ning seejärel tehke pakkumise suunamiseks soovitud muudatused alal **Täpsustus**.</span><span class="sxs-lookup"><span data-stu-id="344de-115">On the **Send quotation** page, select **Options**, and on the **Print destination settings** page, select whether you want the quotation sent to **Screen**, **E-mail**, **File**, **Print archive**, or **Printer**, and then make any adjustments that you want in the **Specification** area to route the quotation.</span></span>
6. <span data-ttu-id="344de-116">Valige lehelt **Prindisihtkoha sätted** suvand **OK**.</span><span class="sxs-lookup"><span data-stu-id="344de-116">On the **Print destination settings** page, select **OK**.</span></span>
