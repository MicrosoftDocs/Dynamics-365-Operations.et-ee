[!include [banner](../includes/banner.md)]

<span data-ttu-id="0d15d-104">Projektijuht saab plaanida projekti jaoks vajalikke ressursirolle.</span><span class="sxs-lookup"><span data-stu-id="0d15d-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="0d15d-105">Ressursihaldur näeb neid plaanitud ressursse taotlustena lehel **Ressursi täitmine** ja saab määrata tegelikke ressursse.</span><span class="sxs-lookup"><span data-stu-id="0d15d-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="0d15d-106">Valige lehelt **Kõik projektid** projekt **XYZ täiendusfaas 2**.</span><span class="sxs-lookup"><span data-stu-id="0d15d-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="0d15d-107">Valige **Projekt** ja seejärel **Redigeeri**.</span><span class="sxs-lookup"><span data-stu-id="0d15d-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="0d15d-108">Tehke vahekaardil **Projektimeeskond ja plaanimine** valik **Lisa**.</span><span class="sxs-lookup"><span data-stu-id="0d15d-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="0d15d-109">Valige dialoogiboksist **Lisa rollid** roll **Tarkvaraarendaja**.</span><span class="sxs-lookup"><span data-stu-id="0d15d-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="0d15d-110">Valige **Loo** ja sulgege seejärel projekti leht.</span><span class="sxs-lookup"><span data-stu-id="0d15d-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="0d15d-111">Tehke lehel **Ressursi täitmine** projektile **XYZ täiendusprojekti faas 2** valik **Tarkvaraarendaja 1**.</span><span class="sxs-lookup"><span data-stu-id="0d15d-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="0d15d-112">Valige töötaja ja valige siis **Määra**.</span><span class="sxs-lookup"><span data-stu-id="0d15d-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="0d15d-113">Veenduge, et rida **Tarkvaraarendaja 1** oleks projekti **XYZ täiendusprojekti faas 2** puhul eemaldatud.</span><span class="sxs-lookup"><span data-stu-id="0d15d-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="0d15d-114">Vahekaardil **Projektimeeskond ja plaanimine** projekti **XYZ täiendusfaas 2** puhul veenduge, et töötaja, kelle eelmises toimingus valisite, on lisatud kui **Tarkvaraarendaja**.</span><span class="sxs-lookup"><span data-stu-id="0d15d-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
