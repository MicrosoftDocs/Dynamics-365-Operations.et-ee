# <a name="project-mobile-applications-overview"></a>Projecti mobiilirakenduste ülevaade

[!include [banner](../includes/banner.md)]

Tootele Microsoft Dynamics 365 Project Timesheet on saadaval kolm projekti ajaga seotud mobiilirakendust: Projekti ajakirje ja Minu ajatabelid / Minu ülevaatamise ajatabelid (optimeeritud mobiilile).

## <a name="project-timesheet-mobile-app"></a>Mobiilirakendus Project Timesheet

Mobiilirakendus Dynamics 365 Project Timesheet võimaldab kasutajatel mobiilses seadmes projektide ajatabeleid sisestada ja kinnitada. See mobiilirakendus tõstab esile rakenduse Dynamics 365 Finance Projektihalduse ja -arvestuse ala ajatabelite funktsiooni. Rakendus parendab kasutajate tootlikkust ja tõhusust ning hõlbustab projektide ajatabelite õigeaegset sisestamist ja kinnitamist.

## <a name="project-time-entry-workspace"></a>Projekti ajakirje tööruum

Projekti ajakirje tööruum võimaldab kasutajatel projektis töötatud aega sisestada ja salvestada. Siiski ei lase see kasutajatel projekti ajatabeleid sisestada ega kinnitada. Seda saab kasutada ainult ajakirjeteks. See mobiilne tööruum on mõeldud kasutamiseks mobiilirakendusega Finance and Operations.

## <a name="my-timesheetstimesheets-for-my-review"></a>Minu ajatabelid / Minu ülevaatamise ajatabelid

Minu ajatabelid (optimeeritud mobiilile) ja Minu ülevaatamise ajatabelid (optimeeritud mobiilile) moodustavad lahenduse, mis võimaldab töötajatel mobiilses seadmes projekti ajatabeleid luua, sisestada ning kinnitada. Töötajad pääsevad funktsioonile Financeligi rakenduse URL-i kaudu. See lahendus ei nõua mobiilirakendust Finance and Operations.
