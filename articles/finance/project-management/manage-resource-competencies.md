## <a name="set-up-competencies-certificates"></a><span data-ttu-id="440ed-112">Kompetentside seadistamine: tunnistused</span><span class="sxs-lookup"><span data-stu-id="440ed-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="440ed-113">Valige loendilehelt **Töötajad** selle töötaja rida, kellele tunnistuse andmeid lisate.</span><span class="sxs-lookup"><span data-stu-id="440ed-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>
2. <span data-ttu-id="440ed-114">Tehke tegumiriba vahekaardil **Töötaja** grupis **Kompetentsid** valik **Tunnistused**.</span><span class="sxs-lookup"><span data-stu-id="440ed-114">On the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Certificates**.</span></span>
3. <span data-ttu-id="440ed-115">Valige **Uus** ja tehke siis väljal **Tunnistuse tüüp** valik **PMP**.</span><span class="sxs-lookup"><span data-stu-id="440ed-115">Select **New**, and then, in the **Certificate type** field, select **PMP**.</span></span>
4. <span data-ttu-id="440ed-116">Tehke väljal **Alguskuupäev** valik **10/1/2015** ja valige **Salvesta**.</span><span class="sxs-lookup"><span data-stu-id="440ed-116">In the **Start date** field, select **10/1/2015**, and select **Save**.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="440ed-117">Kompetentside seadistamine: oskused</span><span class="sxs-lookup"><span data-stu-id="440ed-117">Set up competencies: Skills</span></span>

1. <span data-ttu-id="440ed-118">Veenduge lehel **Töötajad**, et eelmises protseduuris kasutatud töötaja on endiselt valitud.</span><span class="sxs-lookup"><span data-stu-id="440ed-118">On the **Workers** list page, make sure that the worker that you used in the previous procedure is still selected.</span></span> <span data-ttu-id="440ed-119">Seejärel tehke tegumiriba vahekaardil **Töötaja** grupis **Kompetentsid** valik **Oskused**.</span><span class="sxs-lookup"><span data-stu-id="440ed-119">Then, on the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Skills**.</span></span>
2. <span data-ttu-id="440ed-120">Valige **Uus**.</span><span class="sxs-lookup"><span data-stu-id="440ed-120">Select **New**.</span></span>
3. <span data-ttu-id="440ed-121">Tehke väljal **Oskus** valik **Projektihaldus**.</span><span class="sxs-lookup"><span data-stu-id="440ed-121">In the **Skill** field, select **Project management**.</span></span>
4. <span data-ttu-id="440ed-122">Tehke väljal **Tase** valik **5 Ekspert**.</span><span class="sxs-lookup"><span data-stu-id="440ed-122">In the **Level** field, select **5 Expert**.</span></span>
5. <span data-ttu-id="440ed-123">Tehke väljal **Taseme kuupäev** valik **1-/14/2014**.</span><span class="sxs-lookup"><span data-stu-id="440ed-123">In the **Level date** field, select **1-/14/2014**.</span></span>
6. <span data-ttu-id="440ed-124">Sisestage väljale **Kogemus aastates** väärtus **10**.</span><span class="sxs-lookup"><span data-stu-id="440ed-124">In the **Years of experience** field, enter **10**.</span></span>
7. <span data-ttu-id="440ed-125">Valige **Salvesta** ja sulgege seejärel leht.</span><span class="sxs-lookup"><span data-stu-id="440ed-125">Select **Save**, and then close the page.</span></span>
