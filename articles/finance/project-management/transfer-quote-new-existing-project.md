<span data-ttu-id="1b121-104">Kui projektipakkumise olek on **Kinnitatud**, siis saate kanda üle mõned või kõik pakkumise read uuele või olemasolevale projektile viisardi **Ülekanne projektile** abil.</span><span class="sxs-lookup"><span data-stu-id="1b121-104">When a project quotation has a status of **Confirmed**, you can transfer some or all the quotation lines to a new or existing project by using the **Transfer to project** wizard.</span></span> 

<span data-ttu-id="1b121-105">Viisardi kasutamiseks toimige järgmiselt.</span><span class="sxs-lookup"><span data-stu-id="1b121-105">To use the wizard, follow these steps.</span></span>
