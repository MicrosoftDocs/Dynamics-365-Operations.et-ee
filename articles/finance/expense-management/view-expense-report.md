# <a name="view-an-expense-report-from-general-ledger-or-procurement-and-sourcing"></a><span data-ttu-id="263c2-103">Kuluaruande vaatamine pearaamatust või hangetest</span><span class="sxs-lookup"><span data-stu-id="263c2-103">View an expense report from General ledger or Procurement and sourcing</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="263c2-104">Kui teete päringu kannete nägemiseks pearaamatus või hangetes, võib päring tagastada liiga palju teavet.</span><span class="sxs-lookup"><span data-stu-id="263c2-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="263c2-105">Seetõttu võite soovida otsingut kitsendada.</span><span class="sxs-lookup"><span data-stu-id="263c2-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="263c2-106">Vajaliku teabe saamiseks saate kandesse süvitsi minna, et näha kuluaruannet, millel kanne kuvati.</span><span class="sxs-lookup"><span data-stu-id="263c2-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>
