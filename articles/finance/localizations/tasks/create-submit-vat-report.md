# <a name="create-and-submit-vat-report"></a>KM-aruande loomine ja esitamine

[!include [banner](../../includes/banner.md)]

See protseduur kirjeldab, kuidas luua ja edastada käibemaksuaruannet Saksa maksuametile. Protseduuri loomisel kasutati demoettevõtte DEMF andmeid. See funktsionaalsus on saadaval ainult juriidilistele isikutele, kelle esmane aadress on Saksamaal.

1. Avage Maksud > Deklaratsioonid > Käibemaks > Käibemaksuaruande koostamine tasakaalustusperioodi kohta.
2. Valige või sisestage väärtus väljal Tasakaalustusperiood.
3. Sisestage kuupäev väljale Alguskuupäev.
4. Klõpsake nuppu OK.
5. Valige väljal Loo elektrooniline maksudokument suvand Jah.
6. Klõpsake nuppu OK.
