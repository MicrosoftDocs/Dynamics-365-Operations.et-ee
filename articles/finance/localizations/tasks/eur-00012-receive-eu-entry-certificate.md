# <a name="eur-00012-receive-an-eu-entry-certificate"></a>EUR-00012 EL-i kandesertifikaadi vastuvõtmine

[!include [banner](../../includes/banner.md)]

See protseduur selgitab muu osapoole väljastatud ja teie kliendilt saadud EL-i kandesertifikaadi registreerimist.



Selle protseduuri loomiseks kasutatav demoandmete ettevõte on DEMF.

1. Minge jaotisse Müügireskontrod > Päringud ja aruanded > Kandesertifikaadi tööleht.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Kandesertifikaat.
4. Klõpsake väljal Kliendi konto otsingu avamiseks ripploendi nuppu.
5. Klõpsake loendis valitud real olevat linki.
6. Klõpsake nuppu Salvesta.
    * Saate lisada töölehele lisateavet.  
    * Näiteks saate manustada töölehele EL-i kandesertifikaadi skannitud pildi.
