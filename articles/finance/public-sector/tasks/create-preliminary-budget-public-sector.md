# <a name="create-a-preliminary-budget-for-public-sector"></a>Esialgse eelarve loomine avaliku sektori puhul

[!include [banner](../../includes/banner.md)]

Saate luua esialgsed eelarve registreerimiskanded kindla eelarvemudeli jaoks ja dimensiooniväärtused. Pärast tegeliku eelarve kinnitamist saate luua eelarve algsed registrikirjed. See protseduur loodi PSUS-ettevõtte demoandmetega avaliku sektori sektsioonis.

1. Tehke valik Eelarvestamine > Eelarveregistri kirjed.
2. Klõpsake valikut Uus.
3. Klõpsake väljal Budget model (Eelarve mudel) otsingu avamiseks ripploendi nuppu.
4. Otsige loendist ja valige soovitud kirje.
5. Klõpsake väljal Budget code (Eelarve kood) otsingu avamiseks ripploendi nuppu.
6. Otsige loendist ja valige soovitud kirje.
7. Klõpsake väljal Reason code (Põhjuse kood) otsingu avamiseks ripploendi nuppu.
8. Klõpsake loendis soovitud kirjel.
9. Klõpsake nuppu Salvesta.
10. Klõpsake käsku Lisa rida.
    * Valikuline: kui soovite päises kuupäeva muuta, sisestage uus kuupäev. See kuupäev määrab rahdnusperioodi, millesse eelarve salvestatakse. Ülesannete juhendi kuvamisel teiste väljade täitmiseks klõpsake lehe ülaosas valikut Unlock (Vabasta lukust).  
11. Klõpsake väljal Konto struktuur otsingu avamiseks ripploendi nuppu.
12. Otsige loendist ja valige soovitud kirje.
13. Täpsustage väljal Dimension values (Dimensiooni väärtused) soovitud väärtused.
14. Sisestage number väljale Summa.
    * Saate sisestada ka summa tüübi.  
15. Klõpsake väljal Valuuta otsingu avamiseks ripploendi nuppu.
16. Otsige loendist ja valige soovitud kirje.
17. Klõpsake nuppu Salvesta.
18. Klõpsake Update budget balances (Värskenda eelarve tasakaalud).
19. Klõpsake käsku Uuenda.
    * Uuendamise tulemuste vaatamiseks klõpsake sinisel ribal valikut Message details (Teate üksikasjad).
