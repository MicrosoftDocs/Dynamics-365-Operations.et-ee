# <a name="set-up-and-create-depreciation-profiles"></a>Kulumireeglite seadistamine ja loomine

[!include [banner](../../includes/banner.md)]

Kulumireeglid määratlevad vara kulumi tüübi ja sageduse.


## <a name="create-a-depreciation-profile"></a>Kulumireeglite loomine
1. Avage navigeerimispaanil **Moodulid > Põhivarad > Seadistus > Kuluprofiilid**.
2. Klõpsake valikut **Uus**.
3. Sisestage väärtus väljale **Kuluprofiil**.
4. Sisestage väärtus väljale **Nimi**.
5. Tehke väljas **Meetod** valik. Kui valite saldo vähendamise, peate sisestama protsendi väljale **Protsent**.  
6. Valige suvand väljal **Kuluaasta**.
7. Valige suvand väljal **Perioodi sagedus**.
8. Sulgege vorm.
