# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="3b6f0-103">Põhivara käibelt kõrvaldamine vabas vormis arve abil</span><span class="sxs-lookup"><span data-stu-id="3b6f0-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="3b6f0-104">Siin peatükis selgitatakse, kuidas käibelt kõrvaldada põhivara vabas vormis arve abil.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="3b6f0-105">Navigeerimispaanil avage **Moodulid > Müügireskontro > Arved > Kõik vaba tekstiga arved**.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="3b6f0-106">Valige suvand **Uus**.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-106">Select **New**.</span></span>
3. <span data-ttu-id="3b6f0-107">Väljale **Kliendi konto** sisestage või valige väärtus.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="3b6f0-108">Kinnitage tavaline **Arve** kuupäev ja kohaldatavusel muutke seda.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="3b6f0-109">Kinnitage ülejäänud tavalised päiseväljad nagu **Valuuta** ja kohaldatavusel muutke neid.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="3b6f0-110">Kinnitage jaotise **Arve read** väljad **Kirjeldus** ja **Põhikonto**.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="3b6f0-111">Kinnitage tavaväljad **Käibemaksugrupp** ja **Kauba käibemaksugrupp**.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="3b6f0-112">Sisestage põhivara müügi **Ühiku hind** ja **Summa**.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="3b6f0-113">Valige jaotis **Rea üksikasjad**.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="3b6f0-114">Sisestage väärtus välja **Põhivara** või valige sealne väärtus.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="3b6f0-115">Valige **Sisesta**.</span><span class="sxs-lookup"><span data-stu-id="3b6f0-115">Select **Post**.</span></span>
