# <a name="set-up-accessorial-assignments"></a>Lisade määrangute seadistamine

[!include[task guide banner](../../includes/task-guide-banner.md)]

See protseduur näitab, kuidas seadistada lisade määranguid. Seda teeb üldjuhul transpordikoordinaator. Enne selle juhendi kasutamist peate käivitama juhendi Keskuse lisatasude ja lisade koondandmete seadistamine.


## <a name="set-up-accessorial-assignment"></a>Lisade määrangu seadistamine
1. Avage Transpordihaldus > Seadistus > Hinnang > Lisade määrangud.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Nimi.
4. Laiendage jaotist Üksikasjad.
5. Klõpsake väljal Keskus otsingu avamiseks ripploendi nuppu.
6. Valige loendist Keskus, mille puhul lõite lisade koondandmed juhendi Keskuse lisatasude ja lisade koondandmete seadistamine käitamisel. 
7. Klõpsake väljal Keskuse lisade ID otsingu avamiseks ripploendi nuppu.
8. Klõpsake loendis valitud real olevat linki.
9. Laiendage jaotist Kriteeriumid.
    * Jaotises Kriteeriumid saate valida täpsed kriteeriumid tasu rakendamise puhul siintoodud eri väärtuste põhjal.  
10. Seadke suvand Rakenda alati valikule Jah.
11. Valige suvand väljalt Lisade määramise tase.
12. Laiendage jaotist Arvutamine.
13. Valige väljal Lisatasu tüüp suvand Kindel.
    * Lisatasu tüüp määrab, kuidas tegelikku tasu arvutada. Selles näites on tegemist kindla tasuga.  
14. Sisestage number väljale Lisatasu.
15. Klõpsake nuppu Salvesta.
