# <a name="copy-co-products-from-an-existing-formula-version"></a>Kaastoodete kopeerimine olemasolevast valemiversioonist

[!include[task guide banner](../../includes/task-guide-banner.md)]

See protseduur näitab, kuidas kopeerida kaastooteid olemasolevalt valemiversioonilt teistsugusele väljastatud toote valemiversioonile. Eeldus on, et kaastoodete on seotud vähemalt üks valemiversioon. Selle protseduuri loomiseks kasutati demoettevõtte USP2 andmeid.


## <a name="find-a-released-product"></a>Väljastatud toote otsimine
1. Avage Väljastatud tooted.
2. Klõpsake suvandit Näita filtreid.
    * Olete lisamas filtri dialoogiboksi välja Tootmise tüüp.  
3. Klõpsake käsku Lisa filtri väli, et lisada väli Tootmise tüüp.
    * Järgmises etapis peate enne valiku Rakenda tegemist sisestama käsitsi valemi väljale Tootmise tüüp. See määrab filtri väljastatud toodete loendile.  
4. Sisestage käsitsi valem väljale Tootmise tüüp.
5. Klõpsake käsku Apply (Rakenda).

## <a name="select-a-released-product"></a>Väljastatud toote valimine
1. Otsige loendist ja valige soovitud kirje.
2. Klõpsake valikut Valemiversioonid.
    * Klõpsake tegumiribal Tehnika valikut Valemiversioonid.  

## <a name="copy-co-products"></a>Kaastoodete kopeerimine
1. Klõpsake tegumiribal valikut Valemiversioon.
2. Klõpsake valikut Kaastooted.
3. Klõpsake käsku Kopeeri.
4. Sisestage või valige väärtus väljal Kaubakood.
5. Valige või sisestage väärtus väljal Valemi versioon.
6. Klõpsake nuppu OK.
7. Sulgege leht.
