# <a name="create-a-hierarchy-of-product-classification"></a>Tooteklassifikatsiooni hierarhia loomine

[!include[task guide banner](../../includes/task-guide-banner.md)]

See protseduur selgitab, kuidas luua uut kategooriahierarhiat ja määrata kaubaartikli koodi hierarhiatüüpi. Selle protseduuri loomiseks kasutati demoettevõtte USMF-i andmeid. See protseduur on mõeldud kategooriajuhile.


## <a name="create-the-new-category-hierarchy"></a>Uue kategooriahierarhia loomine
1. Minge jaotisse Tooteteabe haldus > Seadistus > Kategooriad ja atribuudid > Kategooriahierarhiad.
2. Klõpsake valikut Uus.
3. Sisestage väärtus väljale Nimi.
4. Sisestage väljale Kirjeldus soovitud väärtus.
5. Klõpsake käsku Loo.

## <a name="build-the-hierarchy"></a>Hierarhia loomine
1. Klõpsake suvandit Uus kategooriasõlm.
2. Sisestage väärtus väljale Nimi.
3. Sisestage väärtus väljale Kood.
4. Sisestage väärtus väljale Hüüdnimi.
5. Klõpsake suvandit Uus kategooriasõlm.
6. Sisestage väärtus väljale Nimi.
7. Sisestage väärtus väljale Kood.
8. Sisestage väärtus väljale Hüüdnimi.
9. Klõpsake suvandit Uus kategooriasõlm.
10. Sisestage väärtus väljale Nimi.
11. Sisestage väärtus väljale Kood.
12. Sisestage väärtus väljale Hüüdnimi.
13. Klõpsake suvandit Uus kategooriasõlm.
14. Sisestage väärtus väljale Nimi.
15. Sisestage väärtus väljale Kood.
16. Sisestage väärtus väljale Hüüdnimi.
17. Sulgege leht.

## <a name="classify-the-hierarchy"></a>Hierarhia klassifitseerimine
1. Otsige loendist ja valige soovitud kirje.
2. Klõpsake toimingupaanil suvandit Kategooriahierarhia.
3. Klõpsake suvandit Hierarhiatüübi seostamine.
4. Klõpsake valikut Uus.
5. Valige suvand väljal Kategoooriahierarhia tüüp.
    * Valige toote klassifitseerimiseks suvand Kaubaartikli koodi kategooriahierarhia tüüp.  
6. Klõpsake väljal Kategooriahierarhia otsingu avamiseks ripploendi nuppu.
7. Otsige loendist ja valige soovitud kirje.
8. Klõpsake loendis valitud real olevat linki.
9. Sulgege leht.
